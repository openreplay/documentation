اربط معرّف مستخدم (userID) مجهولًا بالجلسة التي يجري تسجيلها.

## التوقيع

```swift
ORTracker.shared.userAnonymousID(_ userID: String)
```

## المعاملات

- `userID: String`: المعرّف المجهول للمستخدم. وهو سلسلة نصية ويمكن أن يكون أي شيء طالما أنه سلسلة نصية ويتيح لك تحديد هوية المستخدم بشكل فريد داخل نظامك (أي البريد الإلكتروني، معرّف قاعدة البيانات، إلخ).

## القيمة المُعادة

لا تُعيد هذه الطريقة أي قيم، وستكون النتائج مرئية في منصة OpenReplay.
