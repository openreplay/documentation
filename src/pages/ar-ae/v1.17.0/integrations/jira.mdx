كيفية دمج Jira Cloud مع OpenReplay.

### 1. إنشاء رمز API جديد

1. سجّل الدخول إلى [رمز JIRA](https://id.atlassian.com/manage/api-tokens) الخاص بك.
2. انقر على *Create API token*.
3. في حقل التسمية، أدخل "openreplay" ثم انقر على *Create*.
4. انقر على *Copy to clipboard*.

### 2. تفعيل JIRA Cloud في OpenReplay

1. انتقل إلى لوحة تحكم OpenReplay الخاصة بك ضمن 'Preferences > Integration'
2. اختر Jira. إذا لم تجده، فهذا يعني أنك قد فعّلت GitHub (لا يمكنك استخدام كليهما في الوقت نفسه، لذا تأكد من تعطيل تكامل GitHub أولاً).
3. أدخل `Username` و`API token` الخاصين بك، ثم اكتب عنوان URL لنسخة Jira Cloud الخاصة بك

عندئذٍ سيظهر زر 'Create Issue' في جميع التسجيلات حتى تتمكن من إنشاء المهام وإسنادها بسرعة في Jira.
