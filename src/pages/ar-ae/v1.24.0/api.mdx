تتيح لك واجهة OpenReplay البرمجية إدارة بيانات المستخدمين وتصدير الأحداث الملتقطة لتحليلها خارجيًا.

## HTTP API
- [المشاريع](/api/projects): لإنشاء مشاريع جديدة.
- [الجلسات](/api/sessions): لاسترجاع جميع الجلسات الخاصة بمستخدم محدد.
- [الأحداث](/api/events): لسحب جميع أحداث تسجيل معين.
- [المستخدمون](/api/users): للتحكم في بيانات المستخدمين وحذفها.
- [المهام](/api/jobs): لإدارة مهام حذف بيانات المستخدمين.
- [Assist](/api/assist): لاسترجاع الجلسات المباشرة.
