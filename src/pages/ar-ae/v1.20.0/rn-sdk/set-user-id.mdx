تتيح لك هذه الدالة تعيين معرّف مستخدم للجلسة النشطة حاليًا. وهذا يسمح لك بربط مستخدم بإعادة التشغيل، مما يمكّنك من تصفية الجلسات والبحث عن جلسات مستخدم معيّن.

## التوقيع

```js
import OR from '@openreplay/react-native'

// ...
OR.tracker.setUserID('id')
```


## المعاملات

- `userID: String`: معرّف المستخدم. وهو عبارة عن سلسلة نصية ويمكن أن يكون أي شيء طالما أنه سلسلة نصية، ويتيح لك تحديد هوية المستخدم بشكل فريد داخل نظامك (مثل البريد الإلكتروني، أو معرّف قاعدة البيانات، وما إلى ذلك).

## القيمة المُعادة

لا تُعيد هذه الدالة أي قيم، وستكون النتائج مرئية في منصة OpenReplay.
