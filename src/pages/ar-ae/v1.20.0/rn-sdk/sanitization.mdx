## مكونات واجهة المستخدم

ستظهر طرق العرض التي تمت تنقيتها مموَّهة في إعادة التشغيل الناتجة. يمكنك إضافة أي مكوّن من مكونات واجهة المستخدم، وسيتم إزالته تلقائيًا (للحفاظ على الذاكرة) بمجرد أن يصبح خارج الشاشة.

### ORSanitizedView

تُصدِّر حزمة SDK المكوّن `ORSanitizedView` الذي يمكنك استخدامه لتغليف مكوناتك الحساسة من أجل تمويهها.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORSanitizedView>
    <SensitiveComponent />
  </OR.ORSanitizedView>

// ...
```
