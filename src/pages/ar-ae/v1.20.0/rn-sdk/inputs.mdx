ستستمع أداة التتبّع إلى حدث تغيّر القيمة (مع تأخير زمني debounced) وترسل القيمة الناتجة إلى الخادم الخلفي لـ OpenReplay.

## إضافة المدخلات المتتبَّعة

تصدّر حزمة التطوير (SDK) المكوّن `ORTrackedInput` الذي يمكنك استخدامه مثل مكوّنات TextInput العادية.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORTrackedInput
    style={styles.input}
    onChangeText={onChangeNumber}
    value={number}
    placeholder="important placeholder"
  />

// ...
```
