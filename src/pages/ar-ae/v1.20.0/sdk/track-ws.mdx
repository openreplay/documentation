إذا لم تكن المعلومات التي يتتبعها أداة التتبع لدينا بشكل افتراضي عند حدوث خطأ JS كافية، يمكنك استخدام هذه الطريقة. تقوم طريقة `handleError` بالإبلاغ يدويًا عن الاستثناءات الملتقطة أو الوعود المرفوضة أو أحداث الخطأ. راجع [الإبلاغ عن الأخطاء](/installation/error-reporting#manuallyloggingexceptions) لمزيد من التفاصيل.

## التوقيع

```ts 
trackWs: (channelName: string) => (msgType: string, data: string, dir: 'up' | 'down') => void
```

## المعاملات

- `channelName: string`: اسم قناة WS.

## القيمة المُرجَعة

تُرجِع خطافًا (hook) يُطبَّق على رسائل ws ويأخذ كمعاملات نوع الرسالة (أو الاسم) والبيانات واتجاه الرسالة (up/down === مُرسَلة/مُستقبَلة).
