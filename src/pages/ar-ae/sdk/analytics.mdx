## حزمة تطوير التحليلات (SDK)

تعرّف على كيفية إعداد حزمة تطوير تحليلات المتتبع في OpenReplay.

## الوحدات

<div class="or-rowlist">
  <LinkRow href="/sdk/analytics/people" title="الأشخاص" icon="user" desc="إدارة ملفات تعريف المستخدمين وسماتهم." />

لمعرفة المزيد عن إدارة بيانات التحليلات الخاصة بك في واجهة OpenReplay، تفضّل بزيارة [إدارة البيانات](/product-analytics/data-management).

## التواقيع عالية المستوى

```ts
// Identify a user with an id (e.g. email, username, etc.)
tracker.analytics.identify(userId: string): void
// same as
tracker.analytics.people.identify()

// Add event to batch with option to send it immediately
tracker.analytics.track(eventName: string, tempProperties?: Record<string, any>, options?: { send_immediately: boolean })
// same as
tracker.analytics.events.track()
```
