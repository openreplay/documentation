ادمج GitHub مع OpenReplay وأنشئ المهام (issues) مباشرةً من صفحة التسجيل.

## استخدام OpenReplay مفتوح المصدر

#### 1. أنشئ رمز وصول شخصيًا

1. سجّل الدخول إلى Github ثم انتقل إلى [Settings > Developers Settings > Personal access tokens](https://github.com/settings/tokens)
2. انقر على 'Generate a new token'
3. حدّد النطاقات `repo:status` و`repo_deployment` و`public_repo` و`read:user` و`user:email`
4. انقر على 'Generate token' ثم انسخ الرمز

#### 2. فعّل GitHub في OpenReplay

1. انتقل إلى لوحة تحكم OpenReplay ثم إلى 'Preferences > Integration'
2. حدّد GitHub. إذا لم تره، فلا بد أنك قد فعّلت Jira (لا يمكنك استخدام كليهما في الوقت نفسه، لذا تأكد من تعطيل تكامل Jira أولًا).
3. الصق الـ `token` الذي أنشأته سابقًا

سيظهر بعد ذلك زر 'Create Issue' في جميع التسجيلات حتى تتمكن من إنشاء المهام وإسنادها بسرعة في GitHub.

## استخدام OpenReplay Cloud

انتقل إلى 'Preferences > Integration' وانقر على GitHub. ستتم إعادة توجيهك إلى GitHub للمصادقة. هذا كل ما عليك فعله. سيظهر بعد ذلك زر 'Create Issue' في جميع التسجيلات حتى تتمكن من إنشاء المهام وإسنادها بسرعة في GitHub.
