كيفية دمج Jira Cloud مع OpenReplay.

## 1. إنشاء رمز API جديد

1. سجّل الدخول إلى [رمز JIRA](https://id.atlassian.com/manage/api-tokens) الخاص بك.
2. انقر على *Create API token*.
3. في حقل التسمية، أدخل "openreplay" ثم انقر على *Create*.
4. انقر على *Copy to clipboard*.

## 2. تفعيل JIRA Cloud في OpenReplay

1. انتقل إلى لوحة تحكم OpenReplay ضمن 'Preferences > Integration'
2. اختر Jira. إذا لم تجده، فلا بد أنك قد فعّلت GitHub (لا يمكنك استخدام كليهما في الوقت نفسه، لذا تأكد من تعطيل تكامل GitHub أولاً).
3. أدخل `Username` و`API token` الخاصين بك، ثم اكتب عنوان URL لمثيل Jira Cloud الخاص بك

عندها سيظهر زر 'Create Issue' في جميع التسجيلات حتى تتمكن من إنشاء المهام وإسنادها في Jira بسرعة.
