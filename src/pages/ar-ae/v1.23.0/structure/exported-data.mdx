يأتي OpenReplay Enterprise Edition مزوَّدًا بموصِّلات مختلفة لتصدير البيانات الملتقطة (سلوك المستخدم والمشكلات التقنية) إلى مستودع البيانات الخاص بك. نحن ندعم:
- Amazon Redshift
- Google BigQuery
- PostgreSQL
- Snowflake
- ClickHouse
