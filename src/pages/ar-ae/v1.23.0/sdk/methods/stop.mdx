توقف هذه الطريقة المتعقّب (tracker) والجلسة النشطة. إذا لم تستخدم هذه الطريقة، فسيتم إيقاف الجلسة تلقائيًا بعد بضع دقائق من إغلاقك لعلامة التبويب أو مغادرتك للموقع.

## التوقيع

```js
stop(): string | undefined
```

## القيمة المُعادة

تُعيد `sessionHash` الخاص بالجلسة المُوقَفة.
