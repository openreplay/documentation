اربط معرّف المستخدم الداخلي الخاص بك بالجلسة التي يجري تسجيلها باتباع الخطوات أدناه.

### باستخدام NPM

أدخِل قيمة userID عند بدء تشغيل الـ tracker:

```js
const tracker = new OpenReplay({
  projectKey: PROJECT_KEY
});

tracker.start({
  userID: "john@doe.com",
  metadata: {
    balance: "10M",
    plan: "free"
  }
});
```

إذا لم يكن ذلك ممكنًا (يُعرف userID لاحقًا في مسار التنقل، أي بعد بدء تشغيل الـ tracker بوقت طويل)، فاستدعِ عندئذٍ الدالة `setUserID` لتحديد هوية مستخدميك. يمكن تغيير هوية المستخدم في أي وقت أثناء الجلسة عن طريق استدعاء `setUserID`. لكن OpenReplay سيحتفظ فقط بآخر معرّف مستخدم تم إدخاله.

```js
tracker.setUserID('john@doe.com'); // after tracker.start()
```

### باستخدام مقتطف JavaScript

في ما يلي مثال على كيفية إدخال userID باستخدام المقتطف. لاحظ المتغيّر/السطر `startOpts` لإدخال قيمة userID.

```js
<!-- OpenReplay Tracking Code -->
<script>
var initOpts = { projectKey: "GxPpaDARdn2345fgt321" };
var startOpts = { userID: getUserID() }; // use startOpts variable to inject userID
(function(A,s,a,y,e,r){
  r=window.OpenReplay=[e,r,y,[s-1, e]];
  s=document.createElement('script');s.src=A;s.async=!a;
  ...
})("//static.openreplay.com/latest/openreplay.js", 1, 0, initOpts, startOpts);
</script>
```

إذا لم يكن ذلك ممكنًا (يُعرف userID لاحقًا في مسار التنقل، أي بعد بدء تشغيل الـ tracker بوقت طويل)، فاستدعِ عندئذٍ الدالة `setUserID` بدلًا من ذلك لتحديد هوية مستخدميك. يمكن تغيير هوية المستخدم في أي وقت أثناء الجلسة عن طريق استدعاء `setUserID`. لكن OpenReplay سيحتفظ فقط بآخر معرّف مستخدم تم إدخاله.

```js
<!-- OpenReplay Tracking Code -->
<script>
var initOpts = { projectKey: "GxPpaDARdn2345fgt321" };
var startOpts = { userID: "" };
(function(A,s,a,y,e,r){
  r=window.OpenReplay=[e,r,y,[s-1, e]];
  s=document.createElement('script');s.src=A;s.async=!a;
  ...
})("//static.openreplay.com/latest/openreplay.js", 1, 0, initOpts, startOpts);
</script>
  ...
  OpenReplay.setUserID("john@doe.com"); // set the userID later in your code
  ...
```

### باستخدام تطبيق iOS

ما عليك سوى استدعاء `setUserID(String)` في أي وقت أثناء عملية إعادة التشغيل الخاصة بك.

```swift
import ORTracker

// ... your code

ORTracker.shared.setUserID(_ userID: String)
```
