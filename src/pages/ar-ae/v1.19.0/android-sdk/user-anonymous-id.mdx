اربط معرّف مستخدم مجهول الهوية بالجلسة التي يجري تسجيلها.

## التوقيع
```kotlin
OpenReplay.userAnonymousID(userID: String)
```

## المعاملات

- `userID: String`: المعرّف المجهول للمستخدم. هذه سلسلة نصية ويمكن أن تكون أي شيء طالما أنها سلسلة نصية وتتيح لك تحديد هوية المستخدم بشكل فريد داخل نظامك (أي البريد الإلكتروني، معرّف قاعدة البيانات، إلخ).

## القيمة المُعادة

لا تُعيد هذه الدالة أي قيم، وستكون النتائج مرئية في منصة OpenReplay.
