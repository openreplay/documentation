سيستمع المتتبّع إلى أحداث ظهور/اختفاء طرق العرض على شاشة الهاتف.

## إضافة طرق العرض المُتتبَّعة

تُصدِّر حزمة SDK المكوّن `ORAnalyticsView` الذي يمكنك استخدامه لتمييز طرق العرض/المكوّنات المهمة.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORAnalyticsView
    screenName="Pay confirmation page"
    viewName="New fancy button"
  >
    <ImportantRedButton />
  </OR.ORAnalyticsView>

// ...
```
