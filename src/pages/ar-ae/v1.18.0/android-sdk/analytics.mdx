ستستمع أداة التتبع إلى أحداث ظهور/اختفاء طرق العرض.

## إضافة طرق العرض المتتبَّعة

### SwiftUI

استخدم `.observeView(title:viewName:)` لمراقبة أحداث الإدخال،

- `title: String` - عنوان الشاشة أو الميزة
- `viewName: String` - اسم طريقة العرض في أداة التتبع

```swift
import ORTracker

TextField("Input", text: $text)
 .observeView(title: "Screen title", viewName: "test input name")
```

### UIKit
```swift
import ORTracker

Analytics.shared.addObservedView(view: inputEl, title: "Screen title", viewName: "test input name")
```
