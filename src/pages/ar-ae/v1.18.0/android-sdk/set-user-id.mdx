تتيح لك هذه الطريقة تعيين معرّف مستخدم على الجلسة النشطة حاليًا. وهذا يسمح لك بربط مستخدم بإعادة التشغيل، مما يتيح لك تصفية الجلسات والبحث فيها لمستخدم معيّن.

## التوقيع
```swift
OpenReplay.setUserID(userID: String)
```

## المعاملات

- `userID: String`: معرّف المستخدم. هذه سلسلة نصية ويمكن أن تكون أي شيء طالما أنها سلسلة نصية، وهي تتيح لك تحديد المستخدم بشكل فريد داخل نظامك (أي البريد الإلكتروني، معرّف قاعدة البيانات، إلخ).

## القيمة المُرجَعة

لا تُرجِع هذه الطريقة أي قيم، وستكون النتائج مرئية في منصة OpenReplay.
