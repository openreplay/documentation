سيستمع المتتبع إلى حدث تغيير القيمة (مع تطبيق debounce) ويرسل القيمة الناتجة إلى الواجهة الخلفية لـ OpenReplay.

## إضافة حقول الإدخال المتتبَّعة

### SwiftUI

استخدم `.trackTextInput(label:masked)` لمراقبة أحداث الإدخال،
- `label: String` - تسمية للإدخال المُسجَّل
- `masked: Bool` - إذا كانت القيمة `true`، فسيسجّل المتتبع حدث الإدخال نفسه فقط دون القيمة.

```kotlin
import com.opnereplay.tracker.OpenReplay

val passwordEditText = binding.password
passwordEditText.trackTextInput(label = "password", masked = true)

```

### View

```kotlin
import com.opnereplay.listeners.Analytics

Analytics.addObservedInput(inputEl)
```
