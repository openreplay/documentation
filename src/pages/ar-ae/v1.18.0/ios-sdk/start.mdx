تُستخدم هذه الطريقة لبدء تشغيل المُتتبِّع (tracker). عند بدء تشغيل المُتتبِّع، يبدأ تسجيل الجلسة.

## التوقيع

```swift
ORTracker.shared.start(projectKey: String, options: OROptions)
	```

### خيارات التهيئة

الخيارات المتاحة:

- `crashes: Bool` يُفعِّل crashlytics.
- `analytics: Bool` يُفعِّل التحليلات للعروض (views).
- `performances: Bool` يُفعِّل مُستمع الأداء
- `logs: Bool` يُفعِّل مُستمع السجلّات
- `screen: Bool` يُفعِّل مُسجِّل الشاشة
- `wifiOnly: Bool` يُجبر المُتتبِّع على البدء فقط إذا كان لدى المستخدم اتصال wifi

الوضع الافتراضي: جميعها `true`
