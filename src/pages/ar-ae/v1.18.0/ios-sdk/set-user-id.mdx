تتيح لك هذه الدالة تعيين معرّف مستخدم على الجلسة النشطة حاليًا. وهذا يسمح لك بربط مستخدم بإعادة التشغيل، مما يمكّنك من تصفية الجلسات والبحث فيها عن مستخدم معيّن.

## التوقيع
```swift
ORTracker.shared.setUserID(_ userID: String)
```

## المعاملات

- `userID: String`: معرّف المستخدم. وهو سلسلة نصية ويمكن أن يكون أي شيء طالما أنه سلسلة نصية ويتيح لك تحديد هوية مستخدم بشكل فريد داخل نظامك (مثل البريد الإلكتروني، أو معرّف قاعدة البيانات، إلخ).

## القيمة المُعادة

لا تُعيد هذه الدالة أي قيم، وستكون النتائج مرئية في منصة OpenReplay.
