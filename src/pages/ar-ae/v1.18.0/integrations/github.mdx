ادمج GitHub مع OpenReplay وأنشئ المهام مباشرة من صفحة التسجيل.

## مع OpenReplay OpenSource

### 1. إنشاء رمز وصول شخصي

1. سجّل الدخول إلى Github ثم انتقل إلى [Settings > Developers Settings > Personal access tokens](https://github.com/settings/tokens)
2. انقر على 'Generate a new token'
3. حدد النطاقات `repo:status` و`repo_deployment` و`public_repo` و`read:user` و`user:email`
4. اضغط على 'Generate token' ثم انسخ الرمز

### 2. تفعيل GitHub في OpenReplay

1. انتقل إلى لوحة تحكم OpenReplay ثم إلى 'Preferences > Integration'
2. حدد GitHub. إذا لم تره، فهذا يعني أنك قد فعّلت Jira (لا يمكنك استخدام كليهما في الوقت نفسه، لذا تأكد من تعطيل تكامل Jira أولاً).
3. الصق الرمز `token` الذي تم إنشاؤه سابقًا

سيظهر بعد ذلك زر 'Create Issue' في جميع التسجيلات حتى تتمكن من إنشاء المهام وتعيينها بسرعة في GitHub.

## مع OpenReplay Cloud

انتقل إلى 'Preferences > Integration' وانقر على GitHub. ستتم إعادة توجيهك إلى GitHub للمصادقة. هذا كل ما عليك فعله. سيظهر بعد ذلك زر 'Create Issue' في جميع التسجيلات حتى تتمكن من إنشاء المهام وتعيينها بسرعة في GitHub.
