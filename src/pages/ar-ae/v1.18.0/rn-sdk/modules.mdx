تصدّر حزمة تطوير React Native عدة مكوّنات وطرق لتمكينك من تخصيص تسجيلك:

## Connector

```js
interface IORTrackerConnector {
  startSession: (
    projectKey: string,
    optionsDict: Options,
    projectUrl?: string
  ) => void;
  setMetadata: (key: string, value: string) => void;
  event: (name: string, payload?: string) => void;
  setUserID: (userID: string) => void;
  userAnonymousID: (userID: string) => void;
}
```

- [startSession()](/rn-sdk/start): تبدأ التسجيل.
- [setMetadata()](/rn-sdk/set-metadata): تعيّن حقل بيانات وصفية وتحفظه كجزء من التسجيل.
- [event()](/rn-sdk/event): أرسل أحداثًا مخصصة لتُسجَّل كجزء من عمليات إعادة التشغيل الخاصة بك باستخدام هذه الطريقة.
- [setUserID()](/rn-sdk/set-user-id): تعرّف على هوية مستخدميك باستخدام هذه الطريقة.
- [patchNetwork()](/rn-sdk/patch-network): تعدّل fetch و/أو XMLHttpRequest لتسجيل طلبات الشبكة.
