تتيح لك واجهة برمجة تطبيقات OpenReplay إدارة بيانات المستخدمين وتصدير الأحداث الملتقطة لتحليلها خارجيًا.

## عنوان URL الأساسي (OpenReplay)

- لخطة Cloud Serverless: `https://api.openreplay.com/api/v1`
- لخطة Cloud Dedicated أو المثيلات مفتوحة المصدر ذاتية الاستضافة: `https://YOUR_INSTANCE_DOMAIN/api/`
