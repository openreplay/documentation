تتيح لك هذه الطريقة تعيين معرّف مستخدم على الجلسة النشطة حاليًا. وهذا يسمح لك بربط مستخدم بإعادة التشغيل، مما يمكّنك من تصفية الجلسات والبحث فيها لمستخدم معيّن.

## التوقيع

```js
import OR from '@openreplay/react-native'

// ...
OR.tracker.setUserID('id')
```


## المعاملات

- `userID: String`: معرّف المستخدم. هذه سلسلة نصية ويمكن أن تكون أي شيء طالما أنها سلسلة نصية وتتيح لك تحديد مستخدم بشكل فريد داخل نظامك (أي البريد الإلكتروني، معرّف قاعدة البيانات، إلخ).

## القيمة المُعادة

لا تُعيد هذه الطريقة أي قيم، وستكون النتائج مرئية في منصة OpenReplay.
