اربط معرّف مستخدم (userID) مجهول الهوية بالجلسة التي يجري تسجيلها.

## التوقيع

```swift
ORTracker.userAnonymousID(userID: String)
```

## المعاملات

- `userID: String`: المعرّف المجهول للمستخدم. هذه سلسلة نصية ويمكن أن تكون أي شيء طالما أنها سلسلة نصية، وهي تتيح لك التعرّف على المستخدم بشكل فريد داخل نظامك (أي البريد الإلكتروني، معرّف قاعدة البيانات، إلخ).

## القيمة المُعادة

لا تُعيد هذه الطريقة أي قيم، وستكون النتائج مرئية في منصة OpenReplay.
