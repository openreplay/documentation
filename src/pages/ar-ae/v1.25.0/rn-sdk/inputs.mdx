ستستمع أداة التتبّع إلى حدث تغيّر القيمة (مع تأخير debounce) وترسل القيمة الناتجة إلى الواجهة الخلفية لـ OpenReplay.

## إضافة حقول إدخال متتبَّعة

تُصدّر حزمة التطوير `ORTrackedInput` التي يمكنك استخدامها مثل مكوّنات TextInput العادية.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORTrackedInput
    style={styles.input}
    onChangeText={onChangeNumber}
    value={number}
    placeholder="important placeholder"
  />

// ...
```
