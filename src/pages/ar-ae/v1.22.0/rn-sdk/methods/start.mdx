تُستخدم هذه الدالة لبدء تشغيل المتعقّب (tracker). عند بدء تشغيل المتعقّب، يبدأ تسجيل الجلسة.

## التوقيع

```swift
import OR from '@openreplay/react-native'

OR.tracker.startSession(options)
```

## خيارات التهيئة

بشكل افتراضي، تكون جميع الخيارات مضبوطة على `true`:
- `crashes: Bool` يُفعّل crashlytics.
- `analytics: Bool` يُفعّل تتبّع التحليلات للعروض المُعلَّمة.
- `performances: Bool` يُفعّل مستمع الأداء
- `logs: Bool` يُفعّل مستمع السجلّات
- `screen: Bool` يُفعّل مُسجّل الشاشة

يمكن استخدام الخيارات لتعطيل ميزات معيّنة، وإلا يمكنك تمرير الكائن الفارغ (`{}`)
