تقوم هذه الطريقة بتعيين دالة رد النداء (callback) التي تُستخدم عند تحميل Feature Flags.

## التوقيع

```js
onFlagsLoad(callback: (flags: IFeatureFlag[]) => void): void
```

## الوسائط

دالة رد النداء التي تُستخدم بمجرد إعادة تحميل feature flags. تُنفَّذ عند كل حدث لإعادة تحميل الـ flags.
