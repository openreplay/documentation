تتيح لك هذه الطريقة تتبع قنوات websocket الفردية واتجاه الرسالة (واردة/صادرة).

## التوقيع

```ts 
trackWs: (channelName: string) => (msgType: string, data: string, dir: 'up' | 'down') => void
```

## المعاملات

- `channelName: string`: اسم قناة WS.

## القيمة المُرجَعة

تُرجِع خطافًا (hook) يُطبَّق على رسائل ws ويأخذ نوع الرسالة (أو الاسم) والبيانات واتجاه الرسالة (up/down === مُرسَلة/مُستقبَلة) كمعاملات.
