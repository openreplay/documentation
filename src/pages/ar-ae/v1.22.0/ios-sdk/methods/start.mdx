تُستخدم هذه الدالة لبدء تشغيل الـ tracker. وعند بدء تشغيل الـ tracker، يبدأ تسجيل الجلسة.

## التوقيع

```swift
ORTracker.shared.start(projectKey: String, options: OROptions)
	```

### خيارات التهيئة

الخيارات المتاحة:

- `crashes: Bool` يُفعّل crashlytics.
- `analytics: Bool` يُفعّل التحليلات لطرق العرض (views).
- `performances: Bool` يُفعّل مستمع الأداء
- `logs: Bool` يُفعّل مستمع السجلات
- `screen: Bool` يُفعّل مسجّل الشاشة
- `wifiOnly: Bool` يُجبر الـ tracker على البدء فقط إذا كان لدى المستخدم اتصال wifi

الإعداد الافتراضي: جميعها `true`
