ربط معرّف مستخدم مجهول (userID) بالجلسة التي يجري تسجيلها.

## التوقيع
```swift
ORTracker.shared.userAnonymousID(_ userID: String)
```

## المعاملات

- `userID: String`: المعرّف المجهول للمستخدم. وهو عبارة عن سلسلة نصية ويمكن أن يكون أي قيمة طالما أنه سلسلة نصية، ويتيح لك تحديد هوية المستخدم بشكل فريد داخل نظامك (مثل البريد الإلكتروني، أو معرّف قاعدة البيانات، إلخ).

## القيمة المُعادة

لا تُعيد هذه الدالة أي قيم، وستكون النتائج مرئية في منصة OpenReplay.
