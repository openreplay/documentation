تُمثّل لوحات المعلومات في OpenReplay مركزك الرئيسي لمراقبة أداء منتجك وتجربة المستخدم وتحليلهما وتحسينهما. وسواء كنت تصحّح الأخطاء أو تتتبع سلوك المستخدمين أو تقيس الأداء، فإن لوحات المعلومات توفّر لك الرؤى التي تحتاجها لاتخاذ قرارات قائمة على البيانات.

تجمع لوحات معلومات OpenReplay الرؤى الأساسية لكل عضو في فريقك:

- **للمطوّرين:** تتبّع المشكلات التقنية مثل أخطاء JavaScript وطلبات الشبكة البطيئة ومشكلات الأداء في الوقت الفعلي.
- **لمديري المنتجات:** حلّل سلوك المستخدمين، وراقب مسارات التحويل، وافهم المسارات التي يسلكها المستخدمون عبر منتجك.
- **للفِرق التي تحلّل أداء الموقع الإلكتروني:** تتبّع المقاييس الرئيسية مثل الصفحات الأكثر زيارةً ومصادر الزيارات والبيانات الديموغرافية للمستخدمين.

## نظرة عامة على لوحات المعلومات والبطاقات المتاحة

عند إنشاء لوحة معلومات، تبدأ كصفحة فارغة جاهزة لتخصيصها. تتكوّن لوحات المعلومات من **بطاقات**، حيث تعرض كل بطاقة مقياسًا محددًا أو مجموعة من البيانات. يمكنك إضافة البطاقات أو إزالتها أو نقلها لتنظيم لوحة المعلومات حسب حاجتك.

تنقسم البطاقات إلى ثلاث فئات رئيسية:

#### Product analytics

تساعدك *Product Analytics* على تتبّع وتحليل تفاعل المستخدمين وسلوكهم داخل منتجك.

| **البطاقة**    | **الوصف**                                                   |
|-------------------|------------------------------------------------------------------|
| **Trends**        | راقب اتجاهات الجلسات عبر الزمن                                |
| **Funnels**       | اعرض تقدّم المستخدمين عبر الخطوات الحرجة وحدّد نقاط التسرّب لتحسين التحويل               |
| **Journeys**      | افهم المسارات التي يسلكها المستخدمون عبر منتجك            |
| **Heatmaps**      | اعرض النقاط الساخنة وأنماط تفاعل المستخدمين              |

#### Monitors

تساعدك *Monitors* على تحديد مشكلات الأداء التقنية التي تؤثّر في تجربة المستخدم ومعالجتها.

| **البطاقة**           | **الوصف**                                                    |
|--------------------------|-------------------------------------------------------------------|
| **JS Errors**            | تتبّع أخطاء JavaScript التي تؤثّر في تجربة المستخدم             |
| **Top Network Requests** | حدّد موارد الشبكة التي يتم الوصول إليها بكثرة                    |
| **4xx/5xx Requests**     | راقب أخطاء العميل والخادم لاكتشاف مشكلات الأداء           |
| **Slow Network Requests**| حدّد التأخيرات الناتجة عن الاستجابات البطيئة                          |

#### Web Analytics

توفّر *Web analytics* رؤى حول البيانات الديموغرافية للمستخدمين ومصادر الزيارات وأنماطها.


| **البطاقة**               | **الوصف**                                                   |
|------------------------------|------------------------------------------------------------------|
| **Top Pages**               | حدّد الصفحات الأكثر زيارةً على موقعك                      |
| **Top Browsers**            | اطّلع على المتصفحات التي يعتمد عليها مستخدموك أكثر من غيرها                        |
| **Top Referrers**           | افهم من أين تأتي زياراتك                          |
| **Top Users**               | اعثر على المستخدمين الأكثر تفاعلاً                             |
| **Top Countries** | حلّل جمهورك حسب الموقع الجغرافي              |
| **Top Devices** | حلّل جمهورك حسب نوع الجهاز             |



## إنشاء أول لوحة معلومات لك

الخطوة 1: انتقل إلى Dashboards، ثم انقر على زر **Create dashboard**.

الخطوة 2: اختر إحدى الفئات الثلاث. حدّد نوع البطاقة التي تريد إضافتها لـ **البدء من الصفر**، أو انقر على **Add existing Card** لإعادة استخدام بطاقة موجودة.

الخطوة 3: انقر على **Create Card** لإضافتها إلى لوحة المعلومات

الخطوة 4: لإضافة بطاقة أخرى، انقر على زر "+" بجوار بطاقة موجودة.

## حالات الاستخدام الشائعة

لوحات المعلومات مرنة ويمكن تكييفها لأغراض متنوعة.

إليك بعض الأمثلة:

- **تحليل سلوك المستخدمين:** استخدم Trends وJourneys لفهم كيفية تفاعل المستخدمين مع منتجك.
- **تحديد المشكلات:** راقب JS Errors و4xx/5xx Requests وSlow Network Requests لمعالجة مشكلات الأداء.
- **فهم الزيارات:** تتبّع Top Pages وReferrers والبيانات الديموغرافية للمستخدمين لمعرفة ما ينجح وأين توجد حاجة إلى التحسين.

أنشئ لوحات معلومات منفصلة لأهداف محددة، مثل مراقبة الحملات التسويقية أو تتبّع الأداء التقني. يمكن أن تعمل لوحات المعلومات كعروض مختلفة لنظامك وأن توفّر ملخصًا متكاملاً لما يحدث.
