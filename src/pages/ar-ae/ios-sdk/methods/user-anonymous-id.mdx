اربط معرّف مستخدم مجهول الهوية بالجلسة التي يجري تسجيلها.

## التوقيع
```swift
ORTracker.shared.userAnonymousID(_ userID: String)
```

## المعاملات

- `userID: String`: المعرّف المجهول الخاص بالمستخدم. وهو عبارة عن سلسلة نصية ويمكن أن يكون أي شيء طالما أنه سلسلة نصية، ويتيح لك تحديد هوية المستخدم بشكل فريد داخل نظامك (مثل البريد الإلكتروني، أو معرّف قاعدة البيانات، وما إلى ذلك).

## القيمة المُرجَعة

لا تُرجِع هذه الطريقة أي قيم، وستكون النتائج مرئية في منصة OpenReplay.
