## إنشاء مشروع

أنشئ مشروعًا جديدًا عن طريق تحديد `name` الخاص به.

### الطريقة
`POST`

### عنوان URL
`/api/v1/projects`

### المعاملات

| الاسم | النوع | الوصف |
|----------|-------------|-------------|
| name | string | اسم المشروع الذي تقوم بإنشائه (مثل MyApp أو Production أو Staging) (مطلوب) |

### ترويسات الطلب

`Authorization: {YOUR_ORGANIZATION_API_KEY}`: يمكن العثور على المفتاح في 'Preferences' > 'Account' > 'Organization API Key'.

### رموز الحالة

`200`: تُرسَل الاستجابة بصيغة JSON في النص

### مثال على الطلب

```shell
curl -X POST \
  https://openreplay.example.com/api/v1/projects/MyFirstProject \
  -H 'content-type: application/json' \
  -H 'Authorization: {YOUR_ORGANIZATION_API_KEY}'
  --data-raw '{
    "name": "MyFirstProject"
}'
```

### مثال على الاستجابة

```json
{
  "data": {
    "projectKey": "QsLA07Oa5sqTfwnNaFwS",
    "name": "MyFirstProject",
    "gdpr": {
      "maskEmails": true,
      "sampleRate": 50,
      "maskNumbers": true,
      "defaultInputMode": "plain"
    }
  }
}
```

## استرداد مشروع

تُرجِع تفاصيل المشروع المحدد.

### الطريقة
`GET`

### عنوان URL
`/api/v1/projects/:projectKey`

### المعاملات

| الاسم | النوع | الوصف |
|----------|-------------|-------------|
| projectKey | string | معرّف المشروع الذي تتتبعه (مطلوب) |

### ترويسات الطلب

`Authorization: {YOUR_ORGANIZATION_API_KEY}`: يمكن العثور على المفتاح في 'Preferences' > 'Account' > 'Organization API Key'.

### رموز الحالة

`200`: تُرسَل الاستجابة بصيغة JSON في النص

### مثال على الطلب

```shell
curl -X GET \
  https://openreplay.example.com/api/v1/projects/7ePSXFuQVidq9pqS6Xyn \
  -H 'content-type: application/json' \
  -H 'Authorization: {YOUR_ORGANIZATION_API_KEY}'
```

### مثال على الاستجابة

```json
{
  "data": {
    "projectKey": "7ePSXFuQVidq9pqS6Xyn",
    "name": "MyFirstProject"
  }
}
```
