باستخدام هذه الطريقة، يمكنك تعيين قيمة أحد حقول البيانات الوصفية المُهيّأة على منصة OpenReplay.

<Aside type="note">
	يمكنك استدعاء هذه الطريقة أي عدد من المرات تشاء، فقط ضع في اعتبارك أنه إذا استخدمت السمة `key` نفسها، فسيتم حفظ آخر قيمة مرّرتها في المنصة. وستُفقد جميع القيم السابقة.
</Aside>

راجع [البيانات الوصفية](/installation/metadata) لمزيد من التفاصيل.

## التوقيع
```js
import OR from '@openreplay/react-native'

// ...
OR.tracker.metadata('key', 'value')
```

## المعاملات

- `key: string`: مفتاح البيانات الوصفية المراد حفظه.
- `value: string`: القيمة المراد حفظها كبيانات وصفية.

## القيمة المُرجَعة

لا تُرجع هذه الطريقة أي قيم. تُحفظ المعلومات مباشرةً كجزء من إعادة التشغيل ويمكن رؤيتها داخل منصة OpenReplay.
