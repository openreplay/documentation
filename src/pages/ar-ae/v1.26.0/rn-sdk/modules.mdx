تصدّر حزمة تطوير React Native عدة وحدات تتيح لك تخصيص التسجيل الخاص بك:

## الوحدات

- [ORTouchTrackingView](/rn-sdk/touch): عرض يُستخدم لتتبّع أحداث اللمس.
- [ORSanitizedView](/rn-sdk/sanitization): عرض يتيح لك تنقية (أو طمس) أجزاء من التسجيل الخاص بك
- [ORTrackedInput](/rn-sdk/inputs): TextInput يسجّل أحداث الإدخال
- [ORAnalyticsView](/rn-sdk/analytics): عرض مُعلَّم يرسل حدثًا بمجرد ظهوره على الشاشة وعند مغادرته لها
