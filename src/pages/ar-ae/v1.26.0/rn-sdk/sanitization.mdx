## مكوّنات واجهة المستخدم

ستظهر طرق العرض المُنقّاة مموّهة في إعادة التشغيل الناتجة. يمكنك إضافة أي مكوّن من مكوّنات واجهة المستخدم وسيُزال تلقائيًا (للحفاظ على الذاكرة) بمجرد أن يختفي عن الشاشة.

### ORSanitizedView

تُصدّر حزمة التطوير `ORSanitizedView` التي يمكنك استخدامها لتغليف مكوّناتك الحسّاسة بهدف تمويهها.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORSanitizedView>
    <SensitiveComponent />
  </OR.ORSanitizedView>

// ...
```
