تتيح لك هذه الطريقة تعيين معرّف مستخدم للجلسة النشطة حاليًا. وهذا يسمح لك بربط مستخدم بإعادة العرض، مما يتيح لك تصفية الجلسات والبحث عنها لمستخدم معيّن.

## التوقيع
```swift
ORTracker.shared.setUserID(_ userID: String)
```

## المعاملات

- `userID: String`: معرّف المستخدم. وهو سلسلة نصية ويمكن أن يكون أي شيء طالما أنه سلسلة نصية، ويتيح لك تحديد هوية المستخدم بشكل فريد داخل نظامك (مثل البريد الإلكتروني، معرّف قاعدة البيانات، إلخ).

## القيمة المُرجَعة

لا تُرجِع هذه الطريقة أي قيم، وستظهر النتائج في منصة OpenReplay.
