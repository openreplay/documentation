OpenReplay is supported in the following browsers:
- Google Chrome 41+
- Firefox 17+
- Microsoft Edge 12+
- Internet Explorer 11+
- Safari 9+
- Safari on iOS 9+
- Opera 28+
- Opera Mobile 62+

If you are recording a session in an unsupported browser, the SDK will throw an exception.
