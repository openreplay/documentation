OpenReplay introduces **AI-Native Experiences** for Session Replay, making it easier than ever to analyze sessions, detect issues, and gain insights without manual effort. AI accelerates issue detection, user behavior analysis, and insights generation by leveraging different models for specific tasks.

<Aside type="ai" title="Note" >
  - Limited to **Cloud (Serverless)** users at this time.
  - In the future, Private Cloud Compute will allow **Self-hosted** and **Cloud (Dedicated)** users to access these AI feature suite.
</Aside>


## AI Feature Suite

- [Summary AI](/ai/summary-ai): Get concise session summaries, providing a concise overview of key interactions and events.
- [Similar Sessions](/ai/similar-sessions): One-click feature to find session replays similar to the one currently being played, using AI-driven embedding models.
- [Explain AI](/ai/explain-ai): Understand network requests and console logs (specifically errors) within DevTools in every session replay.
- [Smart Search](/ai/smart-search): Use AI-powered search bar to write queries in natural language and get relevant filters for finding specific sessions.
- [Clips](/ai/clips): Short clips highlighting key moments from session replays, using AI-driven embedding models.
