When upgrading your OpenReplay instance, it's essential to ensure that your tracker (and tracker-assist plugin if you use it) is compatible with the backend version.

## Compatibility Table

Use the following table to determine which tracker and assist versions to use with your backend:

| Backend Version | Tracker Version | Assist Version |
|:---------------:|:--------------:|:-------------:|
| 1.23.0 | 17.x.x | 11.x.x |
| 1.22.0 | 16.x.x | 11.x.x |
