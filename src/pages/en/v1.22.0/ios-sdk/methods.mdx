## Methods

- [event()](/ios-sdk/methods/event): Send custom events to be recorded as part of your replays with this method.
- [setUserID()](/ios-sdk/methods/set-user-id): Identify your users with this method.
- [userAnonymousID()](/ios-sdk/methods/user-anonymous-id): Associate an anonymous userID with the session being recorded.
- [setMetadata()](/ios-sdk/methods/set-metadata): Set a metadata field and save it as part of the recording.
- [start()](/ios-sdk/methods/start): Start recording a session.
- [stop()](/ios-sdk/methods/stop): Stop the recording of a session with this method.
