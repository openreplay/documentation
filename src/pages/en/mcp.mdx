The **OpenReplay MCP** app connects your OpenReplay instance to an MCP-enabled chat host (Claude Desktop, Claude Code, ChatGPT, Codex) so you can explore product analytics and session replays in plain English. Ask for a chart, drill into a funnel, find a session, then open its replay — all from inside the chat.

It works against both **OpenReplay Cloud** and **self-hosted** instances.

<Aside type="tip" title="What you get" >
  - Search sessions with the same filters you use in the OpenReplay UI.
  - Render timeseries, funnels, user journeys, web vitals, and top-of-X breakdowns inline.
  - Watch session replays directly in the chat.
  - Chain it all together: search → pick a session → open replay → ask Claude to summarize.
</Aside>

## Compatibility

The app speaks standard MCP over stdio, so it works with:

- **Claude Desktop** (macOS, Windows)
- **Claude Code**
- **ChatGPT** (via the MCP connector)
- **Codex**
- Any other host that supports MCP
