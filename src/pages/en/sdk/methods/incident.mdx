You can record the incident to have a mark about this event in the replay.

## Signature

```js 
incident: ({ label?: String, startTime: Number, endTime?: Number }) => void
```

## Parameters

- `label: String`: **[Optional]** Text describing the incident.
- `startTime: Number`: Incident start timestamp.
- `endTime: Number`: **[Optional]** Incident end timestamp.

## Return value

This method doesn't return any values, the saved information can be seen inside the OpenReplay platform.
