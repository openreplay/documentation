## Methods

- [event()](/android-sdk/methods/event): Send custom events to be recorded as part of your replays with this method.
- [setUserID()](/android-sdk/methods/set-user-id): Identify your users with this method.
- [setMetadata()](/android-sdk/methods/set-metadata): Set a metadata field and save it as part of the recording.
- [start()](/android-sdk/methods/start): Start recording a session.
