Associate an anonymous userID with the session being recorded.

## Signature

```swift
ORTracker.userAnonymousID(userID: String)
```

## Parameters

- `userID: String`: The anonymous ID of the user. This is a string and can be anything as long as it's a string and it allows you to uniquely identify a user within your system (i.e. email, database id, etc).

## Return value

This method doesn't return any values, the results will be visible in the OpenReplay platform.
