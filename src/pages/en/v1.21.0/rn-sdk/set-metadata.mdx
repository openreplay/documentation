With this method, you can set the value of one of the metadata fields configured on the OpenReplay platform.

<Aside type="note">
	You may call this method as many times as you wish, just keep in mind that if you use the same `key` attribute, the last value you passed will be saved in the platform. All previous values will be lost.
</Aside>

See [metadata](/installation/metadata) for more details.

## Signature
```js
import OR from '@openreplay/react-native'

// ...
OR.tracker.metadata('key', 'value')
```

## Parameters

- `key: string`: The metadata key to save.
- `value: string`: The value to save as metadata.

## Return Value

This method does not return any values. The information is saved directly as part of the replay and can be seen inside the OpenReplay platform.
