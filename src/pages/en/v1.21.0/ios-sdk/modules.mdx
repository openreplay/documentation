## Methods and modules

- [Analytics](/ios-sdk/analytics): Learn all about tracking specific views.
- [Network](/ios-sdk/network): Learn how to utilize the NetworkListener.
- [Sanitization](/ios-sdk/sanitization): Learn how to sanitize views on recorded screen.
- [Inputs](/ios-sdk/inputs): Learn how to record input events.
- [event()](/ios-sdk/event): Send custom events to be recorded as part of your replays with this method.
- [setUserID()](/ios-sdk/set-user-id): Identify your users with this method.
- [setMetadata()](/ios-sdk/set-metadata): Set a metadata field and save it as part of the recording.
- [start()](/ios-sdk/start): Start recording a session.
- [stop()](/ios-sdk/stop): Stop the recording of a session with this method.
