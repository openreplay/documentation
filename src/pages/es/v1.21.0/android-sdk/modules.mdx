## Métodos y módulos

- [Analytics](/android-sdk/analytics): Aprende todo sobre el seguimiento de vistas específicas.
- [Network](/android-sdk/network): Aprende a utilizar el NetworkListener.
- [Sanitization](/android-sdk/sanitization): Aprende a sanear vistas en la pantalla grabada.
- [Inputs](/android-sdk/inputs): Aprende a registrar eventos de entrada.
- [event()](/android-sdk/event): Envía eventos personalizados para que se registren como parte de tus reproducciones con este método.
- [setUserID()](/android-sdk/set-user-id): Identifica a tus usuarios con este método.
- [setMetadata()](/android-sdk/set-metadata): Establece un campo de metadatos y guárdalo como parte de la grabación.
- [start()](/android-sdk/start): Inicia la grabación de una sesión.
- [stop()](/android-sdk/stop): Detén la grabación de una sesión con este método.
