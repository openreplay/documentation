Con este método, puedes establecer el valor de uno de los campos de metadatos configurados en la plataforma de OpenReplay.

<Aside type="note">
	Puedes llamar a este método tantas veces como quieras, solo ten en cuenta que si usas el mismo atributo `key`, el último valor que pasaste se guardará en la plataforma. Todos los valores anteriores se perderán.
</Aside>

Consulta [metadatos](/installation/metadata) para más detalles. 

## Firma
```kotlin
OpenReplay.setMetadata(key: String, value: String)
```

## Parámetros

- `key: string`: La clave de metadatos que se va a guardar. 
- `value: string`: El valor que se va a guardar como metadato.

## Valor de retorno

Este método no devuelve ningún valor. La información se guarda directamente como parte de la repetición y puede verse dentro de la plataforma de OpenReplay.
