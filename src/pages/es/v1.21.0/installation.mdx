Esta sección ofrece tutoriales categorizados que te ayudarán a configurar y ajustar OpenReplay, desde la instalación básica hasta configuraciones avanzadas.

## Configuración de usuarios

- [Cómo identificar a un usuario](/installation/identify-user): Asocia tu ID de usuario interno con la sesión que se está grabando.
- [Cómo enviar metadatos](/installation/metadata): También conocidos como rasgos o variables de usuario, los metadatos proporcionan información adicional sobre los usuarios mientras se graban las sesiones.

## Configuración de la repetición de sesiones

- [Cómo enviar eventos personalizados](/installation/custom-events): Añade más inteligencia registrando eventos específicos de tu dominio junto con las repeticiones de sesiones.
- [Cómo configurar Canvas y WebGL](/installation/canvas): Aprende a integrar y configurar Canvas y WebGL en tu proyecto.
- [Cómo configurar iFrames entre dominios](/installation/crossdomain-iframe): Configura iFrames entre dominios para una integración sin complicaciones.
- [Cómo configurar WebSockets](/installation/websockets): Configura WebSockets para la comunicación en tiempo real en tu aplicación.

## Redes y privacidad de datos

- [Cómo personalizar las opciones de red (Web)](/installation/network-options): Personaliza las configuraciones de red para tu entorno web.
- [Cómo personalizar las opciones de red (Móvil)](/installation/network-options-mobile): Personaliza las configuraciones de red para tus entornos iOS y Android.
- [Cómo sanear datos (Web)](/installation/sanitize-data): Cómo sanear datos a nivel global y/o granular para tus aplicaciones web.
- [Cómo sanear datos (Móvil)](/installation/sanitize-data-mobile): Cómo sanear datos para aplicaciones iOS y Android.

## Depuración y seguimiento de errores

- [Cómo realizar el seguimiento de errores](/installation/error-reporting): Realiza el seguimiento de errores y excepciones y soluciona problemas con contexto completo.
- [Cómo subir Source-Maps](/installation/upload-sourcemaps): Facilita la depuración viendo el contexto del código fuente obtenido de las trazas de pila en su forma original.
