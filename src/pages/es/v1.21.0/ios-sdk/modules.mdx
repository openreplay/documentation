## Métodos y módulos

- [Analytics](/ios-sdk/analytics): Aprende todo sobre el seguimiento de vistas específicas.
- [Network](/ios-sdk/network): Aprende cómo utilizar el NetworkListener.
- [Sanitization](/ios-sdk/sanitization): Aprende cómo sanear las vistas en la pantalla grabada.
- [Inputs](/ios-sdk/inputs): Aprende cómo registrar los eventos de entrada.
- [event()](/ios-sdk/event): Envía eventos personalizados para que se registren como parte de tus repeticiones con este método.
- [setUserID()](/ios-sdk/set-user-id): Identifica a tus usuarios con este método.
- [setMetadata()](/ios-sdk/set-metadata): Establece un campo de metadatos y guárdalo como parte de la grabación.
- [start()](/ios-sdk/start): Inicia la grabación de una sesión.
- [stop()](/ios-sdk/stop): Detén la grabación de una sesión con este método.
