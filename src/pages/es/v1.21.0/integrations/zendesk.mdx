Cómo integrar Zendesk con OpenReplay.

## Instalación

La forma más sencilla de integrar Zendesk con OpenReplay es usar la [aplicación Text](https://www.zendesk.com/marketplace/apps/support/1691/text/) para añadir un enlace a las sesiones del usuario en tus páginas de tickets.

1. Instala la [aplicación Text](https://www.zendesk.com/marketplace/apps/support/1691/text/) gratuita desde el marketplace de Zendesk.
2. Especifica un título de tu preferencia (p. ej. "OpenReplay Recordings").
3. Para la visualización del texto, puedes seguir este formato asegurándote de cambiar `OPENREPLAY_DOMAIN` y `PROJECT_ID` por el Dominio y el Project ID válidos (si no estás seguro de qué valores elegir, revisa la URL de cualquiera de tus grabaciones existentes; la primera parte es el dominio y el segundo número es el Project ID):

```
https://<OPENREPLAY_DOMAIN>/<PROJECT_ID>/sessions?uid=is|{{ticket.requester.email}}&range=LAST_24_HOURS
```

4. Para ver el enlace, ve a un ticket y abre el panel de Apps haciendo clic en el botón Apps en la parte superior derecha. Consulta la [documentación de la aplicación Text](https://www.zendesk.com/marketplace/apps/support/1691/text/) para más detalles.

## Cambiar el formato

Puedes probar el formato de tu enlace yendo a tu instancia de OpenReplay y cambiando los criterios de búsqueda, empezando por el filtro UserID y luego añadiendo cualquier evento importante. Por ejemplo, la siguiente consulta busca todas las sesiones que tuvieron una solicitud incorrecta y que están asociadas con el correo electrónico del usuario del ticket:

```
https://<OPENREPLAY_DOMAIN>/<PROJECT_ID>/sessions?uid=is|{{ticket.requester.email}}&iss=is|bad_request&range=LAST_24_HOURS
```
