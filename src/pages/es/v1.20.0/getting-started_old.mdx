INTRODUCCIÓN
:::tip[¿Sabías que?]
 Astro te ayuda a crear sitios web más rápidos con la “Arquitectura de Islas”.
:::

:::caution[Uso con pnpm]
Debido a problemas upstream dentro de Prettier, el plugin no se detectará automáticamente al usar [pnpm](https://pnpm.io/). Para que lo encuentre, es necesario añadir el siguiente parámetro al ejecutar Prettier:

```shell
prettier --write --plugin-search-dir=. .
```

También se requieren ajustes adicionales al usar Prettier dentro de VS Code. Consulta el README del plugin para obtener más información.
:::
