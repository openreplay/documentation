Este método vuelve a cargar todos los flags (puede ser útil si la sesión del usuario tuvo algún cambio en los metadatos, el userId o cualquier otra información que pudiera ser necesaria para ciertas condiciones de los flags).

## Firma

```js
reloadFlags(): Promise<void>
```

## Valor de retorno

Devuelve una promesa vacía, ejecuta onFlagsCb cuando se completa correctamente
