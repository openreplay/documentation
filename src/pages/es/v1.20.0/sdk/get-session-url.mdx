Este método te permite obtener la URL de la sesión activa. La URL corresponde al panel de OpenReplay y este método resulta útil si buscas integrar OpenReplay en tu propio stack.

## Firma

```js
getSessionURL({ withCurrentTime?: boolean }): string | undefined
```

## Parámetros

- `withCurrentTime: boolean`: Parámetro opcional, por defecto es `false`. Si es `true`, la URL contendrá la marca de tiempo del momento dentro de la reproducción en el que se llamó al método.

## Valor de retorno

Este método devuelve la URL de la sesión activa para reproducirla en la plataforma de OpenReplay. Opcionalmente, la URL puede incluir la marca de tiempo del momento en que se llama al método para saltar directamente a un punto más adelante en la reproducción.
