Con este método, puedes establecer el valor de uno de los campos de metadatos configurados en la plataforma de OpenReplay.

<Aside type="note">
Puedes llamar a este método tantas veces como quieras, solo ten en cuenta que si usas el mismo atributo `key`, el último valor que hayas pasado será el que se guarde en la plataforma. Todos los valores anteriores se perderán.
</Aside>

Consulta [metadata](/installation/metadata) para más detalles. 

## Firma
```js
setMetadata(key: string, value: string): void
```

## Parámetros

- `key: string`: La clave de metadatos a guardar. 
- `value: string`: El valor a guardar como metadato.

## Valor de retorno

Este método no devuelve ningún valor. La información se guarda directamente como parte de la grabación y se puede ver dentro de la plataforma de OpenReplay.
