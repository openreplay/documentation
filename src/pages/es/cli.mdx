A partir de la v1.10.0 hemos lanzado nuestra utilidad CLI. Con ella, las tareas más habituales que tendrías que ejecutar para gestionar tu versión autoalojada de OpenReplay ahora se pueden realizar con un simple comando.


## Instalación de la CLI

La CLI viene incluida con cada versión autoalojada de OpenReplay.

## Uso de la CLI

La CLI tiene la siguiente sintaxis:

```bash 
openreplay [command] <service>
```

La lista de comandos disponibles es:

- `-h or --help`: Este comando mostrará la lista de comandos y servicios disponibles.
- `-s or --status`: Si quieres conocer el estado de los servicios desplegados, este comando te dirá todo lo que necesitas saber.
- `-u or --upgrade`: Se utiliza para actualizar tu instalación a la última versión disponible.
- `-U or --deprecated-upgrade /path/to/old_vars.yaml`
- `-r or --restart`: Te permite reiniciar todos los servicios. Al hacerlo, puedes solucionar los servicios que no responden.
- `-R or --Reload`: Recarga la configuración de un servicio, así que úsalo solo cuando cambies algo.
- `-p or --install-packages`: Actualiza todas las dependencias de terceros (como `kubectl`, `helm`, etc).
- `-i or --install domain.name.com`: Instala OpenReplay en la máquina.
- `-e or --edit`: Edita la configuración instalada de OpenReplay y la recarga.
- `-l or --logs SERVICE`: Muestra los registros de un servicio en particular.

## Ejemplo

Para obtener el estado del despliegue, ejecuta el siguiente comando de la CLI:

```bash 
openreplay -s 
```
