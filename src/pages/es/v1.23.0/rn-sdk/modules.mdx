El SDK de React Native exporta varios módulos para que personalices tu grabación:

## Módulos

- [ORTouchTrackingView](/rn-sdk/touch): Vista utilizada para rastrear eventos táctiles.
- [ORSanitizedView](/rn-sdk/sanitization): Vista que te permite sanear (o difuminar) partes de tu grabación
- [ORTrackedInput](/rn-sdk/inputs): TextInput que registra eventos de entrada
- [ORAnalyticsView](/rn-sdk/analytics): Vista marcada que envía un evento una vez que se ve en la pantalla y cuando sale de ella
