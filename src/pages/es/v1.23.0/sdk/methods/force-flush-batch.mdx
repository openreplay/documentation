Obliga al tracker a vaciar el lote actual de mensajes sin esperar a que se complete. Resulta útil cuando necesitas cerrar la pestaña o la ventana después de una sesión. Este método también envía un mensaje al webworker, que iniciará la solicitud de red con el lote actual de mensajes.

## Firma

```ts
forceFlushBatch(): void
```
