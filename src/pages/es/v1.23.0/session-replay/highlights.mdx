Los **Highlights** ofrecen una forma estructurada de aislar y guardar partes significativas de la repetición de una sesión. Ya sea una aparición de error, una interacción del usuario o un evento destacado, los Highlights garantizan que estos momentos sean fácilmente accesibles para una revisión futura, la colaboración o la elaboración de informes.

## Casos de uso

Al revisar la repetición de una sesión, usa los **Highlights** para:

1. Aislar y guardar la parte donde un usuario encontró un error (por ejemplo: una llamada a la API fallida, un error de consola…).
2. Identificar momentos en los que el comportamiento del usuario revela problemas de usabilidad o puntos de fricción.
3. Aportar contexto para informar mejoras de diseño o futuras iteraciones.

## Cómo funciona: crear y guardar un highlight

**Paso 1:** Abre la repetición de una sesión, localiza la parte clave y selecciona el botón **'Highlights'** en la barra de menú.

## Gestión de los Highlights

#### Acceder a los Highlights guardados

Abre la pestaña **Highlights** para ver todas las entradas guardadas. Los Highlights se pueden filtrar por:

- Etiquetas (Issue, Design, Note).
- Creador (My Highlights o Team Highlights).
- Términos de búsqueda (por ejemplo, el contenido de la nota).

<div style={{ display: 'flex', justifyContent: 'center', alignItems: 'center' }}>
  <img src="/images/products/session-replay/highlights/highlight_view.png" alt="highlights" style={{ width: '700px', height: 'auto' }} />
</div>

#### Reproducir la repetición desde la parte resaltada

Reproduce la sesión a partir de la parte guardada. Usa el botón **Navigate to Full Session** para revisar la sesión completa desde la marca de tiempo correspondiente.

<div style={{ display: 'flex', justifyContent: 'center', alignItems: 'center' }}>
  <img src="/images/products/session-replay/highlights/highlight_manage.gif" alt="highlights" style={{ width: '800px', height: 'auto' }} />
</div>

<Aside type="tip">
    Los highlights guardados se marcan en la _**línea de tiempo**_ de la repetición de la sesión y son accesibles en la pestaña _**Activity**_.
</Aside>

## Colaboración y uso compartido

- **Ajustar la visibilidad:**

    Define si un highlight es visible solo para ti (Private) o para tu equipo (Team).
    
- **Compartir Highlights:**
    
    Usa la opción **Copy Link** para compartir un highlight concreto con tus colegas.
    
- **Editar o eliminar Highlights:**
    
    Modifica las notas o elimina el highlight si ya no es relevante.

## **Saca el máximo partido a los Highlights**

1. Céntrate en guardar solo las partes más relevantes de las repeticiones de sesión.
2. Usa notas y etiquetas para aportar contexto y categorizar los highlights para una consulta rápida.
3. Comparte los highlights con tu equipo para abordar rápidamente los problemas o informar mejoras de diseño.
