Este método se utiliza para iniciar el tracker. Al iniciar el tracker, comienza la grabación de la sesión.

## Firma

```swift
ORTracker.shared.start(projectKey: String, options: OROptions)
	```

### Opciones de inicialización

Opciones disponibles:

- `crashes: Bool` Habilita crashlytics.
- `analytics: Bool` Habilita la analítica para las vistas.
- `performances: Bool` Habilita el listener de rendimiento
- `logs: Bool` Habilita el listener de logs
- `screen: Bool` Habilita la grabación de pantalla
- `wifiOnly: Bool` Obliga al tracker a iniciarse solo si el usuario tiene conexión wifi

Por defecto: todas en `true`
