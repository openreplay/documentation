## Métodos

- [event()](/android-sdk/methods/event): Envía eventos personalizados para que se registren como parte de tus repeticiones con este método.
- [setUserID()](/android-sdk/methods/set-user-id): Identifica a tus usuarios con este método.
- [setMetadata()](/android-sdk/methods/set-metadata): Establece un campo de metadatos y guárdalo como parte de la grabación.
- [start()](/android-sdk/methods/start): Inicia la grabación de una sesión.
