## Guías prácticas

Aprende de los tutoriales a continuación usando casos de uso del mundo real.

### Plugins
- [Crea tus propios plugins](/tutorials/build-plugins): Aprende a crear tus propios plugins personalizados en tu instancia autoalojada de OpenReplay.
- [Assist](/tutorials/assist): Ofrece la mejor experiencia de soporte e incorporación a tus usuarios finales mediante la reproducción de sesiones en vivo.
- [Plugin Fetch](/tutorials/fetch): Aprende a configurar y sanear la información de las solicitudes usando el plugin Fetch.
- [Plugin Axios](/tutorials/axios): Aprende a configurar y sanear la información de las solicitudes usando el plugin Axios.
- [Plugin Redux](/tutorials/redux): Aprende a configurar el plugin Redux para capturar los cambios de estado en tus reproducciones de sesión.
- [Plugin VueX](/tutorials/vuex): Aprende a configurar el plugin VueX para capturar los cambios de estado en tus reproducciones de sesión.
- [Plugin GraphQL](/tutorials/graphql): Aprende a configurar el plugin GraphQL para capturar los datos de operaciones en tus reproducciones de sesión.
- [Plugin Zustand](/tutorials/zustand): Seguimiento de los datos de estado de Zustand con el plugin Zustand Tracker.
- [Plugin Pinia](/tutorials/pinia): Seguimiento de los datos de estado de Pinia con el plugin Pinia Tracker.

- [Tipos de problemas](/tutorials/issues): Aprende qué significan todos los problemas detectados automáticamente y cómo aprovecharlos al máximo.
- [Mejora tu capacidad de seguimiento con Eventos personalizados](/tutorials/custom-events): Aprende sobre los eventos personalizados y cómo puedes potenciar tus superpoderes de seguimiento.
- Paneles:
   - [Creación de paneles personalizados](/tutorials/custom-dashboard): Aprende a aprovechar los paneles personalizados para visualizar el estado actual de tu sistema.
   - [Creación de un embudo personalizado](/tutorials/funnels): Aprende a crear embudos personalizados para comprender qué está causando las caídas en tu conversión.
   - [Mostrar errores de JavaScript en un panel personalizado](/tutorials/js-errors-dashboard): Agrega una métrica personalizada (errores de JS) a tus paneles con este tutorial.
