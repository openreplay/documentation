Esta función se puede usar para determinar si una instancia concreta del rastreador está activa y grabando o no.

## Firma

```js
isActive(): boolean
```

### Valor de retorno

Devuelve `true` si el rastreador está activo y la sesión se está grabando, y `false` en caso contrario.
