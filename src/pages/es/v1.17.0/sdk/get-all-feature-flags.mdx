Este método devuelve todas las banderas (`IFeatureFlag[]`) que están activas en este momento.

## Firma

```js
getAllFeatureFlags(): IFeatureFlag[]
```

## Valor de retorno

Array de todas las feature flags activas (IFeatureFlag)
