Si la información que nuestro tracker registra de forma predeterminada cuando ocurre un error de JS no es suficiente, puedes usar este método. El método `handleError` informa manualmente las excepciones capturadas, las promesas rechazadas o los eventos de error. Consulta [informes de errores](/installation/error-reporting#manuallyloggingexceptions) para más detalles.

## Firma

```ts 
trackWs: (channelName: string) => (msgType: string, data: string, dir: 'up' | 'down') => void
```

## Parámetros

- `channelName: string`: Nombre del canal WS.

## Valor de retorno

Devuelve un hook que se aplica a los mensajes ws y que toma como parámetros el tipo de mensaje (o nombre), los datos y la dirección del mensaje (up/down === enviado/recibido).
