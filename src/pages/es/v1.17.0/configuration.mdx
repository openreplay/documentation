Tu instancia de OpenReplay se puede configurar fácilmente. Aquí tienes tutoriales paso a paso para habilitar los servicios de correo electrónico, proteger tu instalación y eliminar grabaciones de la base de datos.

## Guías paso a paso
- [Configurar SMTP](/configuration/configure-smtp): Configura SMTP para recibir alertas, informes semanales, invitar a nuevos usuarios a OpenReplay y restablecer contraseñas.
- [Proteger OpenReplay](/configuration/secure-or): Protege OpenReplay configurando SSL, reCaptcha y estableciendo una CSP.
- [Configuración del proxy](/configuration/proxy-settings): Ejecuta OpenReplay detrás de un proxy como NGINX o Apache2.
- [Limpiar el almacenamiento](/configuration/cleanup-storage): Cómo se almacenan las grabaciones y cómo eliminarlas en masa de la base de datos y limpiar tu almacenamiento.
- [Almacenamiento externo de grabaciones](/configuration/external-storage): Cambia el destino de almacenamiento predeterminado de tus grabaciones.
- [Base de datos externa](/configuration/external-db): Usa una base de datos Postgres externa para OpenReplay.
- [Inicio de sesión único (SAML)](/configuration/sso): Configura el inicio de sesión único (SSO) mediante SAML.
