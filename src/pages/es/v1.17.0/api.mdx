La API de OpenReplay le permite gestionar datos de usuario y exportar los eventos capturados para su análisis externo.

## API HTTP
- [Proyectos](/api/projects): Para crear nuevos proyectos.
- [Sesiones](/api/sessions): Para recuperar todas las sesiones de un usuario específico.
- [Eventos](/api/events): Para extraer todos los eventos de una grabación concreta.
- [Usuarios](/api/users): Para controlar y eliminar datos de usuario.
- [Trabajos](/api/jobs): Para gestionar los trabajos de eliminación de datos de usuario.
- [Assist](/api/assist): Para recuperar sesiones en vivo.
