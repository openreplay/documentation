Integra GitHub con OpenReplay y crea issues directamente desde la página de grabación.

## Uso de OpenReplay Open Source

#### 1. Crea un token de acceso personal

1. Inicia sesión en Github y luego ve a [Settings > Developers Settings > Personal access tokens](https://github.com/settings/tokens)
2. Haz clic en 'Generate a new token'
3. Selecciona los scopes `repo:status`, `repo_deployment`,`public_repo`, `read:user` y `user:email`
4. Haz clic en 'Generate token' y luego copia el token

#### 2. Habilita GitHub en OpenReplay

1. Ve al panel de OpenReplay y luego a 'Preferences > Integration'
2. Selecciona GitHub. Si no lo ves, es porque debes tener habilitado Jira (no puedes usar ambos al mismo tiempo, así que asegúrate de deshabilitar primero la integración con Jira).
3. Pega el `token` que generaste anteriormente

Aparecerá entonces un botón 'Create Issue' en todas las grabaciones para que puedas crear y asignar issues rápidamente en GitHub.

## Uso de OpenReplay Cloud

Ve a 'Preferences > Integration' y haz clic en GitHub. Serás redirigido a GitHub para autenticarte. Eso es todo lo que tienes que hacer. Aparecerá entonces un botón 'Create Issue' en todas las grabaciones para que puedas crear y asignar issues rápidamente en GitHub.
