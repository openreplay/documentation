Si buscas añadir el tracker a tu aplicación web y estás usando un framework que no soportamos o simplemente estás
usando VanillaJS, puedes añadir directamente nuestro snippet de JavaScript.

## Obtener el snippet

El snippet de JavaScript que necesitarás para hacer el trabajo se te proporciona cuando configuras el proyecto en el portal de OpenReplay.
Una vez que hayas añadido tu nuevo proyecto, verás algo como esto:

![New project ready](/using-or/snippet/new-project-ready.png)

Cuando veas eso, haz clic en el botón "Complete Project Setup".
Eso debería llevarte a una página donde se te presenta el snippet de JS:


![Getting the JS Snippet](/using-or/snippet/getting-snippet.png)

Ahora, simplemente haz clic en la pestaña "Script" y verás el snippet de JavaScript real:

![The snippet is visible](/using-or/snippet/snippet-visible.png)

Obtén el script y añádelo directamente dentro de tu etiqueta `<head>` y el tracker comenzará a funcionar automáticamente.

<Aside type="tip">
Ten en cuenta que si cambias cualquier opción de grabación de datos en el paso 1 de la captura de pantalla anterior, el snippet final cambiará.
Así que asegúrate de configurar primero tus opciones de grabación y luego copiar el snippet real.
</Aside>
