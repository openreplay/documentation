<Aside type="danger">
    Este método está **obsoleto**, en su lugar use el [parámetro sessionHash del método `start`](/sdk/start).
</Aside>

## Firma

```js 
getSessionToken(): string | null | undefined
```

## Valor de retorno

Devuelve el token hash de la sesión.
