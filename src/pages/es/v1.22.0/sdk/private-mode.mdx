## Modo privado

El **Modo privado** es una nueva configuración que bloquea la grabación de todo el texto y los datos sensibles. Cuando está activado, el tracker de OpenReplay no capturará ningún texto visible, datos de red ni registros de la consola. Todo se oculta en el origen, de modo que nada sensible sale jamás del navegador.

Esto resulta útil si quieres tener visibilidad completa del comportamiento de la interfaz y, al mismo tiempo, mantener el contenido y los datos del usuario totalmente privados.

## Cómo activar el Modo privado

Puedes activar el Modo privado a nivel del tracker al inicializar OpenReplay.

#### Usando NPM

```js
import { tracker } from '@openreplay/tracker'

tracker.configure({
    projectKey: PROJECT_KEY,
    privateMode: true
})
tracker.start();
```

Cuando `privateMode` se establece en `true`, se ocultará lo siguiente:

- Todo el texto visible en la página  
- Los atributos `alt` de las imágenes  
- Los valores `placeholder` de los campos de entrada  
- Los textos de las etiquetas

No se requiere configuración adicional. Una vez activado, la sesión grabará el diseño y las interacciones, pero nada del contenido.


## Cuándo usarlo

Usa `privateMode` si:

- Necesitas garantizar la privacidad completa de los datos del usuario
- Estás probando OpenReplay en un **entorno en la nube**, pero planeas **autoalojarlo más adelante** y **no puedes enviar ningún dato real al exterior**
- Trabajas en sectores regulados donde **la privacidad es lo primero**
