A continuación se muestra la política (CSP) para permitir que OpenReplay grabe sesiones.

```html
worker-src ‘self’ blob: https://*.openreplay.com; script-src ‘self’ https://*.openreplay.com;
```

Ten en cuenta que la directiva `worker-src` no es compatible con Safari (ni con Safari móvil). Usarla no debería impedir que OpenReplay grabe sesiones.

<Aside type="tip">
 Esto es válido para la oferta de OpenReplay Cloud, pero la CSP también se puede habilitar en tu instalación autoalojada. Consulta [aquí](/configuration/secure-or/#content-security-policy-csp) para obtener más detalles.
</Aside>
