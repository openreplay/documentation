El rastreador escuchará el evento de cambio de valor (con debounce) y enviará el valor resultante al backend de OpenReplay.

## Añadir entradas rastreadas

### SwiftUI

usa `.observeInput(text:label:masked)` para observar los eventos de entrada,

- `text: String` - el valor de texto de la entrada
- `label: String` - etiqueta para la entrada registrada
- `masked: Bool` - si es `true`, el rastreador solo registrará el evento de entrada en sí, no el valor; no olvides añadir `.sensitive` para difuminar la entrada en la grabación.

```swift
import ORTracker

TextField("Input", text: $text)
.observeInput(text: $text, label: "tracker input #1", masked: Bool)

```

### UIKit

UIKit usará el placeholder como etiqueta y sender.isSecureTextEntry para enmascarar la entrada

```swift
import ORTracker

Analytics.shared.addObservedInput(inputEl)
```
