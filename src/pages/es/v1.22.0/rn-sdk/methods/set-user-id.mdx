---

Este método te permite establecer un ID de usuario en la sesión actualmente activa. Esto te permite asociar un usuario a la reproducción, lo que te permitiría filtrar y buscar sesiones de un usuario en particular.

## Firma

```js
import OR from '@openreplay/react-native'

// ...
OR.tracker.setUserID('id')
```


## Parámetros

- `userID: String`: El ID del usuario. Es una cadena de texto y puede ser cualquier valor siempre que sea una cadena y te permita identificar de forma única a un usuario dentro de tu sistema (es decir, correo electrónico, ID de base de datos, etc.).

## Valor de retorno

Este método no devuelve ningún valor, los resultados serán visibles en la plataforma de OpenReplay.
