## Componentes de UI

Las vistas sanitizadas aparecerán difuminadas en el replay resultante. Puedes añadir cualquier componente de UI y se eliminará automáticamente (para preservar memoria) en cuanto deje de estar en pantalla.

### ORSanitizedView

El SDK exporta `ORSanitizedView`, que puedes usar para envolver tus componentes sensibles y difuminarlos.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORSanitizedView>
    <SensitiveComponent />
  </OR.ORSanitizedView>

// ...
```
