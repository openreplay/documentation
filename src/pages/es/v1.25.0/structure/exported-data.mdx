OpenReplay Enterprise Edition incluye distintos conectores para exportar los datos capturados (comportamiento del usuario y problemas técnicos) a tu almacén de datos. Admitimos:
- Amazon Redshift
- Google BigQuery
- PostgreSQL
- Snowflake
- ClickHouse
