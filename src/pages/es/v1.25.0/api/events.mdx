## Recuperar eventos

Devuelve la lista de eventos, como clics, visitas a páginas o entradas, contenidos en una grabación de sesión concreta. Estos son eventos de alto nivel (indexados) que pueden buscarse en la aplicación de OpenReplay.

### Método
`GET`

### URL
`/api/v1/:projectKey/sessions/:sessionId/events`

### Parámetros

| Nombre | Tipo | Descripción |
|----------|-------------|-------------|
| projectKey | string | El ID del proyecto que estás rastreando (obligatorio) |
| sessionId | string | El ID de la sesión de la que estás extrayendo los eventos (obligatorio) |

### Encabezados de la solicitud

`Authorization: {YOUR_ORGANIZATION_API_KEY}`: La clave se encuentra en 'Preferences' > 'Account' > 'Organization API Key'.

### Códigos de estado

`200`: Respuesta enviada como JSON en el cuerpo

### Ejemplo de solicitud

```shell
curl -X GET \
  https://openreplay.example.com/api/v1/3sWXSsqHgSKnE87YkNJK/sessions/4813023023591169/events \
  -H 'content-type: application/json' \
  -H 'Authorization: {YOUR_ORGANIZATION_API_KEY}'
```
