La API de OpenReplay te permite gestionar los datos de usuario y exportar los eventos capturados para su análisis externo.

## URL base (OpenReplay)

- Para el plan Cloud Serverless: `https://api.openreplay.com/api/v1`
- Para Cloud Dedicated o instancias de código abierto autoalojadas: `https://YOUR_INSTANCE_DOMAIN/api/`
