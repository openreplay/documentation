La API de OpenReplay te permite gestionar los datos de usuario y exportar los eventos capturados para su análisis externo.

## URL base (OpenReplay)

- Para el plan Cloud Serverless: `https://api.openreplay.com/api/v1`
- Para Cloud Dedicated o instancias de código abierto autoalojadas: `https://YOUR_INSTANCE_DOMAIN/api/`

## API HTTP
- [Proyectos](/api/projects): Para crear nuevos proyectos.
- [Sesiones](/api/sessions): Para recuperar todas las sesiones de un usuario especificado.
- [Eventos](/api/events): Para extraer todos los eventos de una grabación concreta.
- [Usuarios](/api/users): Para controlar y eliminar los datos de usuario.
- [Trabajos](/api/jobs): Para gestionar los trabajos de eliminación de datos de usuario.
- [Assist](/api/assist): Para recuperar sesiones en vivo.
