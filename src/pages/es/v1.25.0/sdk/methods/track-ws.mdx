Este método te permite rastrear canales websocket individuales y la dirección del mensaje (entrante/saliente).

## Firma

```ts 
trackWs: (channelName: string) => (msgType: string, data: string, dir: 'up' | 'down') => void
```

## Parámetros

- `channelName: string`: Nombre del canal WS.

## Valor de retorno

Devuelve un hook para aplicar a los mensajes ws que toma como parámetros el tipo de mensaje (o nombre), los datos y la dirección del mensaje (up/down === enviado/recibido).
