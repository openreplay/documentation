Cómo integrar Jira Cloud con OpenReplay.

### 1. Crear un nuevo token de API

1. Inicia sesión en tu [token de JIRA](https://id.atlassian.com/manage/api-tokens).
2. Haz clic en *Create API token*.
3. En la etiqueta, escribe "openreplay" y haz clic en *Create*.
4. Haz clic en *Copy to clipboard*.

### 2. Habilitar JIRA Cloud en OpenReplay

1. Ve a tu panel de OpenReplay en 'Preferences > Integration'
2. Selecciona Jira. Si no lo ves, es porque debes tener habilitado GitHub (no puedes usar ambos al mismo tiempo, así que asegúrate de deshabilitar primero la integración con GitHub).
3. Introduce tu `Username` y tu `API token`, luego escribe la URL de tu instancia de Jira Cloud

Entonces aparecerá un botón 'Create Issue' en todas las grabaciones para que puedas crear y asignar incidencias en Jira de forma rápida.
