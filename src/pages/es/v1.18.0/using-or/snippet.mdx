Si quieres agregar el rastreador a tu aplicación web y estás usando un framework que no admitimos, o simplemente estás
usando VanillaJS, puedes agregar directamente nuestro fragmento de JavaScript.

## Obtener el fragmento

El fragmento de JavaScript que necesitarás para completar la tarea se te proporciona cuando configuras el proyecto en el portal de OpenReplay.
Una vez que hayas agregado tu nuevo proyecto, verás algo como esto:

![Nuevo proyecto listo](/using-or/snippet/new-project-ready.png)

Cuando lo veas, haz clic en el botón "Complete Project Setup".
Eso debería llevarte a una página donde se te presenta el fragmento JS:


![Obtener el fragmento JS](/using-or/snippet/getting-snippet.png)

Ahora, simplemente haz clic en la pestaña "Script" y verás el fragmento de JavaScript real:

![El fragmento es visible](/using-or/snippet/snippet-visible.png)

Obtén el script y agrégalo directamente dentro de tu etiqueta `<head>`, y el rastreador comenzará a funcionar automáticamente.

<Aside type="tip">
Ten en cuenta que si cambias cualquier opción de grabación de datos en el paso 1 de la captura de pantalla anterior, el fragmento final cambiará.
Así que asegúrate de configurar primero tus opciones de grabación y luego copiar el fragmento real.
</Aside>
