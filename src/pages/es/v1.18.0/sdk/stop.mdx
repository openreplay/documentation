Este método detiene el tracker y la sesión activa. Si no utilizas este método, la sesión se detendrá automáticamente unos minutos después de que cierres la pestaña o salgas de tu sitio.

## Firma

```js
stop(): string | undefined
```

## Valor de retorno

Devuelve el `sessionHash` de la sesión detenida.
