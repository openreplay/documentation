Este método le permite establecer un ID de usuario en la sesión activa en ese momento. Esto le permite asociar un usuario a la repetición, lo que le permitiría filtrar y buscar sesiones de un usuario en particular.

## Firma
```swift
ORTracker.shared.setUserID(_ userID: String)
```

## Parámetros

- `userID: String`: El ID del usuario. Se trata de una cadena y puede ser cualquier cosa siempre que sea una cadena y le permita identificar de forma única a un usuario dentro de su sistema (por ejemplo, correo electrónico, id de base de datos, etc.).

## Valor de retorno

Este método no devuelve ningún valor; los resultados serán visibles en la plataforma de OpenReplay.
