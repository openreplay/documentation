La pila de OpenReplay se puede instalar en una sola máquina con Docker Compose. 

## Requisitos previos

1. Ubuntu 20.04 LTS (se recomienda la versión LTS)
2. IP pública asociada a la máquina. Significa que tu servidor es accesible desde Internet.
3. Debes tener un nombre de dominio asociado a la dirección IP.
4. Elige el tipo de instancia. Las especificaciones mínimas son `2 vCPUs, 8 GB of RAM, 50 GB of storage`, de lo contrario los servicios backend de OpenReplay simplemente no se iniciarán. Por lo tanto, recomendamos **al menos** el tipo `t3.large` (o uno equivalente), que es suficiente para un volumen bajo/moderado. Si esperas mucho tráfico, deberías escalar a partir de aquí.

## Desplegar OpenReplay

```bash
## From your terminal
SSH_KEY=~/Downloads/openreplay-key.pem #! wherever you've saved the SSH key
INSTANCE_IP=REPLACE_WITH_INSTANCE_PUBLIC_IP
chmod 400 $SSH_KEY
ssh -i $SSH_KEY ubuntu@$INSTANCE_IP
```

2. Instala OpenReplay indicando el dominio en el que se ejecutará (por ejemplo, DOMAIN_NAME=openreplay.mycompany.com):

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/openreplay/openreplay/main/scripts/docker-compose/docker-install.sh)"
```

Ya está todo listo, OpenReplay debería ser accesible de forma segura en el dominio que creaste primero. Puedes crear una cuenta visitando la página `/signup` (es decir, openreplay.mycompany.com/signup).
