Asocia un ID de usuario anónimo con la sesión que se está grabando.

## Firma
```kotlin
OpenReplay.userAnonymousID(userID: String)
```

## Parámetros

- `userID: String`: El ID anónimo del usuario. Es una cadena de texto y puede ser cualquier cosa, siempre que sea una cadena y te permita identificar de forma única a un usuario dentro de tu sistema (es decir, correo electrónico, id de base de datos, etc.).

## Valor de retorno

Este método no devuelve ningún valor; los resultados serán visibles en la plataforma de OpenReplay.
