Esta sección contiene todo lo que necesitas para empezar a usar OpenReplay, desde la configuración básica hasta la avanzada.

## Tutoriales principales
- [Cómo identificar a un usuario](/installation/identify-user): Asocia tu ID interno de usuario con la sesión que se está grabando.
- [Cómo añadir usuarios a tu equipo](/tutorials/invite-team-members): Aprende a añadir usuarios a tu equipo.
- [Cómo enviar metadatos](/installation/metadata): También conocidos como traits o variables de usuario, los metadatos proporcionan información adicional sobre los usuarios mientras se graban las sesiones.
- [Cómo enviar eventos personalizados](/installation/custom-events): Añade más inteligencia registrando eventos específicos de tu dominio junto con las repeticiones de sesión.
- [Cómo realizar el reporte de errores](/installation/error-reporting): Reporta errores y excepciones y visualízalos en la pestaña 'Errors'.
- [Cómo subir Source-Maps](/installation/upload-sourcemaps): Facilita la depuración viendo el contexto del código fuente obtenido de las trazas de pila en su forma original.
- [Cómo sanear datos](/installation/sanitize-data): Cómo sanear datos tanto a nivel global como granular.
- [Cómo crear una heurística personalizada](/tutorials/custom-heuristics): Aprende a añadir lógica personalizada al backend para detectar comportamientos de uso específicos.
- [Cómo usar la barra OmniSearch](/tutorials/omnisearch): Aprende a sacar el máximo provecho de nuestra potente barra de búsqueda.
- [Cómo crear tus propios paneles personalizados](/tutorials/custom-dashboard): Crea tus propias métricas y paneles con este tutorial.
