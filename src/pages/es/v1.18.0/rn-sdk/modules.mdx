El SDK de React Native exporta varios componentes y métodos para que personalices tu grabación:

## Connector

```js
interface IORTrackerConnector {
  startSession: (
    projectKey: string,
    optionsDict: Options,
    projectUrl?: string
  ) => void;
  setMetadata: (key: string, value: string) => void;
  event: (name: string, payload?: string) => void;
  setUserID: (userID: string) => void;
  userAnonymousID: (userID: string) => void;
}
```

- [startSession()](/rn-sdk/start): inicia la grabación.
- [setMetadata()](/rn-sdk/set-metadata): Establece un campo de metadatos y lo guarda como parte de la grabación.
- [event()](/rn-sdk/event): Envía eventos personalizados para que se registren como parte de tus reproducciones con este método.
- [setUserID()](/rn-sdk/set-user-id): Identifica a tus usuarios con este método.
- [patchNetwork()](/rn-sdk/patch-network): Parchea fetch y/o XMLHttpRequest para registrar las solicitudes de red.
