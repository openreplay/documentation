## Métodos

- [event()](/ios-sdk/methods/event): Envía eventos personalizados para que se registren como parte de tus repeticiones con este método.
- [setUserID()](/ios-sdk/methods/set-user-id): Identifica a tus usuarios con este método.
- [userAnonymousID()](/ios-sdk/methods/user-anonymous-id): Asocia un userID anónimo con la sesión que se está grabando.
- [setMetadata()](/ios-sdk/methods/set-metadata): Establece un campo de metadatos y guárdalo como parte de la grabación.
- [start()](/ios-sdk/methods/start): Inicia la grabación de una sesión.
- [stop()](/ios-sdk/methods/stop): Detén la grabación de una sesión con este método.
