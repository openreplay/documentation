El tracker escuchará el evento de cambio de valor (con debounce) y enviará el valor resultante al backend de OpenReplay.

## Añadir entradas con seguimiento

El SDK exporta `ORTrackedInput`, que puedes usar como componentes TextInput normales.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORTrackedInput
    style={styles.input}
    onChangeText={onChangeNumber}
    value={number}
    placeholder="important placeholder"
  />

// ...
```
