El tracker escuchará los eventos de aparición/desaparición de vistas en la pantalla del teléfono.

## Añadir vistas con seguimiento

El SDK exporta `ORAnalyticsView`, que puedes usar para marcar las vistas/componentes importantes.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORAnalyticsView
    screenName="Pay confirmation page"
    viewName="New fancy button"
  >
    <ImportantRedButton />
  </OR.ORAnalyticsView>

// ...
```
