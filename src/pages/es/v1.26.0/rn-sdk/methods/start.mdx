Este método se utiliza para iniciar el tracker. Al iniciar el tracker, comienza la grabación de la sesión.

## Firma

```swift
import OR from '@openreplay/react-native'

OR.tracker.startSession(options)
```

## Opciones de inicialización

Por defecto, todas las opciones están marcadas como `true`:
- `crashes: Bool` Habilita crashlytics.
- `analytics: Bool` Habilita el seguimiento de analítica de las vistas marcadas.
- `performances: Bool` Habilita el listener de rendimiento
- `logs: Bool` Habilita el listener de logs
- `screen: Bool` Habilita la grabación de pantalla

Las opciones pueden usarse para deshabilitar ciertas funciones; de lo contrario, puedes pasar el objeto vacío (`{}`)
