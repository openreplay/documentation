Tu instancia de OpenReplay se puede configurar fácilmente. Aquí tienes tutoriales paso a paso para habilitar los servicios de correo electrónico, proteger tu instalación y eliminar grabaciones de la base de datos.

## Guías paso a paso
- [Configure SMTP](/configuration/configure-smtp): Configura SMTP para recibir alertas, informes semanales, invitar a nuevos usuarios a OpenReplay y restablecer contraseñas.
- [Secure OpenReplay](/configuration/secure-or): Protege OpenReplay configurando SSL, reCaptcha y estableciendo una CSP.
- [Proxy Settings](/configuration/proxy-settings): Ejecuta OpenReplay detrás de un proxy como NGINX o Apache2.
- [Cleanup Storage](/configuration/cleanup-storage): Cómo se almacenan las grabaciones y cómo eliminarlas de forma masiva de la base de datos y limpiar tu almacenamiento.
- [External Recordings' Storage](/configuration/external-storage): Cambia el destino de almacenamiento predeterminado de tus grabaciones.
- [External Database](/configuration/external-db): Usa una base de datos Postgres externa para OpenReplay.
- [Single-Sign-On (SAML)](/configuration/sso): Configura el inicio de sesión único (SSO) mediante SAML.
