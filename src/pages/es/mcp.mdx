La app **OpenReplay MCP** conecta tu instancia de OpenReplay con un host de chat compatible con MCP (Claude Desktop, Claude Code, ChatGPT, Codex) para que puedas explorar la analítica de producto y las reproducciones de sesión en lenguaje natural. Pide un gráfico, profundiza en un embudo, encuentra una sesión y luego abre su reproducción, todo desde dentro del chat.

Funciona tanto con **OpenReplay Cloud** como con instancias **autoalojadas**.

<Aside type="tip" title="Lo que obtienes" >
  - Busca sesiones con los mismos filtros que usas en la interfaz de OpenReplay.
  - Renderiza series temporales, embudos, recorridos de usuario, web vitals y desgloses top-of-X de forma integrada.
  - Mira reproducciones de sesión directamente en el chat.
  - Encadénalo todo: buscar → elegir una sesión → abrir la reproducción → pedirle a Claude que la resuma.
</Aside>

## Compatibilidad

La app habla MCP estándar sobre stdio, así que funciona con:

- **Claude Desktop** (macOS, Windows)
- **Claude Code**
- **ChatGPT** (a través del conector MCP)
- **Codex**
- Cualquier otro host compatible con MCP
