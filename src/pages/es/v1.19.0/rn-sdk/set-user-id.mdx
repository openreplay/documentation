---

Este método te permite establecer un ID de usuario en la sesión actualmente activa. Esto te permite asociar un usuario a la repetición, lo que te permitiría filtrar y buscar sesiones de un usuario en particular.

## Firma

```js
import OR from '@openreplay/react-native'

// ...
OR.tracker.setUserID('id')
```


## Parámetros

- `userID: String`: El ID del usuario. Es una cadena de texto y puede ser cualquier cosa siempre que sea una cadena, y te permite identificar de forma única a un usuario dentro de tu sistema (por ejemplo, correo electrónico, id de base de datos, etc.).

## Valor de retorno

Este método no devuelve ningún valor; los resultados serán visibles en la plataforma de OpenReplay.
