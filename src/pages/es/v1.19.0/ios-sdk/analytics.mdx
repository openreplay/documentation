El tracker escuchará los eventos de aparición/desaparición de las vistas.

## Añadir vistas rastreadas

### SwiftUI

usa `.observeView(title:viewName:)` para observar los eventos de entrada,

- `title: String` - título de la pantalla o función
- `viewName: String` - nombre de la vista del tracker

```swift
import ORTracker

TextField("Input", text: $text)
 .observeView(title: "Screen title", viewName: "test input name")
```

### UIKit
```swift
import ORTracker

Analytics.shared.addObservedView(view: inputEl, title: "Screen title", viewName: "test input name")
```
