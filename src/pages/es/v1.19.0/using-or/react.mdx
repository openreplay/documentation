Poner en funcionamiento el tracker en una aplicación React es bastante sencillo. 

## Agregar el código de seguimiento

Aunque puedes [obtener una versión para copiar y pegar del código directamente desde la plataforma cuando creas un nuevo proyecto](/using-or/snippet#getting-the-snippet), ese código es demasiado básico y hará que sea mucho más difícil personalizarlo y agregar plugins adicionales.

Suponiendo que estás construyendo una SPA (Single Page Application), todo lo que tienes que hacer es agregar el siguiente código a tu punto de entrada:

```jsx
import Tracker from '@openreplay/tracker';

const tracker = new Tracker({
  projectKey: "<your project key>",  
});

function YourMainComponent({props}) {
    //some code here...

    useEffect(() => {
        tracker.start();
    }, [])

    return //....
}
```

En otras palabras, puedes instanciar el tracker al inicio del script y, una vez que la página se carga, puedes iniciarlo utilizando un hook `useEffect`.

<Aside type="tip">
**Una nota para los usuarios autoalojados**: si estás utilizando la versión autoalojada de OpenReplay, también tendrás que especificar la propiedad de configuración `ingestPoint`. 
Esta propiedad especifica el punto de ingestión para los datos del tracker. Los usuarios de la nube no necesitan esto, 
porque por defecto el tracker ya sabrá dónde está la versión SaaS de este punto, pero si lo alojas tú mismo, necesitarás especificarlo 
(debería ser algo como `https://openreplay.mydomain.com/ingest`) 
</Aside>

### Manejo de la "projectKey" en tu código

Como nota, la `projectKey` que configuras es la que proporciona la plataforma de OpenReplay. Por razones de seguridad, este valor no debe almacenarse codificado de forma fija en tu código, sino más bien en algún tipo de archivo/sistema de configuración.

Existen diferentes opciones para resolver este problema; dependiendo de tu configuración, podrías usar algo como [el soporte de variables ENV de `create-react-app`](https://create-react-app.dev/docs/adding-custom-environment-variables/).
