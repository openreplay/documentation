Este método se utiliza para iniciar el tracker. Al iniciar el tracker, comienza la grabación de la sesión.

## Firma

```kotlin
OpenReplay.start(applicationContext: Context, projectKey: String, options: OROptions)
	```

### Opciones de inicialización

Opciones disponibles:

- `crashes: Bool` Habilita crashlytics.
- `analytics: Bool` Habilita la analítica de las vistas.
- `performances: Bool` Habilita el listener de rendimiento
- `logs: Bool` Habilita el listener de logs
- `screen: Bool` Habilita el grabador de pantalla
- `wifiOnly: Bool` Obliga al tracker a iniciarse solo si el usuario tiene una conexión wifi

Por defecto: todo en `true`
