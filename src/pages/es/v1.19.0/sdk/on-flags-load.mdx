Este método establece la función de devolución de llamada (callback) que se usará cuando se carguen las Feature Flags.

## Firma

```js
onFlagsLoad(callback: (flags: IFeatureFlag[]) => void): void
```

## Argumentos

Función de devolución de llamada que se usará una vez que las feature flags se recarguen. Se ejecuta en cada evento de recarga de flags.
