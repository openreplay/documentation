Les cartes de chaleur offrent une représentation visuelle des interactions des utilisateurs sur l'application de votre site web, en mettant en évidence les zones de forte activité telles que les clics et les mouvements de souris. Elles vous aident à identifier ce qui attire l'attention des utilisateurs, les boutons ou contenus qu'ils manquent complètement, ainsi que les zones source de friction.

## **Cas d'usage** des cartes de chaleur

L'utilisation des cartes de chaleur peut vous aider à répondre à de nombreuses questions, telles que :

- Avec quels éléments les utilisateurs interagissent-ils le plus sur votre page d'accueil ?
- Les CTA clés sont-ils ignorés ?
- Où les utilisateurs cliquent-ils dans votre barre de navigation ?
- Les utilisateurs subissent-ils de la rage de clic (click rage) sur des boutons ou éléments spécifiques ?
- Quelles sections de votre page retiennent l'attention des utilisateurs, et lesquelles sont ignorées ?

## Premiers pas avec les cartes de chaleur

#### Créer une nouvelle carte de chaleur

Vous pouvez accéder aux cartes de chaleur dans le menu Cards, sous la catégorie « Product Analytics ». Sélectionnez « Heatmaps » pour commencer à analyser les interactions des utilisateurs.

#### Anatomie de base de la carte de chaleur

1. **Barre de menu :** Changez la disposition d'affichage, renommez, supprimez ou ajoutez la carte de chaleur à un tableau de bord.
2. **Spécification de l'URL :** Saisissez l'URL de la page que vous souhaitez analyser. Si vous avez un doute, le système propose des suggestions basées sur les pages les plus populaires de votre site.
3. **Filtres :** Appliquez des filtres tels que les attributs utilisateur ou les caractéristiques de session pour affiner les données.
4. **Zone de carte de chaleur :** Visualisez les mises en évidence des clics de souris et de la rage de clic. Utilisez le bouton « Get New Session » pour actualiser la capture d'écran.
5. **Plage de temps :** Ajustez la période d'analyse aux dernières 24 heures, aux 7 derniers jours ou à une plage personnalisée.
6. **Liste des sessions :** Consultez les sessions pour des relectures détaillées.

## Configurer une carte de chaleur

#### Saisissez l'URL de votre page de carte de chaleur

Utilisez le champ URL pour spécifier la page que vous souhaitez analyser. Les options incluent :

- **L'URL contient :** Affiche les données de toutes les URL qui contiennent le texte saisi (sensible à la casse).
- **L'URL commence par :** Affiche les résultats de toutes les URL commençant par le texte saisi.
- **L'URL se termine par :** Affiche les résultats de toutes les URL se terminant par les textes saisis.

#### Filtrez les données de votre carte de chaleur

Les filtres vous aident à affiner les données pour obtenir des informations plus exploitables. Les filtres disponibles incluent :
    - Attributs utilisateur (par ex., User ID, userAnanymousId ).
    - Caractéristiques de session (par ex., pays, ville, OS, navigateur, UTM Source,...  ).
    - Problèmes (par ex., rage de clic, clic mort, mauvaise requête,...  ).


##  Tirez le meilleur parti des cartes de chaleur

1. Concentrez-vous sur les pages à fort trafic pour obtenir les informations les plus exploitables.
2. Utilisez les filtres pour segmenter les données par attributs utilisateur ou paramètres de session.
3. Combinez les informations des cartes de chaleur avec les relectures de session pour découvrir le « pourquoi » derrière les actions des utilisateurs.
