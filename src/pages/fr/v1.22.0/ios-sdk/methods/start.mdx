Cette méthode est utilisée pour démarrer le tracker. En démarrant le tracker, l'enregistrement de la session commence.

## Signature

```swift
ORTracker.shared.start(projectKey: String, options: OROptions)
	```

### Options d'initialisation

Options disponibles :

- `crashes: Bool` Active crashlytics.
- `analytics: Bool` Active l'analytique pour les vues.
- `performances: Bool` Active le listener de performances
- `logs: Bool` Active le listener de logs
- `screen: Bool` Active l'enregistreur d'écran
- `wifiOnly: Bool` Force le tracker à ne démarrer que si l'utilisateur dispose d'une connexion wifi

Par défaut : toutes à `true`
