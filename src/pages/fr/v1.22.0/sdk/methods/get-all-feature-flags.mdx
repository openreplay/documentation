Cette méthode renvoie tous les drapeaux (`IFeatureFlag[]`) qui sont actifs en ce moment.

## Signature

```js
getAllFeatureFlags(): IFeatureFlag[]
```

## Valeur de retour

Tableau de tous les feature flags actifs (IFeatureFlag)
