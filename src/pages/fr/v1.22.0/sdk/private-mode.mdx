## Mode privé

Le **Mode privé** est un nouveau paramètre qui empêche l'enregistrement de tout texte et de toute donnée sensible. Lorsqu'il est activé, le tracker OpenReplay ne capture aucun texte visible, aucune donnée réseau ni aucun journal de console. Tout est masqué à la source, de sorte qu'aucune donnée sensible ne quitte jamais le navigateur.

C'est utile si vous souhaitez une visibilité totale sur le comportement de l'interface tout en gardant le contenu et les données de l'utilisateur entièrement privés.

## Comment activer le Mode privé

Vous pouvez activer le Mode privé au niveau du tracker lors de l'initialisation d'OpenReplay.

#### Avec NPM

```js
import { tracker } from '@openreplay/tracker'

tracker.configure({
    projectKey: PROJECT_KEY,
    privateMode: true
})
tracker.start();
```

Lorsque `privateMode` est défini sur `true`, les éléments suivants sont masqués :

- Tout le texte visible sur la page  
- Les attributs `alt` des images  
- Les valeurs `placeholder` des champs de saisie  
- Les textes des libellés

## Quand l'utiliser

Utilisez `privateMode` si :

- Vous devez garantir une confidentialité complète des données utilisateur
- Vous testez OpenReplay dans un **environnement cloud**, mais prévoyez de **l'auto-héberger plus tard** et **ne pouvez transmettre aucune donnée réelle vers l'extérieur**
- Vous travaillez dans des secteurs réglementés où **la confidentialité passe avant tout**
