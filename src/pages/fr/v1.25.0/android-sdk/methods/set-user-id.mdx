Cette méthode vous permet de définir un ID utilisateur sur la session actuellement active. Cela vous permet d'associer un utilisateur à l'enregistrement, ce qui vous permettrait de filtrer et de rechercher les sessions d'un utilisateur en particulier.

## Signature
```swift
OpenReplay.setUserID(userID: String)
```

## Paramètres

- `userID: String` : L'ID de l'utilisateur. Il s'agit d'une chaîne de caractères et peut être n'importe quoi tant qu'il s'agit d'une chaîne, et il vous permet d'identifier de manière unique un utilisateur au sein de votre système (c'est-à-dire e-mail, id de base de données, etc.).

## Valeur de retour

Cette méthode ne renvoie aucune valeur, les résultats seront visibles sur la plateforme OpenReplay.
