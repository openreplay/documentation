Avec cette méthode, vous pouvez définir la valeur de l'un des champs de métadonnées configurés sur la plateforme OpenReplay.

<Aside type="note">
	Vous pouvez appeler cette méthode autant de fois que vous le souhaitez, mais gardez à l'esprit que si vous utilisez le même attribut `key`, la dernière valeur que vous avez passée sera enregistrée dans la plateforme. Toutes les valeurs précédentes seront perdues.
</Aside>

Consultez [metadata](/installation/metadata) pour plus de détails. 

## Signature
```swift
ORTracker.shared.setMetadata(key: String, value: String)
```

## Paramètres

- `key: string` : La clé de métadonnées à enregistrer. 
- `value: string` : La valeur à enregistrer en tant que métadonnées.

## Valeur de retour

Cette méthode ne renvoie aucune valeur. Les informations sont enregistrées directement dans le cadre de l'enregistrement et peuvent être consultées dans la plateforme OpenReplay.
