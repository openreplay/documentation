Force le tracker à vider le lot de messages actuel sans attendre la fin du traitement. Utile lorsque vous devez fermer l'onglet ou la fenêtre après une session. Cette méthode envoie également un message au webworker, qui lancera la requête réseau avec le lot de messages actuel.

## Signature

```ts
forceFlushBatch(): void
```
