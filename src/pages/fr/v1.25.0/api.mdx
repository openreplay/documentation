L'API OpenReplay vous permet de gérer les données utilisateur et d'exporter les événements capturés pour une analyse externe.

## URL de base (OpenReplay)

- Pour le plan Cloud Serverless : `https://api.openreplay.com/api/v1`
- Pour Cloud Dedicated ou les instances open source auto-hébergées : `https://YOUR_INSTANCE_DOMAIN/api/`
