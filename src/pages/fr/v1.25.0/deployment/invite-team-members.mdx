OpenReplay vous permet de gérer les membres de l'équipe en les invitant ou en les supprimant selon vos besoins. Cet article décrit les étapes pour gérer les membres de votre équipe.

## Inviter des membres de l'équipe

Pour inviter des membres de l'équipe :

1. Accédez à **Preferences > Team** dans le tableau de bord OpenReplay.
2. Cliquez sur le bouton **Add Team Member**.
3. Renseignez les informations de l'utilisateur :
   - **Full Name**
   - **Email Address**
4. (Facultatif) Cochez Admin Privileges si l'utilisateur a besoin d'un accès à la gestion de l'équipe et des projets.
5. (Enterprise Edition uniquement) Attribuez un rôle pour définir son accès aux projets et aux fonctionnalités.
6. Cliquez sur **Invite**.

<div class="or-aside note">

1. **Pour les utilisateurs Self-Hosted** : L'inscription initiale est limitée à un seul utilisateur lors de la configuration. Les utilisateurs supplémentaires doivent être invités. Les invitations par e-mail nécessitent une configuration SMTP ; sans elle, partagez les liens d'invitation manuellement. Consultez notre [guide SMTP](/configuration/configure-smtp).

2. **Pour les utilisateurs Cloud (Serverless ou Dedicated)** : Le créateur du compte a le rôle d'**Owner** avec des privilèges d'administrateur. Les nouveaux membres rejoignent l'équipe en tant que Members standard, mais peuvent se voir accorder des **Admin Privileges**. Tous les membres de l'équipe invités reçoivent automatiquement des invitations par e-mail lorsqu'ils sont ajoutés

</div>

## Supprimer des membres de l'équipe

Pour supprimer un membre de l'équipe :

1. Accédez à **Preferences > Team** dans le tableau de bord OpenReplay.
2. Repérez l'utilisateur dans la liste de l'équipe.
3. Cliquez sur l'icône **Edit** à côté de son nom.
4. Cliquez sur l'icône **Delete**.
