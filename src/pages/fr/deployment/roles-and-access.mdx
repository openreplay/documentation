Les rôles et accès permettent aux administrateurs de définir des autorisations spécifiques pour les projets et les fonctionnalités. 

> **Remarque :** Cette fonctionnalité n'est disponible que pour l'**[édition Enterprise](https://openreplay.com/enterprise.html)**.

### Ajouter un rôle

1. Accédez à **Preferences > Roles and Access** dans le tableau de bord OpenReplay
2. Cliquez sur le bouton **Add**
3. Définissez le rôle :
   - **Title** : Nom du rôle (par exemple, Developer)
   - **Project Access** : Sélectionnez les projets accessibles
   - **Feature Access** : Choisissez les fonctionnalités disponibles
4. Cliquez sur **Add**

### Modifier un rôle
1. Accédez à **Preferences > Roles and Access**
2. Trouvez le rôle que vous souhaitez modifier
3. Cliquez sur l'icône **Edit**
4. Mettez à jour les paramètres d'accès
5. Cliquez sur **Update**

> **Remarque :** Le rôle **Owner** ne peut pas être modifié.

### Supprimer un rôle
1. Accédez à **Preferences > Roles and Access**
2. Trouvez le rôle que vous souhaitez modifier
3. Cliquez sur l'icône **Edit**
4. Cliquez sur l'icône **Delete**
5. Cliquez sur **Proceed** pour confirmer la suppression
