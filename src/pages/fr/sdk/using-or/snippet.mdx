Si vous cherchez à ajouter le tracker à votre application web et que vous utilisez un framework que nous ne prenons pas en charge ou que vous utilisez simplement
VanillaJS, vous pouvez ajouter directement notre snippet JavaScript.

## Obtenir le snippet

Le snippet JavaScript dont vous aurez besoin pour accomplir cette tâche vous est fourni lorsque vous configurez le projet sur le portail OpenReplay.
Une fois que vous avez ajouté votre nouveau projet, vous verrez quelque chose comme ceci :

![New project ready](/using-or/snippet/new-project-ready.png)

Une fois que vous voyez cela, cliquez sur le bouton "Complete Project Setup".
Cela devrait vous amener à une page où le snippet JS vous est présenté :


![Getting the JS Snippet](/using-or/snippet/getting-snippet.png)

Maintenant, cliquez simplement sur l'onglet "Script" et vous verrez le véritable snippet JavaScript :

![The snippet is visible](/using-or/snippet/snippet-visible.png)

Récupérez le script et ajoutez-le directement à l'intérieur de votre balise `<head>` et le tracker commencera à fonctionner automatiquement.

<Aside type="tip">
Gardez à l'esprit que si vous modifiez l'une des options d'enregistrement des données à l'étape 1 de la capture d'écran ci-dessus, le snippet final changera.
Assurez-vous donc de configurer d'abord vos options d'enregistrement, puis de copier le véritable snippet.
</Aside>
