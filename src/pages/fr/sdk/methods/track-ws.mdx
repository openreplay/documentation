Cette méthode vous permet de suivre des canaux websocket individuels ainsi que la direction du message (entrant/sortant).

## Signature

```ts 
trackWs: (channelName: string) => (msgType: string, data: string, dir: 'up' | 'down') => void
```

## Paramètres

- `channelName: string` : Nom du canal WS.

## Valeur de retour

Renvoie un hook à appliquer aux messages ws qui prend en paramètres le type de message (ou nom), les données et la direction du message (up/down === envoyé/reçu).
