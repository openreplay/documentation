Comment intégrer Jira Cloud à OpenReplay.

## 1. Créer un nouveau jeton d'API

1. Connectez-vous à votre [jeton JIRA](https://id.atlassian.com/manage/api-tokens).
2. Cliquez sur *Create API token*.
3. Dans le libellé, saisissez « openreplay » puis cliquez sur *Create*.
4. Cliquez sur *Copy to clipboard*.

## 2. Activer JIRA Cloud dans OpenReplay

1. Accédez à votre tableau de bord OpenReplay sous « Preferences > Integration »
2. Sélectionnez Jira. Si vous ne le voyez pas, c'est que vous avez probablement activé GitHub (vous ne pouvez pas utiliser les deux en même temps, veillez donc à désactiver d'abord l'intégration GitHub).
3. Saisissez votre `Username` et votre `API token`, puis indiquez l'URL de votre instance Jira Cloud

Un bouton « Create Issue » apparaîtra alors dans tous les enregistrements afin que vous puissiez créer et assigner rapidement des tickets dans Jira.
