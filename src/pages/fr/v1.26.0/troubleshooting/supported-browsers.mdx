---

OpenReplay est pris en charge par les navigateurs suivants :
- Google Chrome 41+
- Firefox 17+
- Microsoft Edge 12+
- Internet Explorer 11+
- Safari 9+
- Safari sur iOS 9+
- Opera 28+
- Opera Mobile 62+

Si vous enregistrez une session dans un navigateur non pris en charge, le SDK lèvera une exception.
