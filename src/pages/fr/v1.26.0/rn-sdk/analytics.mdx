Le tracker écoutera les événements d'apparition/disparition des vues sur l'écran du téléphone.

## Ajouter des vues suivies

Le SDK exporte `ORAnalyticsView`, que vous pouvez utiliser pour marquer les vues/composants importants.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORAnalyticsView
    screenName="Pay confirmation page"
    viewName="New fancy button"
  >
    <ImportantRedButton />
  </OR.ORAnalyticsView>

// ...
```
