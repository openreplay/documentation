Associez un userID anonyme à la session en cours d'enregistrement.

## Signature

```swift
ORTracker.userAnonymousID(userID: String)
```

## Paramètres

- `userID: String` : L'ID anonyme de l'utilisateur. Il s'agit d'une chaîne de caractères et peut être n'importe quoi, du moment que c'est une chaîne, et il vous permet d'identifier de manière unique un utilisateur au sein de votre système (c'est-à-dire e-mail, id de base de données, etc.).

## Valeur de retour

Cette méthode ne retourne aucune valeur ; les résultats seront visibles sur la plateforme OpenReplay.
