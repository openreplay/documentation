## Méthodes et modules

- [Analytics](/ios-sdk/analytics) : Découvrez tout sur le suivi de vues spécifiques.
- [Network](/ios-sdk/network) : Découvrez comment utiliser le NetworkListener.
- [Sanitization](/ios-sdk/sanitization) : Découvrez comment masquer des vues sur l'écran enregistré.
- [Inputs](/ios-sdk/inputs) : Découvrez comment enregistrer les événements de saisie.
- [event()](/ios-sdk/event) : Envoyez des événements personnalisés pour qu'ils soient enregistrés dans vos rejeux grâce à cette méthode.
- [setUserID()](/ios-sdk/set-user-id) : Identifiez vos utilisateurs avec cette méthode.
- [setMetadata()](/ios-sdk/set-metadata) : Définissez un champ de métadonnées et enregistrez-le dans le cadre de l'enregistrement.
- [start()](/ios-sdk/start) : Démarrez l'enregistrement d'une session.
- [stop()](/ios-sdk/stop) : Arrêtez l'enregistrement d'une session avec cette méthode.
