OpenReplay Enterprise Edition est livré avec différents connecteurs permettant d'exporter les données capturées (comportement des utilisateurs et problèmes techniques) vers votre data warehouse. Nous prenons en charge :
- Amazon Redshift
- Google BigQuery
- PostgreSQL
- Snowflake
- ClickHouse
-
