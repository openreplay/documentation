Cette méthode recharge tous les flags (cela peut être utile si la session de l'utilisateur a subi une modification de ses métadonnées, de son userId ou de toute autre information qui pourrait être requise pour certaines conditions de flags).

## Signature

```js
reloadFlags(): Promise<void>
```

## Valeur de retour

Renvoie une promesse vide, exécute onFlagsCb en cas de succès
