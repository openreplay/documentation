---

Cette méthode vous permet de définir un identifiant utilisateur sur la session actuellement active. Cela vous permet d'associer un utilisateur au replay, ce qui vous permettrait de filtrer et de rechercher les sessions d'un utilisateur en particulier.

## Signature

```js
import OR from '@openreplay/react-native'

// ...
OR.tracker.setUserID('id')
```


## Paramètres

- `userID: String` : L'identifiant de l'utilisateur. Il s'agit d'une chaîne de caractères et peut être n'importe quoi tant qu'il s'agit d'une chaîne, et il vous permet d'identifier de manière unique un utilisateur au sein de votre système (par exemple, e-mail, id de base de données, etc.).

## Valeur de retour

Cette méthode ne renvoie aucune valeur ; les résultats seront visibles dans la plateforme OpenReplay.
