Cette section propose des guides pratiques classés par catégorie pour vous aider à installer et configurer OpenReplay, de l'installation de base aux configurations avancées.

## Configuration des utilisateurs

- [Comment identifier un utilisateur](/installation/identify-user) : Associez votre identifiant utilisateur interne à la session en cours d'enregistrement.
- [Comment envoyer des métadonnées](/installation/metadata) : Également appelées traits ou variables utilisateur, les métadonnées fournissent des informations supplémentaires sur les utilisateurs pendant l'enregistrement des sessions.

## Configuration de la relecture de sessions

- [Comment envoyer des événements personnalisés](/installation/custom-events) : Apportez plus d'intelligence en enregistrant des événements spécifiques à votre domaine en parallèle des relectures de sessions.
- [Comment configurer Canvas et WebGL](/installation/canvas) : Découvrez comment intégrer et configurer Canvas et WebGL dans votre projet.
- [Comment configurer des iFrames inter-domaines](/installation/crossdomain-iframe) : Configurez des iFrames inter-domaines pour une intégration transparente.
- [Comment configurer les WebSockets](/installation/websockets) : Configurez les WebSockets pour la communication en temps réel dans votre application.

## Réseau et confidentialité des données

- [Comment personnaliser les options réseau (Web)](/installation/network-options) : Personnalisez les configurations réseau pour votre environnement web.
- [Comment personnaliser les options réseau (Mobile)](/installation/network-options-mobile) : Personnalisez les configurations réseau pour vos environnements iOS et Android.
- [Comment assainir les données (Web)](/installation/sanitize-data) : Comment assainir les données à un niveau global et/ou granulaire pour vos applications web.
- [Comment assainir les données (Mobile)](/installation/sanitize-data-mobile) : Comment assainir les données pour les applications iOS et Android.

## Débogage et suivi des erreurs

- [Comment effectuer le suivi des erreurs](/installation/error-reporting) : Suivez les erreurs et les exceptions et résolvez les problèmes avec un contexte complet.
- [Comment téléverser des Source-Maps](/installation/upload-sourcemaps) : Facilitez le débogage en visualisant le contexte du code source obtenu à partir des traces de pile sous leur forme originale.
