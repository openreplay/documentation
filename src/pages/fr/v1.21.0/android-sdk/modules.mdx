## Méthodes et modules

- [Analytics](/android-sdk/analytics) : Découvrez tout sur le suivi de vues spécifiques.
- [Network](/android-sdk/network) : Découvrez comment utiliser le NetworkListener.
- [Sanitization](/android-sdk/sanitization) : Découvrez comment masquer des vues sur l'écran enregistré.
- [Inputs](/android-sdk/inputs) : Découvrez comment enregistrer les événements de saisie.
- [event()](/android-sdk/event) : Envoyez des événements personnalisés afin qu'ils soient enregistrés dans vos relectures avec cette méthode.
- [setUserID()](/android-sdk/set-user-id) : Identifiez vos utilisateurs avec cette méthode.
- [setMetadata()](/android-sdk/set-metadata) : Définissez un champ de métadonnées et enregistrez-le dans l'enregistrement.
- [start()](/android-sdk/start) : Démarrez l'enregistrement d'une session.
- [stop()](/android-sdk/stop) : Arrêtez l'enregistrement d'une session avec cette méthode.
