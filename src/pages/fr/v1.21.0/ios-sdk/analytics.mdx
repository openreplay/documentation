Le tracker écoutera les événements d'apparition/disparition des vues.

## Ajouter des vues suivies

### SwiftUI

utilisez `.observeView(title:viewName:)` pour observer les événements de saisie,

- `title: String` - titre de l'écran ou de la fonctionnalité
- `viewName: String` - nom de la vue du tracker

```swift
import ORTracker

TextField("Input", text: $text)
 .observeView(title: "Screen title", viewName: "test input name")
```

### UIKit
```swift
import ORTracker

Analytics.shared.addObservedView(view: inputEl, title: "Screen title", viewName: "test input name")
```
