Associe un userID anonyme à la session en cours d'enregistrement.

## Signature

```swift
ORTracker.shared.userAnonymousID(_ userID: String)
```

## Paramètres

- `userID: String` : L'ID anonyme de l'utilisateur. Il s'agit d'une chaîne de caractères qui peut être n'importe quoi tant qu'il s'agit d'une chaîne et qu'elle vous permet d'identifier de manière unique un utilisateur au sein de votre système (c'est-à-dire e-mail, id de base de données, etc.).

## Valeur de retour

Cette méthode ne renvoie aucune valeur, les résultats seront visibles dans la plateforme OpenReplay.
