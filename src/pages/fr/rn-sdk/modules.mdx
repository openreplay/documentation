Le SDK React Native exporte plusieurs modules pour vous permettre de personnaliser votre enregistrement :

## Modules

- [ORTouchTrackingView](/rn-sdk/touch) : Vue utilisée pour suivre les événements tactiles.
- [ORSanitizedView](/rn-sdk/sanitization) : Vue qui vous permet d'assainir (ou de flouter) des parties de votre enregistrement
- [ORTrackedInput](/rn-sdk/inputs) : TextInput qui enregistre les événements de saisie
- [ORAnalyticsView](/rn-sdk/analytics) : Vue marquée qui envoie un événement dès qu'elle apparaît à l'écran et lorsqu'elle le quitte
