Cette méthode vous permet de définir un ID utilisateur sur la session actuellement active. Cela vous permet d'associer un utilisateur à la session enregistrée, ce qui vous permettrait de filtrer et de rechercher les sessions d'un utilisateur en particulier.

## Signature

```js
import OR from '@openreplay/react-native'

// ...
OR.tracker.setUserID('id')
```


## Paramètres

- `userID: String` : L'ID de l'utilisateur. Il s'agit d'une chaîne de caractères et peut être n'importe quelle valeur tant qu'il s'agit d'une chaîne et qu'elle vous permet d'identifier de manière unique un utilisateur au sein de votre système (c.-à-d. e-mail, identifiant de base de données, etc.).

## Valeur de retour

Cette méthode ne renvoie aucune valeur, les résultats seront visibles dans la plateforme OpenReplay.
