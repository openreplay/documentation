**Explain AI** est une fonctionnalité d’OpenReplay basée sur l’IA qui fournit des informations sur les requêtes réseau (**Fetch/XHR**) et les journaux de console (**erreurs**) dans DevTools. Elle aide les développeurs à comprendre ce qui n’a pas fonctionné dans une session sans avoir à inspecter manuellement les journaux.

<Aside type="caution" title="Disponibilité" >
  - Limité pour le moment aux utilisateurs **Cloud (Serverless)**.
</Aside>


## Comment utiliser Explain AI pour les erreurs de console
1. Ouvrez un **Rejeu de session** dans OpenReplay.
2. Accédez au panneau **DevTools**.
3. Cliquez sur l’onglet **Console**.  
4. Sélectionnez **Errors**, puis survolez l’erreur : sur le côté droit, vous pouvez cliquer sur le cercle avec l’info-bulle **'Understand this log'**

## Comment utiliser Explain AI pour les requêtes réseau

1. Ouvrez un **Rejeu de session** dans OpenReplay.
2. Accédez au panneau **DevTools**.
3. Cliquez sur l’onglet **Network**.  
4. Sélectionnez **Fetch/XHR**, puis survolez l’erreur : sur le côté droit, vous pouvez cliquer sur le cercle avec l’info-bulle **'Understand this request'**
