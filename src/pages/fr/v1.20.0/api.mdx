L'API OpenReplay vous permet de gérer les données utilisateur et d'exporter les événements capturés pour une analyse externe.

## API HTTP
- [Projets](/api/projects) : Pour créer de nouveaux projets.
- [Sessions](/api/sessions) : Pour récupérer toutes les sessions d'un utilisateur spécifié.
- [Événements](/api/events) : Pour extraire tous les événements d'un enregistrement particulier.
- [Utilisateurs](/api/users) : Pour contrôler et supprimer les données utilisateur.
- [Tâches](/api/jobs) : Pour gérer les tâches de suppression des données utilisateur.
- [Assist](/api/assist) : Pour récupérer les sessions en direct.
