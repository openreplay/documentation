## Récupérer les événements

Renvoie la liste des événements tels que les clics, les visites de pages ou les saisies contenus dans un enregistrement de session particulier. Il s'agit d'événements de haut niveau (indexés) qui peuvent être recherchés dans l'application OpenReplay.

### Méthode
`GET`

### URL
`/api/v1/:projectKey/sessions/:sessionId/events`

### Paramètres

| Nom | Type | Description |
|----------|-------------|-------------|
| projectKey | string | L'ID du projet que vous suivez (requis) |
| sessionId | string | L'ID de la session à partir de laquelle vous récupérez les événements (requis) |

### En-têtes de la requête

`Authorization: {YOUR_ORGANIZATION_API_KEY}` : La clé se trouve dans 'Preferences' > 'Account' > 'Organization API Key'.

### Codes de statut

`200` : Réponse envoyée au format JSON dans le corps

### Exemple de requête

```shell
curl -X GET \
  https://openreplay.example.com/api/v1/3sWXSsqHgSKnE87YkNJK/sessions/4813023023591169/events \
  -H 'content-type: application/json' \
  -H 'Authorization: {YOUR_ORGANIZATION_API_KEY}'
```
