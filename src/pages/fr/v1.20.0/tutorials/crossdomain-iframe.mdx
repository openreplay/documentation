---

Pour activer le suivi des iFrames inter-domaines, vous devez ajouter l'option `crossdomain.enabled` au constructeur du tracker, comme indiqué ci-dessous :

```html
<iframe
	title="test"
	src="http://iframe1.website.com/iframe-path"
	width="300"
	height="300"
></iframe>
``` 

Ajoutez l'option `crossdomain` suivante au constructeur de votre tracker :

```js
const tracker = new OpenReplay({
	projectKey: PROJECT_KEY,
	crossdomain: {
		enabled: true
	},
	captureIFrames: true
});
```

Ensuite, initialisez l'instance du tracker à l'intérieur de l'iFrame souhaité et ajoutez `crossdomain.parentDomain` à son constructeur :

```js
const tracker = new OpenReplay({
	projectKey: PROJECT_KEY,
	crossdomain: {
	  // Specifies the domain of the parent window
	  // If not specified, it will be set to '*'
	  // Which means that child window will send messages to all domains inside the browser window
	  // Consult your CSP settings to ensure that '*' is allowed or specify the parent domain
		parentDomain: '*',
	  enabled: true
	}
});
```

Veuillez noter que si vous utilisez le snippet du tracker, nous recommandons d'utiliser l'instance standard (openreplay.js) plutôt que `openreplay-assist.js` pour les iframes enfants (étant donné que la session assist ne doit se connecter qu'à l'instance parente).
