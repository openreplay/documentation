Suivez les tutoriels ci-dessous pour tirer le meilleur parti d'OpenReplay.

## Guides pratiques

Apprenez à partir des tutoriels ci-dessous en vous appuyant sur des cas d'usage concrets.

### Plugins
- [Créez vos propres plugins](/tutorials/build-plugins) : Apprenez à créer vos propres plugins personnalisés dans votre instance auto-hébergée d'OpenReplay.
- [Assist](/tutorials/assist) : Offrez la meilleure expérience d'assistance et d'intégration à vos utilisateurs finaux grâce à la relecture de session en direct.
- [Plugin Fetch](/tutorials/fetch) : Apprenez à configurer et à assainir les informations de requête à l'aide du plugin Fetch.
- [Plugin Axios](/tutorials/axios) : Apprenez à configurer et à assainir les informations de requête à l'aide du plugin Axios.
- [Plugin Redux](/tutorials/redux) : Apprenez à configurer le plugin Redux pour capturer les changements d'état dans vos relectures de session.
- [Plugin VueX](/tutorials/vuex) : Apprenez à configurer le plugin VueX pour capturer les changements d'état dans vos relectures de session.
- [Plugin GraphQL](/tutorials/graphql) : Apprenez à configurer le plugin GraphQL pour capturer les données d'opération dans vos relectures de session.
- [Plugin Zustand](/tutorials/zustand) : Suivi des données d'état depuis Zustand avec le plugin Zustand Tracker.
- [Plugin Pinia](/tutorials/pinia) : Suivi des données d'état depuis Pinia avec le plugin Pinia Tracker.

### Fonctionnalités

- [OmniSearch](/tutorials/omnisearch) : Profitez de notre puissante fonction de recherche pour trouver le bon type de session pour votre projet.
- [Ajouter des utilisateurs à votre équipe](/tutorials/invite-team-members) : Apprenez à ajouter des utilisateurs à votre instance auto-hébergée d'OpenReplay.
- [Types de problèmes](/tutorials/issues) : Découvrez la signification de tous les problèmes détectés automatiquement et comment en tirer le meilleur parti.
- [Améliorez votre capacité de suivi avec les événements personnalisés](/tutorials/custom-events) : Découvrez les événements personnalisés et comment décupler vos super-pouvoirs de suivi.
- Tableaux de bord :
   - [Création de tableaux de bord personnalisés](/tutorials/custom-dashboard) : Apprenez à exploiter les tableaux de bord personnalisés pour visualiser l'état actuel de votre système.
   - [Création d'entonnoirs personnalisés](/tutorials/funnels) : Apprenez à créer des entonnoirs personnalisés pour comprendre ce qui provoque la chute de vos conversions.
   - [Afficher les erreurs JavaScript sur un tableau de bord personnalisé](/tutorials/js-errors-dashboard) : Ajoutez une métrique personnalisée (erreurs JS) à vos tableaux de bord grâce à ce tutoriel.
- [Spot](/tutorials/spot) : Enregistrez les bugs directement depuis votre navigateur, avec tout le contexte dont les développeurs ont besoin pour les corriger.
