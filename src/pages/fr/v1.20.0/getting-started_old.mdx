:::

:::caution[Utilisation avec pnpm]
En raison de problèmes en amont dans Prettier, le plugin ne sera pas détecté automatiquement lors de l'utilisation de [pnpm](https://pnpm.io/). Pour qu'il le trouve, le paramètre suivant doit être ajouté lors de l'exécution de Prettier :

```shell
prettier --write --plugin-search-dir=. .
```

Des réglages supplémentaires sont également requis lors de l'utilisation de Prettier dans VS Code. Consultez le README du plugin pour plus d'informations.
:::
