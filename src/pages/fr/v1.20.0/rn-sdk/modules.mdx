Le SDK React Native exporte plusieurs composants et méthodes pour vous permettre de personnaliser votre enregistrement :

## Connector

```js
interface IORTrackerConnector {
  startSession: (
    projectKey: string,
    optionsDict: Options,
    projectUrl?: string
  ) => void;
  setMetadata: (key: string, value: string) => void;
  event: (name: string, payload?: string) => void;
  setUserID: (userID: string) => void;
  userAnonymousID: (userID: string) => void;
}
```

- [startSession()](/rn-sdk/start) : démarre l'enregistrement.
- [setMetadata()](/rn-sdk/set-metadata) : Définit un champ de métadonnées et l'enregistre dans le cadre de l'enregistrement.
- [event()](/rn-sdk/event) : Envoie des événements personnalisés afin qu'ils soient enregistrés dans le cadre de vos relectures avec cette méthode.
- [setUserID()](/rn-sdk/set-user-id) : Identifiez vos utilisateurs avec cette méthode.
- [patchNetwork()](/rn-sdk/patch-network) : Applique un correctif à fetch et/ou XMLHttpRequest pour enregistrer les requêtes réseau.

## Components

- [ORTouchTrackingView](/rn-sdk/touch) : Vue utilisée pour suivre les événements tactiles.
- [ORSanitizedView](/rn-sdk/sanitization) : Vue qui vous permet d'assainir (ou de flouter) des parties de votre enregistrement
- [ORTrackedInput](/rn-sdk/inputs) : TextInput qui enregistre les événements de saisie
- [ORAnalyticsView](/rn-sdk/analytics) : Vue marquée qui envoie un événement une fois qu'elle apparaît à l'écran et lorsqu'elle le quitte
