Intégrez Microsoft Teams à OpenReplay et partagez vos analyses avec le reste de l'équipe, directement depuis la page de l'enregistrement.

### 1. Créez un jeton d'accès personnel

1. Connectez-vous à votre espace de travail Microsoft Teams puis allez dans l'onglet Teams > votre canal spécifique
2. Appuyez sur les ... à côté du nom de votre canal puis sélectionnez Connecteurs
3. Dans la liste des connecteurs, recherchez 'Incoming Webhook' et sélectionnez configurer
3. Allez sur la page 'Incoming Webhooks' de votre application nouvellement créée puis 'Activate Incoming Webhooks'
4. Définissez le nom sur OpenReplay (vous pouvez utiliser cette <a href="https://docs.openreplay.com/static/OpenReplay.png" target="_blank">icône</a>)
5. Appuyez sur 'Create' puis copiez l'URL du Webhook

### 2. Activez Microsoft Teams dans OpenReplay

1. Allez dans le tableau de bord OpenReplay puis 'Preferences > Integration'
2. Sélectionnez `MS Teams` et collez l'URL du Webhook copiée précédemment avant de cliquer sur 'Add'
3. Si tout s'est bien passé, vous devriez recevoir un message de bienvenue sur Microsoft Teams

Vous pouvez désormais aussi utiliser le bouton de partage dans n'importe quel enregistrement et communiquer vos analyses au reste de l'équipe.
