Cette méthode définit la fonction de rappel (callback) à utiliser lorsque les Feature Flags sont chargés.

## Signature

```js
onFlagsLoad(callback: (flags: IFeatureFlag[]) => void): void
```

## Arguments

Fonction de rappel à utiliser une fois que les feature flags sont rechargés. Exécutée à chaque événement de rechargement des flags.
