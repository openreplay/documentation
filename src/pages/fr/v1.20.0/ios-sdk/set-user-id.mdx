Cette méthode vous permet de définir un ID utilisateur sur la session actuellement active. Cela vous permet d'associer un utilisateur à l'enregistrement, ce qui vous permettrait de filtrer et de rechercher les sessions d'un utilisateur en particulier.

## Signature
```swift
ORTracker.shared.setUserID(_ userID: String)
```

## Paramètres

- `userID: String` : L'ID de l'utilisateur. Il s'agit d'une chaîne de caractères et peut être n'importe quoi, du moment qu'il s'agit d'une chaîne et qu'elle vous permet d'identifier de façon unique un utilisateur au sein de votre système (par exemple, e-mail, id de base de données, etc.).

## Valeur de retour

Cette méthode ne renvoie aucune valeur ; les résultats seront visibles sur la plateforme OpenReplay.
