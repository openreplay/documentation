Si vous souhaitez ajouter le tracker à votre application web et que vous utilisez un framework que nous ne prenons pas en charge, ou que vous
utilisez simplement VanillaJS, vous pouvez ajouter directement notre extrait JavaScript.

## Obtenir l'extrait

L'extrait JavaScript dont vous aurez besoin pour accomplir cette tâche vous est fourni lorsque vous configurez le projet sur le portail OpenReplay.
Une fois que vous avez ajouté votre nouveau projet, vous verrez quelque chose comme ceci :

![Nouveau projet prêt](/using-or/snippet/new-project-ready.png)

Lorsque vous voyez cela, cliquez sur le bouton « Complete Project Setup ».
Cela devrait vous amener à une page où l'extrait JS vous est présenté :


![Obtenir l'extrait JS](/using-or/snippet/getting-snippet.png)

Maintenant, cliquez simplement sur l'onglet « Script » et vous verrez l'extrait JavaScript réel :

![L'extrait est visible](/using-or/snippet/snippet-visible.png)

Récupérez le script et ajoutez-le directement à l'intérieur de votre balise `<head>`, et le tracker commencera à fonctionner automatiquement.

<Aside type="tip">
Gardez à l'esprit que si vous modifiez l'une des options d'enregistrement de données à l'étape 1 de la capture d'écran ci-dessus, l'extrait final changera.
Veillez donc à d'abord configurer vos options d'enregistrement, puis à copier l'extrait réel.
</Aside>
