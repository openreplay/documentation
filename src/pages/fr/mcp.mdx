L'app **OpenReplay MCP** connecte votre instance OpenReplay à un hôte de chat compatible MCP (Claude Desktop, Claude Code, ChatGPT, Codex) afin que vous puissiez explorer l'analytique produit et les rejeux de session en langage naturel. Demandez un graphique, explorez un entonnoir, trouvez une session, puis ouvrez son rejeu, le tout depuis le chat.

Elle fonctionne aussi bien avec **OpenReplay Cloud** qu'avec les instances **auto-hébergées**.

<Aside type="tip" title="Ce que vous obtenez" >
  - Recherchez des sessions avec les mêmes filtres que dans l'interface OpenReplay.
  - Affichez des séries temporelles, des entonnoirs, des parcours utilisateurs, les web vitals et des classements top-of-X de façon intégrée.
  - Regardez les rejeux de session directement dans le chat.
  - Enchaînez le tout : rechercher → choisir une session → ouvrir le rejeu → demander à Claude de le résumer.
</Aside>

## Compatibilité

L'app parle le MCP standard via stdio, elle fonctionne donc avec :

- **Claude Desktop** (macOS, Windows)
- **Claude Code**
- **ChatGPT** (via le connecteur MCP)
- **Codex**
- Tout autre hôte qui prend en charge MCP
