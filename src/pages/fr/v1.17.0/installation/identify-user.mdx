Associez votre id-utilisateur-interne à la session en cours d'enregistrement en suivant les étapes ci-dessous.

### Avec NPM

Injectez le userID au démarrage du tracker :

```js
const tracker = new OpenReplay({
  projectKey: PROJECT_KEY
});

tracker.start({
  userID: "john@doe.com",
  metadata: {
    balance: "10M",
    plan: "free"
  }
});
```

Si cela n'est pas possible (le userID n'est connu que plus tard dans le flux de navigation, donc bien après le démarrage du tracker), appelez alors la méthode `setUserID` pour identifier vos utilisateurs. L'identité de l'utilisateur peut être modifiée à tout moment pendant la session en appelant `setUserID`. Cependant, OpenReplay ne conservera que le dernier ID utilisateur injecté.

```js
tracker.setUserID('john@doe.com'); // after tracker.start()
```

### Avec l'extrait de code JavaScript

Voici un exemple montrant comment injecter le userID à l'aide de l'extrait de code. Notez la variable/ligne `startOpts` permettant d'injecter le userID.

```js
<!-- OpenReplay Tracking Code -->
<script>
var initOpts = { projectKey: "GxPpaDARdn2345fgt321" };
var startOpts = { userID: getUserID() }; // use startOpts variable to inject userID
(function(A,s,a,y,e,r){
  r=window.OpenReplay=[e,r,y,[s-1, e]];
  s=document.createElement('script');s.src=A;s.async=!a;
  ...
})("//static.openreplay.com/latest/openreplay.js", 1, 0, initOpts, startOpts);
</script>
```

Si cela n'est pas possible (le userID n'est connu que plus tard dans le flux de navigation, donc bien après le démarrage du tracker), appelez alors plutôt la méthode `setUserID` pour identifier vos utilisateurs. L'identité de l'utilisateur peut être modifiée à tout moment pendant la session en appelant `setUserID`. Cependant, OpenReplay ne conservera que le dernier ID utilisateur injecté.

```js
<!-- OpenReplay Tracking Code -->
<script>
var initOpts = { projectKey: "GxPpaDARdn2345fgt321" };
var startOpts = { userID: "" };
(function(A,s,a,y,e,r){
  r=window.OpenReplay=[e,r,y,[s-1, e]];
  s=document.createElement('script');s.src=A;s.async=!a;
  ...
})("//static.openreplay.com/latest/openreplay.js", 1, 0, initOpts, startOpts);
</script>
  ...
  OpenReplay.setUserID("john@doe.com"); // set the userID later in your code
  ...
```

### Avec l'application iOS

Appelez simplement `setUserID(String)` à n'importe quel moment pendant votre relecture.

```swift
import ORTracker

// ... your code

ORTracker.shared.setUserID(_ userID: String)
```
