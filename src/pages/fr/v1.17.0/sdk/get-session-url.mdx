Cette méthode vous permet d'obtenir l'URL de la session active. L'URL pointe vers le tableau de bord d'OpenReplay et cette méthode est utile si vous souhaitez intégrer OpenReplay dans votre propre stack.

## Signature

```js
getSessionURL(withCurrentTime?: boolean): string | undefined
```

## Paramètres

- `withCurrentTime: boolean` : Paramètre optionnel, dont la valeur par défaut est `false`. Si `true`, l'URL contiendra l'horodatage du moment, dans la relecture, où la méthode a été appelée.

## Valeur de retour

Cette méthode renvoie l'URL de la session active à rejouer sur la plateforme d'OpenReplay. Facultativement, l'URL peut contenir l'horodatage du moment où la méthode est appelée afin de sauter directement à cet endroit de la relecture.
