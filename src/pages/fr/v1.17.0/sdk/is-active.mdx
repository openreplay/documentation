Cette fonction peut être utilisée pour déterminer si une instance particulière du tracker est active et en cours d'enregistrement ou non.

## Signature

```js
isActive(): boolean
```

### Valeur de retour

Renvoie `true` si le tracker est actif et que la session est en cours d'enregistrement, et `false` sinon.
