Copiez/collez le code du tracker depuis le tableau de bord OpenReplay, sous 'Preferences > Projects'. Le code est unique pour chaque projet.

![Tracking Code](/static/GTM-5.JPG#center)

Ensuite, accédez à votre compte Google Tag Manager et suivez les étapes ci-dessous :

1. Cliquez sur *Add a new tag*

![Add New Tag](/static/GTM-3.png#center)

2. *Choose a tag type* pour commencer la configuration
3. Sélectionnez *Custom HTML* dans la liste, puis collez l'extrait copié dans le champ de texte

![Custom HTML](/static/GTM-4.JPG#center)

4. Cliquez sur *Choose a trigger* pour définir des actions et sélectionnez **All Pages**

![All Pages](/static/GTM-7.JPG#center)

5. Nommez la balise puis cliquez sur *Save*
6. *Submit* vos modifications

![Submit](/static/submit.JPG#center)

7. Assurez-vous que votre environnement en production est sélectionné, puis appuyez sur *Publish*

![Publish](/static/GTM-10.png#center)
