Associe un identifiant utilisateur anonyme à la session en cours d'enregistrement.

## Signature
```swift
ORTracker.shared.userAnonymousID(_ userID: String)
```

## Paramètres

- `userID: String` : L'identifiant anonyme de l'utilisateur. Il s'agit d'une chaîne de caractères et peut être n'importe quoi, du moment qu'il s'agit d'une chaîne, et il vous permet d'identifier de manière unique un utilisateur au sein de votre système (par exemple, e-mail, identifiant de base de données, etc.).

## Valeur de retour

Cette méthode ne renvoie aucune valeur ; les résultats seront visibles sur la plateforme OpenReplay.
