Associe un identifiant utilisateur anonyme à la session en cours d'enregistrement.

## Signature
```kotlin
OpenReplay.userAnonymousID(userID: String)
```

## Paramètres

- `userID: String` : L'identifiant anonyme de l'utilisateur. Il s'agit d'une chaîne de caractères et peut être n'importe quoi, du moment que c'est une chaîne et qu'elle vous permet d'identifier de manière unique un utilisateur au sein de votre système (c'est-à-dire e-mail, id de base de données, etc.).

## Valeur de retour

Cette méthode ne renvoie aucune valeur ; les résultats seront visibles dans la plateforme OpenReplay.
