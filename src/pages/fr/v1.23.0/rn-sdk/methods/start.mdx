Cette méthode est utilisée pour démarrer le tracker. En démarrant le tracker, l'enregistrement de la session commence.

## Signature

```swift
import OR from '@openreplay/react-native'

OR.tracker.startSession(options)
```

## Options d'initialisation

Par défaut, toutes les options sont définies sur `true` :
- `crashes: Bool` Active crashlytics.
- `analytics: Bool` Active le suivi analytique des vues marquées.
- `performances: Bool` Active le listener de performance
- `logs: Bool` Active le listener de logs
- `screen: Bool` Active l'enregistreur d'écran

Les options peuvent être utilisées pour désactiver certaines fonctionnalités ; sinon, vous pouvez passer l'objet vide (`{}`)
