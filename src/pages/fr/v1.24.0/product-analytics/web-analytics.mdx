Web Analytics, dans le module Analytics d'OpenReplay, vous aide à suivre des métriques globales sur l'utilisation de votre site web ou de votre application. Une fois votre tracker configuré et les sessions capturées, vous pouvez surveiller des données telles que les pages les plus consultées, la localisation des utilisateurs et les sources de trafic afin de mieux comprendre les tendances générales d'utilisation.

## **Que pouvez-vous apprendre avec Web Analytics ?**

Avec Web Analytics, vous pouvez répondre à des questions telles que :

- Quelles sont les pages les plus visitées ?
- Quels navigateurs et appareils sont les plus utilisés ?
- Où se trouvent vos utilisateurs ?
- Quelles sont les principales sources de trafic vers votre site ?
- Qui sont vos utilisateurs les plus actifs ?

Ces informations vous aident à identifier des tendances, à optimiser le contenu et à prioriser les améliorations.

##  Configurer votre tableau de bord Web Analytics

1. **Créer un tableau de bord** :
    - Accédez au menu **Dashboards**.
    - Cliquez sur **Add Dashboard** et nommez-le **Web Analytics**.

##  Approfondir les informations de Web Analytics

Prenons la carte **Top Browsers** comme exemple :

1. Ouvrez la carte **Top Browsers** dans votre tableau de bord.
2. Supposons que vous remarquiez qu'un pourcentage important d'utilisateurs sont sur Chrome. Cliquez sur **Chrome** pour filtrer les sessions de ces utilisateurs spécifiquement.
3. Utilisez le composant **Omnisearch** pour aller plus loin :
    - Appliquez des filtres tels que des événements spécifiques (par exemple, des clics) ou des attributs de session pour affiner davantage votre vue.
    - Par exemple, trouvez les sessions des utilisateurs de Chrome ayant rencontré une erreur spécifique.

En vous concentrant sur les données propres à chaque navigateur, vous pouvez révéler des tendances ou résoudre plus efficacement les problèmes liés aux navigateurs.

**Ajuster la plage de temps**

Par défaut, les tableaux de bord affichent les données des 7 derniers jours. Pour ajuster cela :

1. Cliquez sur le **menu déroulant de plage de temps** (intitulé **'Last 7 days'**).
2. Choisissez une plage prédéfinie (par exemple, hier) ou définissez une période personnalisée.

##  **Création alternative de cartes**

Vous pouvez également créer des cartes directement depuis la **Vue Cartes** :

1. Accédez à **Cards View** dans le module Analytics.
2. Créez une nouvelle carte pour n'importe quelle métrique Web Analytics.
3. Cliquez sur **Add to Dashboard** et sélectionnez le tableau de bord **Web Analytics**.

##  Tirez le meilleur parti de Web Analytics

Web Analytics vous donne un accès rapide à des métriques globales qui vous aident à suivre les tendances d'utilisation de votre application. Associez-la à des outils tels que les rejeux de session et les filtres pour révéler des informations plus approfondies et passer à l'action.
