Avec cette méthode, vous pouvez définir la valeur de l'un des champs de métadonnées configurés sur la plateforme OpenReplay.

<Aside type="note">
	Vous pouvez appeler cette méthode autant de fois que vous le souhaitez, gardez simplement à l'esprit que si vous utilisez le même attribut `key`, la dernière valeur que vous avez transmise sera enregistrée dans la plateforme. Toutes les valeurs précédentes seront perdues.
</Aside>

Consultez [métadonnées](/installation/metadata) pour plus de détails. 

## Signature
```kotlin
OpenReplay.setMetadata(key: String, value: String)
```

## Paramètres

- `key: string` : La clé de métadonnées à enregistrer. 
- `value: string` : La valeur à enregistrer en tant que métadonnée.

## Valeur de retour

Cette méthode ne renvoie aucune valeur. L'information est enregistrée directement dans le cadre de la relecture et peut être consultée dans la plateforme OpenReplay.
