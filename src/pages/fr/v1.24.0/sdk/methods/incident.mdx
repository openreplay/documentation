Cette méthode vous permet de marquer un événement important ou un incident pendant la session d'un utilisateur. Les incidents sont généralement utilisés pour mettre en évidence des occurrences importantes, telles que des erreurs, des plantages ou d'autres événements notables qui nécessitent une attention. En marquant ces incidents, vous pouvez facilement les identifier et les analyser dans la relecture de session.

## Signature

```js
// call incident function and pass label, startTime, and endTime (label and endTime are optional)
incident({ label: 'incident', startTime: Date.now() - 1000, endTime: Date.now() })
```

## Paramètres

- `label: string` : Le nom de l'événement marqué.
- `startTime: number` : Horodatage signalant le début de l'événement.
- `endTime: number` : Horodatage de la fin de l'événement.
