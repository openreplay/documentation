Cette méthode vous permet d'obtenir l'URL de la session active. L'URL correspond au tableau de bord d'OpenReplay et cette méthode est utile si vous souhaitez intégrer OpenReplay à votre propre stack.

## Signature

```js
getSessionURL({ withCurrentTime?: boolean }): string | undefined
```

## Paramètres

- `withCurrentTime: boolean` : Paramètre facultatif, par défaut il vaut `false`. S'il est `true`, l'URL contiendra l'horodatage du moment, à l'intérieur du replay, où la méthode a été appelée.

## Valeur de retour

Cette méthode renvoie l'URL de la session active à rejouer sur la plateforme d'OpenReplay. Optionnellement, l'URL peut contenir l'horodatage du moment où la méthode est appelée afin de sauter directement plus loin dans le replay.
