Votre instance OpenReplay peut être configurée facilement. Voici des tutoriels étape par étape pour activer les services de messagerie, sécuriser votre installation et supprimer des enregistrements de la base de données.

## Guides étape par étape
- [Configurer SMTP](/configuration/configure-smtp) : Configurez SMTP pour recevoir des alertes, des rapports hebdomadaires, inviter de nouveaux utilisateurs à OpenReplay et réinitialiser les mots de passe.
- [Sécuriser OpenReplay](/configuration/secure-or) : Sécurisez OpenReplay en configurant le SSL, reCaptcha et en définissant une CSP.
- [Paramètres du proxy](/configuration/proxy-settings) : Exécutez OpenReplay derrière un proxy comme NGINX ou Apache2.
- [Nettoyer le stockage](/configuration/cleanup-storage) : Comment les enregistrements sont stockés, et comment les supprimer en masse de la base de données et nettoyer votre stockage.
- [Stockage externe des enregistrements](/configuration/external-storage) : Modifiez la destination de stockage par défaut de vos enregistrements.
- [Base de données externe](/configuration/external-db) : Utilisez une base de données Postgres externe pour OpenReplay.
- [Authentification unique (SAML)](/configuration/sso) : Configurez l'authentification unique (SSO) via SAML.
