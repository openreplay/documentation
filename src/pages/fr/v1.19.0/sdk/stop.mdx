---

Cette méthode arrête le tracker et la session active. Si vous n'utilisez pas cette méthode, la session sera arrêtée automatiquement quelques minutes après que vous ayez fermé l'onglet ou quitté votre site.

## Signature

```js
stop(): string | undefined
```

## Valeur de retour

Renvoie le `sessionHash` de la session arrêtée.
