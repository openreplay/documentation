Le tracker écoutera l'événement de changement de valeur (avec debounce) et enverra la valeur obtenue au backend d'OpenReplay.

## Ajouter des saisies suivies

Le SDK exporte `ORTrackedInput`, que vous pouvez utiliser comme des composants TextInput classiques.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORTrackedInput
    style={styles.input}
    onChangeText={onChangeNumber}
    value={number}
    placeholder="important placeholder"
  />

// ...
```
