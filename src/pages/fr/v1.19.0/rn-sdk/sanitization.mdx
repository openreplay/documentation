## Composants de l'interface

Les vues assainies apparaîtront floutées dans le replay résultant. Vous pouvez ajouter n'importe quel composant d'interface et il sera automatiquement supprimé (pour préserver la mémoire) dès qu'il n'est plus à l'écran.

### ORSanitizedView

Le SDK exporte `ORSanitizedView`, que vous pouvez utiliser pour envelopper vos composants sensibles afin de les flouter.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORSanitizedView>
    <SensitiveComponent />
  </OR.ORSanitizedView>

// ...
```
