> **Remarque :** Cette intégration n'est disponible que pour les utilisateurs d'OpenReplay Cloud.

Segment vous permet de collecter les données de vos utilisateurs depuis chaque source dans une seule API exportable. Vous pouvez ensuite envoyer ces informations à tous vos outils préférés instantanément.

Avec cette intégration, vous n'avez aucun code à ajouter à votre site. Si vous avez un compte Segment, tout ce que vous avez à faire est de définir votre ProjectKey OpenReplay, puis d'activer l'intégration comme indiqué dans les instructions ci-dessous.

## 1. Sélectionnez OpenReplay dans la liste des intégrations disponibles

Recherchez simplement OpenReplay dans la liste des intégrations disponibles, puis sélectionnez-le.

![OpenReplay dans la liste des intégrations](/static/segment-1.png#center)

## 2. Définissez votre ProjectKey puis activez l'intégration

Votre ProjectKey se trouve dans votre tableau de bord OpenReplay sous 'Preferences > Projects', puis appuyez sur *Tracking Code*. Une fois ajouté, il vous suffit d'actionner l'interrupteur pour activer l'intégration.

![Paramètres OpenReplay](/static/segment-2.png#center)

## 3. Traits utilisateur Segment (Identify)

Découvrez comment ajouter des [Métadonnées dans OpenReplay](/installation/metadata).

## 4. C'est terminé !

Bravo ! Il ne reste plus qu'à attendre que les modifications se propagent dans le CDN de Segment. Cela devrait prendre environ 10 minutes. En attendant, vous pouvez supprimer le code de suivi d'OpenReplay sur votre site, s'il y en a un.
