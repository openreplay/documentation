如何将 Jira Cloud 与 OpenReplay 集成。

## 1. 创建新的 API 令牌

1. 登录你的 [JIRA 令牌](https://id.atlassian.com/manage/api-tokens)。
2. 点击 *Create API token*。
3. 在标签中输入 “openreplay”，然后点击 *Create*。
4. 点击 *Copy to clipboard*。

## 2. 在 OpenReplay 中启用 JIRA Cloud

1. 进入 OpenReplay 控制面板的 'Preferences > Integration'
2. 选择 Jira。如果你没有看到它，那么你一定是已经启用了 GitHub（你不能同时使用两者，因此请务必先禁用 GitHub 集成）。
3. 填写你的 `Username` 和 `API token`，然后输入你的 Jira Cloud 实例的 URL

随后所有录制中都会出现一个 'Create Issue' 按钮，这样你就可以快速在 Jira 中创建并分配问题。
