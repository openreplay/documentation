您的 OpenReplay 实例可以轻松配置。以下是有关启用电子邮件服务、保护您的安装以及从数据库中删除录制内容的分步教程。

## 分步指南
- [Configure SMTP](/configuration/configure-smtp)：设置 SMTP 以接收警报、每周报告、邀请新用户加入 OpenReplay 以及重置密码。
- [Secure OpenReplay](/configuration/secure-or)：通过配置 SSL、reCaptcha 和设置 CSP 来保护 OpenReplay。
- [Proxy Settings](/configuration/proxy-settings)：在 NGINX 或 Apache2 等代理后面运行 OpenReplay。
- [Cleanup Storage](/configuration/cleanup-storage)：录制内容的存储方式，以及如何从数据库中批量删除并清理您的存储。
- [External Recordings' Storage](/configuration/external-storage)：更改录制内容的默认存储位置。
- [External Database](/configuration/external-db)：为 OpenReplay 使用外部 Postgres 数据库。
- [Single-Sign-On (SAML)](/configuration/sso)：通过 SAML 设置单点登录 (SSO)。
