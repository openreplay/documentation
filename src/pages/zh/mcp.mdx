**OpenReplay MCP** 应用将你的 OpenReplay 实例连接到支持 MCP 的聊天主机（Claude Desktop、Claude Code、ChatGPT、Codex），让你可以用自然语言探索产品分析和会话回放。请求一张图表、深入分析一个漏斗、找到一个会话，然后打开它的回放——全部都在聊天中完成。

它同时适用于 **OpenReplay Cloud** 和**自托管**实例。

<Aside type="tip" title="你能获得什么" >
  - 使用与 OpenReplay 界面中相同的筛选器搜索会话。
  - 内联渲染时间序列、漏斗、用户旅程、web vitals 以及 top-of-X 细分。
  - 直接在聊天中观看会话回放。
  - 将这一切串联起来：搜索 → 选择一个会话 → 打开回放 → 请 Claude 进行总结。
</Aside>

## 兼容性

该应用通过 stdio 使用标准 MCP，因此它适用于：

- **Claude Desktop**（macOS、Windows）
- **Claude Code**
- **ChatGPT**（通过 MCP 连接器）
- **Codex**
- 任何其他支持 MCP 的主机
