此方法允许你为当前活动的会话设置用户 ID。这样可以将某个用户与回放关联起来，从而让你能够筛选和搜索某个特定用户的会话。

## 签名

```js
import OR from '@openreplay/react-native'

// ...
OR.tracker.setUserID('id')
```


## 参数

- `userID: String`：用户的 ID。这是一个字符串，只要它是字符串并且能够让你在系统中唯一标识一个用户（即电子邮件、数据库 id 等），它可以是任何值。

## 返回值

此方法不返回任何值，结果将在 OpenReplay 平台中可见。
