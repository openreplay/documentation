跟踪器将监听值变化事件（已防抖处理），并将最终值发送到 OpenReplay 后端。

## 添加被跟踪的输入框

SDK 导出了 `ORTrackedInput`，你可以将其当作常规的 TextInput 组件使用。

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORTrackedInput
    style={styles.input}
    onChangeText={onChangeNumber}
    value={number}
    placeholder="important placeholder"
  />

// ...
```
