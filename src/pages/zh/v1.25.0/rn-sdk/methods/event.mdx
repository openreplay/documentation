此方法允许你上报一个“功能性事件”。通过功能性事件，你可以标记用户会话中那些在功能层面具有意义、但我们的追踪器默认无法自行捕获和标记的时刻。功能性事件的一个例子是用户将某件商品加入购物车。虽然我们的追踪器能够记录该操作，但它并不知道要将其标记为会话时间线中的相关事件。不过，借助此方法，你可以手动完成这一标记。

更多详情请参阅[自定义事件](/installation/custom-events#functionalevents)。

## 签名

```js
import OR from '@openreplay/react-native'

OR.tracker.event("name", JSON.stringify({ key: "value" }))
```

## 参数

- `key: string`：用于保存信息的键名。该键可在 OpenReplay 平台内用于过滤和搜索。
- `payload: string`：事件的内容。可以是任意经过字符串编码的（JSON）数据。
