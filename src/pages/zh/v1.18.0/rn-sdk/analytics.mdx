跟踪器将监听手机屏幕上视图的出现/消失事件。

## 添加被跟踪的视图

SDK 导出了 `ORAnalyticsView`，你可以用它来标记重要的视图/组件。

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORAnalyticsView
    screenName="Pay confirmation page"
    viewName="New fancy button"
  >
    <ImportantRedButton />
  </OR.ORAnalyticsView>

// ...
```
