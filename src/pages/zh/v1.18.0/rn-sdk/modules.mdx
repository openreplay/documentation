React Native SDK 导出了多个组件和方法，供你自定义录制：

## Connector

```js
interface IORTrackerConnector {
  startSession: (
    projectKey: string,
    optionsDict: Options,
    projectUrl?: string
  ) => void;
  setMetadata: (key: string, value: string) => void;
  event: (name: string, payload?: string) => void;
  setUserID: (userID: string) => void;
  userAnonymousID: (userID: string) => void;
}
```

- [startSession()](/rn-sdk/start)：开始录制。
- [setMetadata()](/rn-sdk/set-metadata)：设置一个元数据字段并将其作为录制的一部分保存。
- [event()](/rn-sdk/event)：使用此方法发送自定义事件，使其作为回放的一部分被记录下来。
- [setUserID()](/rn-sdk/set-user-id)：使用此方法识别你的用户。
- [patchNetwork()](/rn-sdk/patch-network)：对 fetch 和/或 XMLHttpRequest 打补丁以记录网络请求。

## Components

- [ORTouchTrackingView](/rn-sdk/touch)：用于追踪触摸事件的视图。
- [ORSanitizedView](/rn-sdk/sanitization)：让你能够对录制内容的部分进行脱敏（或模糊处理）的视图
- [ORTrackedInput](/rn-sdk/inputs)：记录输入事件的 TextInput
- [ORAnalyticsView](/rn-sdk/analytics)：被标记的视图，当它出现在屏幕上以及离开屏幕时会发送一个事件
