此方法用于启动追踪器。启动追踪器后，会话录制随即开始。

## 签名

```swift
import OR from '@openreplay/react-native'

OR.tracker.startSession(options)
```

## 初始化选项

默认情况下，所有选项都被设为 `true`：
- `crashes: Bool` 启用 crashlytics。
- `analytics: Bool` 启用对已标记视图的分析追踪。
- `performances: Bool` 启用性能监听器
- `logs: Bool` 启用日志监听器
- `screen: Bool` 启用屏幕录制器

可以使用这些选项来禁用某些功能；否则，你可以传入空对象（`{}`）
