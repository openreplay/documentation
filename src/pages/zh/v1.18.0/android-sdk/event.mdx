此方法允许你报告一个“功能事件”。通过功能事件，你可以标记用户会话中那些从功能角度来看具有重要意义的时刻，而这些时刻在默认情况下，我们的追踪器无法自行捕获并标记。功能事件的一个例子是用户将商品添加到购物车。虽然我们的追踪器能够记录该操作，但它并不知道要将其标记为会话时间线中的一个重要事件。不过，借助此方法，你可以手动完成这一标记。

更多详细信息请参阅[自定义事件](/installation/custom-events#functionalevents)。

## 签名

```swift
ORTracker.shared.event(name: String, payload: Encodable?)
```

## 参数

- `key: string`：用于保存信息的键名。该键可在 OpenReplay 平台内用于过滤和搜索。
- `payload: Encodable`：事件的内容。可以是任何可编码（JSON）数据。
