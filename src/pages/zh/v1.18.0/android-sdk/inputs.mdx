跟踪器将监听值变更事件（经过防抖处理），并将最终的值发送到 OpenReplay 后端。

## 添加被跟踪的输入

### SwiftUI

使用 `.trackTextInput(label:masked)` 来观察输入事件，
- `label: String` - 记录输入所用的标签
- `masked: Bool` - 如果为 `true`，跟踪器将仅记录输入事件本身，而不记录其值。

```kotlin
import com.opnereplay.tracker.OpenReplay

val passwordEditText = binding.password
passwordEditText.trackTextInput(label = "password", masked = true)

```

### View

```kotlin
import com.opnereplay.listeners.Analytics

Analytics.addObservedInput(inputEl)
```
