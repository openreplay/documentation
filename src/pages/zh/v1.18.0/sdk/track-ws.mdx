如果在发生 JS 错误时我们的追踪器默认记录的信息还不够，你可以使用此方法。`handleError` 方法用于手动上报捕获到的异常、被拒绝的 Promise 或错误事件。更多详情请参阅[错误上报](/installation/error-reporting#manuallyloggingexceptions)。

## 签名

```ts 
trackWs: (channelName: string) => (msgType: string, data: string, dir: 'up' | 'down') => void
```

## 参数

- `channelName: string`：WS 通道的名称。

## 返回值

返回一个用于处理 ws 消息的钩子（hook），它以消息类型（或名称）、数据和消息方向（up/down === 发送/接收）作为参数。
