此方法用于设置在加载 Feature Flags 时所使用的回调函数。

## 签名

```js
onFlagsLoad(callback: (flags: IFeatureFlag[]) => void): void
```

## 参数

在 feature flags 重新加载后所使用的回调函数。在每次 flag 重新加载事件时执行。
