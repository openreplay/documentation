此方法允许你上报一个“功能性事件”。通过功能性事件，你可以标记用户会话中那些从功能角度来看具有相关性的时刻，而这些时刻在默认情况下是我们的追踪器无法自行捕捉并标记的。功能性事件的一个示例是用户将商品添加到购物车。虽然我们的追踪器能够记录该操作，但它并不知道要将其标记为你会话时间线中的一个相关事件。不过，借助此方法，你可以手动完成这一操作。

更多详情请参阅[自定义事件](/installation/custom-events#functionalevents)。

## 签名

```swift
ORTracker.shared.event(name: String, payload: Encodable?)
```

## 参数

- `key: string`：将用于保存信息的键名。该键可在 OpenReplay 平台内用于过滤和搜索。
- `payload: Encodable`：事件的内容。可以是任何可编码的（JSON）数据。
