> **注意：** 此集成仅对 OpenReplay Cloud 用户可用。

Segment 让你能够将来自各个来源的用户数据收集到一个可导出的 API 中。然后，你可以即时将这些信息发送到所有你喜爱的工具。

借助此集成，你无需向你的网站添加任何代码。如果你拥有 Segment 账户，你只需设置你的 OpenReplay ProjectKey，然后按照下面的说明启用此集成即可。

## 1. 从可用集成列表中选择 OpenReplay

只需在可用集成列表中搜索 OpenReplay，然后选择它。

![集成列表中的 OpenReplay](/static/segment-1.png#center)

## 2. 设置你的 ProjectKey 然后启用集成

你的 ProjectKey 可以在你的 OpenReplay 仪表盘中的 'Preferences > Projects' 下找到，然后点击 *Tracking Code*。添加完成后，你只需拨动开关即可激活此集成。

![OpenReplay 设置](/static/segment-2.png#center)

## 3. Segment 用户特征（Identify）

了解如何在 OpenReplay 中添加[元数据](/installation/metadata)。

## 4. 全部完成！

干得好！现在只需等待更改在 Segment 的 CDN 中传播即可。这大约需要 10 分钟。与此同时，如果你的网站上有 OpenReplay 的跟踪代码，你可以将其移除。
