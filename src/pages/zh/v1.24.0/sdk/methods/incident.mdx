此方法允许你在用户会话期间标记一个重要事件或一次事故。事故通常用于突出显示重要情况，例如错误、崩溃或其他需要关注的值得注意的事件。通过标记这些事故，你可以在会话回放中轻松识别和分析它们。

## 签名

```js
// call incident function and pass label, startTime, and endTime (label and endTime are optional)
incident({ label: 'incident', startTime: Date.now() - 1000, endTime: Date.now() })
```

## 参数

- `label: string`：所标记事件的名称。
- `startTime: number`：表示事件开始的时间戳。
- `endTime: number` ：事件结束的时间戳。
