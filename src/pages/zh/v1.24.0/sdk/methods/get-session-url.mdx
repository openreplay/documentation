此方法允许你获取当前活动会话的 URL。该 URL 指向 OpenReplay 的仪表板，如果你希望将 OpenReplay 集成到自己的技术栈中，此方法会非常有用。

## 签名

```js
getSessionURL({ withCurrentTime?: boolean }): string | undefined
```

## 参数

- `withCurrentTime: boolean`：可选参数，默认值为 `false`。如果设置为 `true`，URL 将包含调用该方法时在回放中所处时刻的时间戳。

## 返回值

此方法返回当前活动会话的 URL，用于在 OpenReplay 平台上进行回放。URL 还可以选择性地包含调用该方法时的时间戳，以便直接跳转到回放中靠后的位置。
