跟踪器将监听视图的出现/消失事件。

## 添加被跟踪的视图

### SwiftUI

使用 `.observeView(title:viewName:)` 来观察输入事件，

- `title: String` - 屏幕或功能的标题
- `viewName: String` - 跟踪器视图的名称

```swift
import ORTracker

TextField("Input", text: $text)
 .observeView(title: "Screen title", viewName: "test input name")
```

### UIKit
```swift
import ORTracker

Analytics.shared.addObservedView(view: inputEl, title: "Screen title", viewName: "test input name")
```
