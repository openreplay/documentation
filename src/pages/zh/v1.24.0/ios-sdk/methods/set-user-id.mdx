此方法允许你为当前活动的会话设置用户 ID。这样你就可以将某个用户与回放关联起来，从而能够筛选和搜索某个特定用户的会话。

## 签名
```swift
ORTracker.shared.setUserID(_ userID: String)
```

## 参数

- `userID: String`：用户的 ID。它是一个字符串，只要是字符串，可以是任何内容，并且能让你在系统中唯一地标识一个用户（例如电子邮件、数据库 id 等）。

## 返回值

此方法不返回任何值，结果将显示在 OpenReplay 平台上。
