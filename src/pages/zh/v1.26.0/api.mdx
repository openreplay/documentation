OpenReplay API 允许你管理用户数据并导出捕获的事件以供外部分析。

## 基础 URL（OpenReplay）

- 对于 Cloud Serverless 套餐：`https://api.openreplay.com/api/v1`
- 对于 Cloud Dedicated 或开源自托管实例：`https://YOUR_INSTANCE_DOMAIN/api/`

## HTTP API
- [项目](/api/projects)：用于创建新项目。
- [会话](/api/sessions)：用于检索指定用户的所有会话。
- [事件](/api/events)：用于拉取某次特定录制的所有事件。
- [用户](/api/users)：用于控制和删除用户数据。
- [任务](/api/jobs)：用于管理用户数据删除任务。
- [Assist](/api/assist)：用于检索实时会话。
