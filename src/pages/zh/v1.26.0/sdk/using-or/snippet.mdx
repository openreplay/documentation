如果您希望将 tracker 添加到您的 Web 应用程序中，而您使用的框架我们不支持，或者您只是
使用 VanillaJS，那么您可以直接添加我们的 JavaScript 代码片段。

## 获取代码片段

完成这项工作所需的 JavaScript 代码片段会在您于 OpenReplay 门户上设置项目时提供给您。
当您添加新项目后，您会看到类似下面的内容：

![New project ready](/using-or/snippet/new-project-ready.png)

看到之后，点击 "Complete Project Setup" 按钮。
这应该会带您进入一个页面，其中向您展示了 JS 代码片段：


![Getting the JS Snippet](/using-or/snippet/getting-snippet.png)

现在，只需点击 "Script" 选项卡，您就会看到实际的 JavaScript 代码片段：

![The snippet is visible](/using-or/snippet/snippet-visible.png)

获取该脚本并将其直接添加到您的 `<head>` 标签内，tracker 就会自动开始运行。

<Aside type="tip">
请记住，如果您在上面截图的第 1 步中更改了任何数据记录选项，最终的代码片段也会随之改变。
因此，请务必先配置好您的记录选项，然后再复制实际的代码片段。
</Aside>

## 有疑问？

如果您在使用 JavaScript 代码片段设置 tracker 时遇到任何问题，请通过我们的 [Slack 社区](https://slack.openreplay.com/) 联系我们，直接向我们的开发人员提问！
