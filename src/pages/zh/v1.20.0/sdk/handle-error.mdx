如果在发生 JS 错误时我们的 tracker 默认跟踪的信息不够用，你可以使用此方法。`handleError` 方法用于手动报告被捕获的异常、被拒绝的 promise 或错误事件。更多详情请参阅[错误报告](/installation/error-reporting#manuallyloggingexceptions)。

## 签名

```js 
handleError: (e: Error | ErrorEvent | PromiseRejectionEvent, metadata?: Record<string, any>) => void
```

## 参数

- `e: Error | ErrorEvent | PromiseRejectionEvent`：被捕获的实际错误对象。
- `metadata: Record<string, any>`：**[可选]** 一个类似字典的结构，包含你想要与此错误关联保存的所有额外信息。

## 返回值

此方法不返回任何值，保存的信息可以在 OpenReplay 平台中查看。
