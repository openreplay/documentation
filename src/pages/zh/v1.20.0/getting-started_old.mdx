:::caution[与 pnpm 一起使用]
由于 Prettier 上游的问题，使用 [pnpm](https://pnpm.io/) 时不会自动检测到该插件。为了让它找到该插件，运行 Prettier 时需要添加以下参数：

```shell
prettier --write --plugin-search-dir=. .
```

在 VS Code 中使用 Prettier 时还需要额外的设置。更多信息请参阅该插件的 README。
:::
