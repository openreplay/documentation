按照以下教程操作，充分利用 OpenReplay。

## 操作指南

通过以下基于真实使用场景的教程进行学习。

### 插件
- [创建你自己的插件](/tutorials/build-plugins)：了解如何在自托管的 OpenReplay 实例中构建自定义插件。
- [Assist](/tutorials/assist)：通过实时会话回放，为你的终端用户提供最佳的支持和引导体验。
- [Fetch 插件](/tutorials/fetch)：了解如何使用 Fetch 插件设置和净化请求信息。
- [Axios 插件](/tutorials/axios)：了解如何使用 Axios 插件设置和净化请求信息。
- [Redux 插件](/tutorials/redux)：了解如何设置 Redux 插件，以在会话回放中捕获状态变化。
- [VueX 插件](/tutorials/vuex)：了解如何设置 VueX 插件，以在会话回放中捕获状态变化。
- [GraphQL 插件](/tutorials/graphql)：了解如何设置 GraphQL 插件，以在会话回放中捕获操作数据。
- [Zustand 插件](/tutorials/zustand)：使用 Zustand Tracker 插件跟踪来自 Zustand 的状态数据。
- [Pinia 插件](/tutorials/pinia)：使用 Pinia Tracker 插件跟踪来自 Pinia 的状态数据。

### 功能

- [OmniSearch](/tutorials/omnisearch)：充分利用我们强大的搜索功能，为你的项目找到合适类型的会话。
- [将用户添加到你的团队](/tutorials/invite-team-members)：了解如何将用户添加到你的自托管 OpenReplay 实例中。
- [问题类型](/tutorials/issues)：了解所有自动检测到的问题的含义，以及如何充分利用它们。
- [使用自定义事件增强你的跟踪能力](/tutorials/custom-events)：了解自定义事件，以及如何提升你的跟踪超能力。
- 仪表板：
   - [创建自定义仪表板](/tutorials/custom-dashboard)：了解如何利用自定义仪表板来可视化你的系统当前状态。
   - [创建自定义漏斗](/tutorials/funnels)：了解如何创建自定义漏斗，以了解是什么导致了你的转化下降。
   - [在自定义仪表板上显示 JavaScript 错误](/tutorials/js-errors-dashboard)：通过本教程，将自定义指标（JS 错误）添加到你的仪表板中。
- [Spot](/tutorials/spot)：直接从你的浏览器录制 bug，并附带开发者修复所需的全部上下文。
