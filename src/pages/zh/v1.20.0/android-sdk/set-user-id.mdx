此方法允许你为当前活动的会话设置用户 ID。这样你就可以将用户与回放关联起来，从而能够针对某个特定用户筛选和搜索会话。

## 签名
```swift
OpenReplay.setUserID(userID: String)
```

## 参数

- `userID: String`：用户的 ID。这是一个字符串，只要是字符串即可，可以是任意内容，它允许你在你的系统中唯一地标识一个用户（即电子邮件、数据库 id 等）。

## 返回值

此方法不返回任何值，结果将显示在 OpenReplay 平台上。
