此方法用于启动追踪器。启动追踪器后，会话的录制随即开始。

## 签名

```swift
ORTracker.shared.start(projectKey: String, options: OROptions)
	```

### 初始化选项

可用选项：

- `crashes: Bool` 启用崩溃分析（crashlytics）。
- `analytics: Bool` 启用视图分析。
- `performances: Bool` 启用性能监听器
- `logs: Bool` 启用日志监听器
- `screen: Bool` 启用屏幕录制器
- `wifiOnly: Bool` 强制追踪器仅在用户处于 wifi 连接时才启动

默认值：全部为 `true`
