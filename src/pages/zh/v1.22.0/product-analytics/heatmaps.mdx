热力图以可视化方式呈现用户在您网站应用上的交互行为，突出显示点击和鼠标移动等高活跃度区域。它们帮助您识别哪些内容吸引了用户的注意、用户完全错过了哪些按钮或内容，以及哪些区域造成了使用障碍。

## 热力图的**使用场景**

使用热力图可以帮助您回答许多问题，例如：

- 在您的着陆页上，用户与哪些元素的交互最多？
- 关键的 CTA 是否被忽略了？
- 用户在导航栏的哪些位置进行点击？
- 用户在特定按钮或元素上是否出现了点击狂躁（click rage）？
- 您页面的哪些部分吸引了用户的注意，哪些被忽略了？

## 开始使用热力图

#### 创建新的热力图

您可以在 Cards 菜单中的 "Product Analytics" 类别下访问热力图。选择 "Heatmaps" 即可开始分析用户交互。

#### 热力图卡片的基本结构

1. **菜单栏：** 更改显示布局、重命名、删除热力图卡片，或将其添加到仪表板。
2. **URL 指定：** 输入您要分析的页面的 URL。如果您不确定，系统会根据您网站上最热门的页面提供建议。
3. **过滤器：** 应用诸如用户属性或会话特征之类的过滤器，以缩小数据范围。
4. **热力图区域：** 查看鼠标点击高亮和点击狂躁高亮。使用 "Get New Session" 按钮刷新屏幕截图。
5. **时间范围：** 将分析时段调整为过去 24 小时、过去 7 天或自定义范围。
6. **会话列表：** 查看会话以进行详细回放。

## 设置热力图

#### 输入热力图页面的 URL

使用 URL 字段指定您要分析的页面。可用选项包括：

- **URL 包含：** 显示所有包含所输入文本的 URL 的数据（区分大小写）。
- **URL 开头为：** 显示所有以所输入文本开头的 URL 的结果。
- **URL 结尾为：** 显示所有以所输入文本结尾的 URL 的结果。

#### 过滤您的热力图数据

过滤器帮助您优化数据，从而获得更具可操作性的洞察。可用的过滤器包括：
    - 用户属性（例如 User ID、userAnanymousId ）。
    - 会话特征（例如 国家、城市、操作系统、浏览器、UTM Source,...  ）。
    - 问题（例如 点击狂躁、无效点击、错误请求,...  ）。


##  充分利用热力图

1. 专注于高流量页面，以获得最具可操作性的洞察。
2. 使用过滤器按用户属性或会话参数对数据进行细分。
3. 将热力图洞察与会话回放相结合，以揭示用户行为背后的"为什么"。
