会话回放让你能够通过重现用户在应用中的操作过程来理解用户行为。借助 OpenReplay，你可以轻松排查问题、分析交互并改进你的产品。

## 指南
- [Omnisearch](/session-replay/omnisearch)：使用我们强大的搜索功能在会话中快速搜索。
- [精彩片段](/session-replay/highlights)：分离并保存会话回放中有意义的片段
- [将会话事件转化为 E2E 测试](/session-replay/generate-e2e-test)：将用户操作过程转化为 Cypress、Playwright 或 Puppeteer 中的 e2e 测试。

## 高级设置

- [识别用户](/session-replay/identify-user)：通过用户识别功能跨会话追踪单个用户。
- [发送元数据](/session-replay/metadata)：附加自定义元数据以丰富会话洞察。
- [Canvas 和 WebGL](/session-replay/canvas)：回放复杂的视觉内容，包括基于 Canvas 和 WebGL 的交互。
- [跨域 iFrame](/session-replay/crossdomain-iframe)：启用跨域 iFrame 追踪。
- [WebSockets](/session-replay/websockets)：  记录 WebSocket 通道并在会话回放中显示它们。
- [错误追踪](/session-replay/error-reporting)：直接从会话回放中识别并排查错误。
