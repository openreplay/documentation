## 隐私模式

**隐私模式**是一项新设置，可阻止记录所有文本和敏感数据。启用后，OpenReplay 跟踪器将不会捕获任何可见文本、网络数据或控制台日志。所有内容都会在源头被遮蔽，因此任何敏感数据都不会离开浏览器。

如果您希望在保持用户内容和数据完全私密的同时，全面掌握 UI 的行为，这将非常有用。

## 如何启用隐私模式

您可以在初始化 OpenReplay 时在跟踪器级别启用隐私模式。

#### 使用 NPM

```js
import { tracker } from '@openreplay/tracker'

tracker.configure({
    projectKey: PROJECT_KEY,
    privateMode: true
})
tracker.start();
```

当 `privateMode` 设置为 `true` 时，以下内容将被遮蔽：

- 页面上所有可见的文本  
- 图片的 `alt` 属性  
- 输入框中的 `placeholder` 值  
- 标签文本

无需额外设置。启用后，会话将记录布局和交互，但不会记录任何内容。


## 何时使用

在以下情况下使用 `privateMode`：

- 您需要确保完全的用户数据隐私
- 您正在**云环境**中测试 OpenReplay，但计划**稍后进行自托管**，并且**无法将任何真实数据发送到外部**
- 您在**隐私优先**的受监管行业中工作
