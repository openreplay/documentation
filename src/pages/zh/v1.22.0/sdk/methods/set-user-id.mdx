此方法允许你为当前活动的会话设置用户 ID。这样可以将某个用户与回放关联起来，从而让你能够针对某个特定用户筛选和搜索会话。

## 签名
```js
setUserID(id: string): void
```

## 参数

- `id: string`：用户的 ID。它是一个字符串，只要是字符串就可以是任何内容，它让你能够在系统中唯一地标识某个用户（即电子邮件、数据库 id 等）。

## 返回值

此方法不返回任何值，结果将在 OpenReplay 平台中可见。
