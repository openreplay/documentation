跟踪器将监听值变更事件（已防抖），并将最终的值发送到 OpenReplay 后端。

## 添加被跟踪的输入

### SwiftUI

使用 `.observeInput(text:label:masked)` 来观察输入事件，

- `text: String` - 输入的文本值
- `label: String` - 已记录输入的标签
- `masked: Bool` - 如果为 `true`，跟踪器将只记录输入事件本身，而不记录其值；别忘了添加 `.sensitive` 以在录制中模糊该输入。

```swift
import ORTracker

TextField("Input", text: $text)
.observeInput(text: $text, label: "tracker input #1", masked: Bool)

```

### UIKit

UIKit 将使用 placeholder 作为标签，并使用 sender.isSecureTextEntry 来遮蔽输入

```swift
import ORTracker

Analytics.shared.addObservedInput(inputEl)
```
