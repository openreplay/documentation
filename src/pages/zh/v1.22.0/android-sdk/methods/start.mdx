此方法用于启动 tracker。启动 tracker 后，会话录制随即开始。

## 签名

```kotlin
OpenReplay.start(applicationContext: Context, projectKey: String, options: OROptions)
	```

### 初始化选项

可用选项：

- `crashes: Bool` 启用 crashlytics。
- `analytics: Bool` 为视图启用分析。
- `performances: Bool` 启用性能监听器
- `logs: Bool` 启用日志监听器
- `screen: Bool` 启用屏幕录制器
- `wifiOnly: Bool` 强制 tracker 仅在用户拥有 wifi 连接时才启动

默认值：全部为 `true`
