此方法会重新加载所有 flag（如果用户会话的元数据、userId 或某些 flag 条件所需的任何其他信息发生了变化，这会很有用）。

## 签名

```js
reloadFlags(): Promise<void>
```

## 返回值

返回一个空的 promise，成功时执行 onFlagsCb
