此方法允许你跟踪各个 websocket 通道以及消息的方向（传入/传出）。

## 签名

```ts 
trackWs: (channelName: string) => (msgType: string, data: string, dir: 'up' | 'down') => void
```

## 参数

- `channelName: string`：WS 通道的名称。

## 返回值

返回一个用于应用到 ws 消息的 hook，该 hook 以消息类型（或名称）、数据和消息方向（up/down === 已发送/已接收）作为参数。
