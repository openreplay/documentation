## UI 组件

经过脱敏的视图将在生成的回放中被模糊处理。你可以添加任何 UI 组件，一旦它不再显示在屏幕上，就会被自动移除（以节省内存）。

### ORSanitizedView

SDK 导出了 `ORSanitizedView`，你可以用它来包裹敏感组件，从而将其模糊处理。

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORSanitizedView>
    <SensitiveComponent />
  </OR.ORSanitizedView>

// ...
```
