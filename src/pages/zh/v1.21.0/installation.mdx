本节提供了分类的操作指南，帮助你设置和配置 OpenReplay，涵盖从基础安装到高级配置的各个方面。

## 用户设置

- [如何识别用户](/installation/identify-user)：将你的内部用户 ID 与正在录制的会话关联起来。
- [如何发送元数据](/installation/metadata)：元数据也称为特征或用户变量，在录制会话时提供有关用户的额外信息。

## 会话回放设置

- [如何发送自定义事件](/installation/custom-events)：通过在会话回放中同步记录特定领域的事件来增加更多智能。
- [如何配置 Canvas 和 WebGL](/installation/canvas)：了解如何为你的项目集成和配置 Canvas 和 WebGL。
- [如何设置跨域 iFrame](/installation/crossdomain-iframe)：配置跨域 iFrame 以实现无缝集成。
- [如何配置 WebSockets](/installation/websockets)：在你的应用程序中设置 WebSockets 以进行实时通信。

## 网络与数据隐私

- [如何自定义网络选项（Web）](/installation/network-options)：为你的 Web 环境自定义网络配置。
- [如何自定义网络选项（移动端）](/installation/network-options-mobile)：为你的 iOS 和 Android 环境自定义网络配置。
- [如何清理数据（Web）](/installation/sanitize-data)：如何在全局和/或细粒度级别为你的 Web 应用程序清理数据。
- [如何清理数据（移动端）](/installation/sanitize-data-mobile)：如何为 iOS 和 Android 应用程序清理数据。

## 调试与错误跟踪

- [如何进行错误跟踪](/installation/error-reporting)：跟踪错误和异常，并在完整上下文中排查问题。
- [如何上传 Source-Maps](/installation/upload-sourcemaps)：通过查看从堆栈跟踪中获取的原始形式的源代码上下文，让调试更加轻松。
