如果您想将跟踪器添加到您的 Web 应用程序中，而您使用的框架是我们不支持的，或者您只是
使用 VanillaJS，您可以直接添加我们的 JavaScript 代码片段。

## 获取代码片段

完成此任务所需的 JavaScript 代码片段会在您于 OpenReplay 门户中设置项目时提供给您。
添加新项目后，您将看到类似如下的内容：

![新项目已就绪](/using-or/snippet/new-project-ready.png)

看到这个之后，点击 "Complete Project Setup" 按钮。
这会将您带到一个向您展示 JS 代码片段的页面：


![获取 JS 代码片段](/using-or/snippet/getting-snippet.png)

现在，只需点击 "Script" 选项卡，您就会看到实际的 JavaScript 代码片段：

![代码片段可见](/using-or/snippet/snippet-visible.png)

获取该脚本并将其直接添加到您的 `<head>` 标签内，跟踪器将自动开始工作。

<Aside type="tip">
请记住，如果您更改上面截图中第 1 步的任何数据录制选项，最终的代码片段也会随之改变。
因此，请确保先配置好您的录制选项，然后再复制实际的代码片段。
</Aside>
