将匿名用户 ID 与正在录制的会话关联起来。

## 签名
```kotlin
OpenReplay.userAnonymousID(userID: String)
```

## 参数

- `userID: String`：用户的匿名 ID。它是一个字符串，只要是字符串并且能让你在系统中唯一标识一个用户即可（即电子邮件、数据库 id 等），其值可以是任意内容。

## 返回值

此方法不返回任何值，结果将在 OpenReplay 平台中显示。
