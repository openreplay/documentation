使用此方法，你可以设置在 OpenReplay 平台上配置的某个元数据字段的值。

<Aside type="note">
你可以根据需要多次调用此方法，但请记住，如果你使用相同的 `key` 属性，最后一次传入的值将被保存到平台中。之前的所有值都将丢失。
</Aside>

更多详情请参阅 [metadata](/installation/metadata)。 

## 签名
```js
setMetadata(key: string, value: string): void
```

## 参数

- `key: string`：要保存的元数据键。 
- `value: string`：要作为元数据保存的值。

## 返回值

此方法不返回任何值。该信息会直接作为回放的一部分保存，并可在 OpenReplay 平台中查看。
