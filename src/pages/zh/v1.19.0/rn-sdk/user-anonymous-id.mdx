将匿名 userID 与正在录制的会话关联起来。

## 签名

```swift
ORTracker.shared.userAnonymousID(_ userID: String)
```

## 参数

- `userID: String`：用户的匿名 ID。这是一个字符串，只要它是字符串并且能让你在系统中唯一标识一个用户（即电子邮件、数据库 id 等），它可以是任何内容。

## 返回值

此方法不返回任何值，结果将在 OpenReplay 平台中可见。
