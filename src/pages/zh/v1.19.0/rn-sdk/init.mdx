## 安装

以下命令将为您安装跟踪器及随附的 SDK，以便您充分利用跟踪器的所有功能。

```bash
npm i @openreplay/react-native
```

## 初始化

在实例化 OpenReplay 的跟踪器时，您可以提供多个配置选项来自定义录制及录制体验的各个方面。您必须在构造函数中设置 `projectKey` 选项。您可以在 OpenReplay 仪表板的 'Preferences > Projects' 下获取该值。

<Aside type="tip">
	请注意，如果您有多个项目，需要谨慎选择项目密钥。
</Aside>


### 添加跟踪代码

将以下代码添加到您的根文件中：

```js
import Openreplay from '@openreplay/react-native';

// ...
useEffect(() => {
	OpenReplay.tracker.startSession(
		'yourProjectKey',
		options, // explained below, set {} if empty
		'https://local.openreplay.instance/ingest' // not required if you're using managed version
	);
}, [])
```

### 数据脱敏

有关更多详情，请参阅[如何在 React Native 回放中进行数据脱敏](/rn-sdk/sanitization)。

## 初始化选项

默认情况下，所有选项均标记为 `true`：

- `crashes: Bool` 启用 crashlytics。
- `analytics: Bool` 启用对标记视图的分析跟踪。
- `performances: Bool` 启用性能监听器。
- `logs: Bool` 启用日志监听器。
- `screen: Bool` 启用屏幕录制器。
