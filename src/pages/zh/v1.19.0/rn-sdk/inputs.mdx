跟踪器将监听值变更事件（经过去抖处理），并将得到的值发送到 OpenReplay 后端。

## 添加被跟踪的输入

SDK 导出了 `ORTrackedInput`，你可以像使用常规的 TextInput 组件那样使用它。

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORTrackedInput
    style={styles.input}
    onChangeText={onChangeNumber}
    value={number}
    placeholder="important placeholder"
  />

// ...
```
