从 OpenReplay 仪表板的 'Preferences > Projects' 下复制并粘贴跟踪器代码。每个项目的代码都是唯一的。

![Tracking Code](/static/GTM-5.JPG#center)

然后进入你的 Google Tag Manager 账户，并按照以下步骤操作：

1. 点击 *Add a new tag*

![Add New Tag](/static/GTM-3.png#center)

2. *Choose a tag type* 以开始设置
3. 从列表中选择 *Custom HTML*，然后将复制的代码片段粘贴到文本字段中

![Custom HTML](/static/GTM-4.JPG#center)

4. 点击 *Choose a trigger* 来设置一些操作，并选择 **All Pages**

![All Pages](/static/GTM-7.JPG#center)

5. 为标签命名，然后点击 *Save*
6. *Submit* 你的更改

![Submit](/static/submit.JPG#center)

7. 确保已选择你的生产环境，然后点击 *Publish*

![Publish](/static/GTM-10.png#center)
