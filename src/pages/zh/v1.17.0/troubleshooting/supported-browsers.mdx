OpenReplay 支持以下浏览器：
- Google Chrome 41+
- Firefox 17+
- Microsoft Edge 12+
- Internet Explorer 11+
- Safari 9+
- iOS 上的 Safari 9+
- Opera 28+
- Opera Mobile 62+

如果你在不受支持的浏览器中录制会话，SDK 将会抛出异常。

## 有疑问？
如果你对浏览器兼容性还有其他疑问，欢迎通过我们的 [Slack](https://slack.openreplay.com) 联系我们，或查看我们的[论坛](https://forum.openreplay.com)。
