将匿名 userID 与正在录制的会话相关联。

## 签名
```swift
ORTracker.shared.userAnonymousID(_ userID: String)
```

## 参数

- `userID: String`：用户的匿名 ID。它是一个字符串，只要是字符串即可为任意值，它能让你在你的系统中唯一标识一个用户（例如电子邮件、数据库 id 等）。

## 返回值

此方法不返回任何值，结果将在 OpenReplay 平台中可见。
