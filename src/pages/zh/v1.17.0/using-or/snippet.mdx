如果您想将跟踪器添加到您的 Web 应用程序中，而您使用的框架我们尚不支持，或者您只是
使用 VanillaJS，那么您可以直接添加我们的 JavaScript 代码片段。

## 获取代码片段

完成这项工作所需的 JavaScript 代码片段，会在您于 OpenReplay 门户上设置项目时提供给您。
添加新项目后，您会看到类似这样的内容：

![New project ready](/using-or/snippet/new-project-ready.png)

看到之后，点击 "Complete Project Setup" 按钮。
这应该会带您进入一个页面，页面上会显示 JS 代码片段：


![Getting the JS Snippet](/using-or/snippet/getting-snippet.png)

现在，只需点击 "Script" 选项卡，您就会看到实际的 JavaScript 代码片段：

![The snippet is visible](/using-or/snippet/snippet-visible.png)

获取该脚本并将其直接添加到您的 `<head>` 标签内，跟踪器就会自动开始工作。

<Aside type="tip">
请记住，如果您更改了上方截图中第 1 步的任何数据记录选项，最终的代码片段也会随之改变。
因此，请务必先配置好您的记录选项，然后再复制实际的代码片段。
</Aside>
