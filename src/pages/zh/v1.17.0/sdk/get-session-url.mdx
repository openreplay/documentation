此方法可让你获取当前活动会话的 URL。该 URL 指向 OpenReplay 的仪表盘，如果你希望将 OpenReplay 集成到自己的技术栈中，此方法会很有用。

## 签名

```js
getSessionURL(withCurrentTime?: boolean): string | undefined
```

## 参数

- `withCurrentTime: boolean`：可选参数，默认值为 `false`。如果为 `true`，该 URL 将包含调用此方法时在回放中所处时刻的时间戳。

## 返回值

此方法返回当前活动会话的 URL，用于在 OpenReplay 平台上进行回放。可选地，该 URL 可以包含调用此方法时刻的时间戳，以便直接跳转到回放中的相应位置。
