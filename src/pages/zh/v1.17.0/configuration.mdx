您的 OpenReplay 实例可以轻松配置。以下是分步教程，介绍如何启用电子邮件服务、保护您的安装以及从数据库中删除录制。

## 分步指南
- [配置 SMTP](/configuration/configure-smtp)：设置 SMTP 以接收告警、每周报告、邀请新用户加入 OpenReplay 以及重置密码。
- [保护 OpenReplay](/configuration/secure-or)：通过配置 SSL、reCaptcha 和设置 CSP 来保护 OpenReplay。
- [代理设置](/configuration/proxy-settings)：在 NGINX 或 Apache2 等代理后运行 OpenReplay。
- [清理存储](/configuration/cleanup-storage)：录制如何存储，以及如何从数据库中批量删除并清理您的存储。
- [外部录制存储](/configuration/external-storage)：更改录制的默认存储位置。
- [外部数据库](/configuration/external-db)：为 OpenReplay 使用外部 Postgres 数据库。
- [单点登录 (SAML)](/configuration/sso)：通过 SAML 设置单点登录 (SSO)。
