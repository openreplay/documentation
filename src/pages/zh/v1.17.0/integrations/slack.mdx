将 Slack 与 OpenReplay 集成，直接从录制页面与团队其他成员分享洞察。

## 使用 OpenReplay OpenSource

### 1. 创建个人访问令牌

1. 登录你的 Slack 工作区，然后前往[应用创建页面](https://api.slack.com/apps?new_app=1)
2. 将其命名为 "OpenReplay" 并将其连接到你的工作区
3. 前往你新创建应用的 'Incoming Webhooks' 页面，然后点击 'Activate Incoming Webhooks'
4. 点击 'Add New Webhook to Workspace'，并选择你将用于分享洞察的频道
5. 点击 'Allow'，然后复制 Webhook URL

### 2. 在 OpenReplay 中启用 Slack

1. 前往 OpenReplay 仪表板，然后进入 'Preferences > Integration'
2. 选择 Slack，并在点击 'Add' 之前粘贴之前复制的 Webhook URL
3. 如果一切顺利，你应该会在 Slack 上收到一条欢迎消息

现在你还可以在任意录制中使用分享按钮，将你的洞察传达给团队其他成员。

## 使用 OpenReplay Cloud

前往 'Preferences > Integration' 并点击 Slack。你将被重定向到 Slack 进行身份验证和配置。这就是你需要做的全部操作。之后，你便可以在任意录制中使用分享按钮，将你的洞察传达给团队其他成员。
