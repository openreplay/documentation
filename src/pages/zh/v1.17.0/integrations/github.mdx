将 GitHub 与 OpenReplay 集成，直接从录制页面创建 issue。

## 使用 OpenReplay OpenSource

### 1. 创建个人访问令牌

1. 登录 Github，然后前往 [Settings > Developers Settings > Personal access tokens](https://github.com/settings/tokens)
2. 点击 'Generate a new token'
3. 选择 `repo:status`、`repo_deployment`、`public_repo`、`read:user` 和 `user:email` 作用域
4. 点击 'Generate token'，然后复制该令牌

### 2. 在 OpenReplay 中启用 GitHub

1. 前往 OpenReplay 仪表板，然后进入 'Preferences > Integration'
2. 选择 GitHub。如果看不到它，那么你一定已经启用了 Jira（你无法同时使用两者，因此请确保先禁用 Jira 集成）。
3. 粘贴之前生成的 `token`

之后所有录制中都会出现一个 'Create Issue' 按钮，让你可以快速在 GitHub 中创建并分配 issue。

## 使用 OpenReplay Cloud

前往 'Preferences > Integration' 并点击 GitHub。你将被重定向到 GitHub 进行身份验证。这就是你需要做的全部操作。之后所有录制中都会出现一个 'Create Issue' 按钮，让你可以快速在 GitHub 中创建并分配 issue。
