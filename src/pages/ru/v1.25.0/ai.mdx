OpenReplay представляет **нативные возможности ИИ** для воспроизведения сессий, благодаря чему анализировать сессии, выявлять проблемы и получать аналитику без ручных усилий стало проще, чем когда-либо. ИИ ускоряет обнаружение проблем, анализ поведения пользователей и генерацию аналитики, используя различные модели для конкретных задач.

<Aside type="ai" title="Примечание" >
  - На данный момент доступно только пользователям **Cloud (Serverless)**.
  - В будущем Private Cloud Compute позволит пользователям **Self-hosted** и **Cloud (Dedicated)** получить доступ к этому набору функций ИИ.
</Aside>
