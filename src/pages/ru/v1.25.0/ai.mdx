OpenReplay представляет **нативные возможности ИИ** для воспроизведения сессий, благодаря чему анализировать сессии, выявлять проблемы и получать аналитику без ручных усилий стало проще, чем когда-либо. ИИ ускоряет обнаружение проблем, анализ поведения пользователей и генерацию аналитики, используя различные модели для конкретных задач.

<Aside type="ai" title="Примечание" >
  - На данный момент доступно только пользователям **Cloud (Serverless)**.
  - В будущем Private Cloud Compute позволит пользователям **Self-hosted** и **Cloud (Dedicated)** получить доступ к этому набору функций ИИ.
</Aside>


## Набор функций ИИ

- [Summary AI](/ai/summary-ai): Получайте краткие сводки сессий, дающие сжатый обзор ключевых взаимодействий и событий.
- [Similar Sessions](/ai/similar-sessions): Функция в один клик для поиска записей сессий, похожих на воспроизводимую в данный момент, с использованием моделей эмбеддингов на основе ИИ.
- [Explain AI](/ai/explain-ai): Разбирайтесь в сетевых запросах и журналах консоли (в частности, ошибках) в DevTools в каждой записи сессии.
- [Smart Search](/ai/smart-search): Используйте панель поиска на основе ИИ, чтобы писать запросы на естественном языке и получать релевантные фильтры для поиска конкретных сессий.
- [Clips](/ai/clips): Короткие клипы, выделяющие ключевые моменты записей сессий, с использованием моделей эмбеддингов на основе ИИ.
