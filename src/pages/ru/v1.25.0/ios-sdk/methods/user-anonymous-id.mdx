Связывает анонимный идентификатор пользователя с записываемой сессией.

## Сигнатура
```swift
ORTracker.shared.userAnonymousID(_ userID: String)
```

## Параметры

- `userID: String`: Анонимный идентификатор пользователя. Это строка, и она может быть любой, главное, чтобы это была строка, и она позволяет однозначно идентифицировать пользователя в вашей системе (например, электронная почта, идентификатор в базе данных и т. д.).

## Возвращаемое значение

Этот метод не возвращает никаких значений, результаты будут видны на платформе OpenReplay.
