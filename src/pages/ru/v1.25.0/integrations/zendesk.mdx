Как интегрировать Zendesk с OpenReplay.

## Установка

Самый простой способ интегрировать Zendesk с OpenReplay — использовать [приложение Text](https://www.zendesk.com/marketplace/apps/support/1691/text/), чтобы добавить ссылку на сессии пользователя на страницы ваших тикетов.

1. Установите бесплатное [приложение Text](https://www.zendesk.com/marketplace/apps/support/1691/text/) из маркетплейса Zendesk.
2. Укажите желаемый заголовок (например, "OpenReplay Recordings").
3. Для отображения текста вы можете использовать следующий формат, не забыв заменить `OPENREPLAY_DOMAIN` и `PROJECT_ID` на действительные Domain и Project ID (если вы не уверены, какие значения выбрать, посмотрите URL любой из ваших существующих записей: первая его часть — это домен, а второе число — Project ID):

```
https://<OPENREPLAY_DOMAIN>/<PROJECT_ID>/sessions?uid=is|{{ticket.requester.email}}&range=LAST_24_HOURS
```

4. Чтобы просмотреть ссылку, перейдите к тикету и откройте панель Apps, нажав на кнопку Apps в верхней правой части. Подробнее см. в [документации приложения Text](https://www.zendesk.com/marketplace/apps/support/1691/text/).

## Изменение формата

Вы можете протестировать формат вашей ссылки, перейдя в свой экземпляр OpenReplay и изменив критерии поиска, начав с фильтра UserID, а затем добавив любые важные события. Например, приведённый ниже запрос ищет все сессии, в которых был неудачный запрос и которые связаны с email пользователя тикета:

```
https://<OPENREPLAY_DOMAIN>/<PROJECT_ID>/sessions?uid=is|{{ticket.requester.email}}&iss=is|bad_request&range=LAST_24_HOURS
```
