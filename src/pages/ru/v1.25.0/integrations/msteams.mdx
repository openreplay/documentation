Интегрируйте Microsoft Teams с OpenReplay и делитесь аналитикой с остальной командой прямо со страницы записи.

## 1. Создайте персональный токен доступа

1. Войдите в свою рабочую область Microsoft Teams, затем перейдите на вкладку Teams > ваш конкретный канал
2. Нажмите на ... рядом с названием канала, затем выберите Connectors
3. В списке коннекторов найдите 'Incoming Webhook' и выберите configure 
3. Перейдите на страницу 'Incoming Webhooks' для вашего только что созданного приложения, затем 'Activate Incoming Webhooks'
4. Задайте имя OpenReplay (вы можете использовать эту <a href="https://docs.openreplay.com/static/OpenReplay.png" target="_blank">иконку</a>)
5. Нажмите 'Create', затем скопируйте URL вебхука

## 2. Включите Microsoft Teams в OpenReplay

1. Перейдите в панель управления OpenReplay, затем в 'Preferences > Integration'
2. Выберите `MS Teams` и вставьте ранее скопированный URL вебхука перед нажатием 'Add'
3. Если всё прошло успешно, вы должны получить приветственное сообщение в Microsoft Teams

Теперь вы также можете использовать кнопку «Поделиться» в любой записи и передавать свою аналитику остальной команде.
