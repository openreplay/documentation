Ваш экземпляр OpenReplay можно легко настроить. Здесь приведены пошаговые руководства по включению почтовых сервисов, защите вашей установки и удалению записей из базы данных.

## Пошаговые руководства
- [Configure SMTP](/configuration/configure-smtp): Настройте SMTP для получения оповещений, еженедельных отчётов, приглашения новых пользователей в OpenReplay и сброса паролей.
- [Secure OpenReplay](/configuration/secure-or): Защитите OpenReplay, настроив SSL, reCaptcha и установив CSP.
- [Proxy Settings](/configuration/proxy-settings): Запустите OpenReplay за прокси-сервером, таким как NGINX или Apache2.
- [Cleanup Storage](/configuration/cleanup-storage): Как хранятся записи и как массово удалять их из базы данных и очищать ваше хранилище.
- [External Recordings' Storage](/configuration/external-storage): Измените место хранения по умолчанию для ваших записей.
- [External Database](/configuration/external-db): Используйте внешнюю базу данных Postgres для OpenReplay.
- [Single-Sign-On (SAML)](/configuration/sso): Настройте единый вход (SSO) через SAML.
