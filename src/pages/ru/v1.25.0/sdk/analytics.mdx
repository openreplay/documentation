## SDK аналитики

Узнайте, как настроить SDK аналитики трекера OpenReplay.

## Модули

- [Пользователи](/sdk/analytics/people): Управление профилями и атрибутами пользователей.
- [События](/sdk/analytics/events): Отслеживание и анализ взаимодействий и событий пользователей.

Чтобы узнать больше об управлении данными аналитики в интерфейсе OpenReplay, посетите [Управление данными](/product-analytics/data-management).

## Сигнатуры верхнего уровня

```ts
// Identify a user with an id (e.g. email, username, etc.)
tracker.analytics.identify(userId: string): void
// same as
tracker.analytics.people.identify()

// Add event to batch with option to send it immediately
tracker.analytics.track(eventName: string, tempProperties?: Record<string, any>, options?: { send_immediately: boolean })
// same as
tracker.analytics.events.track()
```
