Этот метод позволяет получить URL активной сессии. URL ведёт на панель управления OpenReplay, и этот метод полезен, если вы хотите интегрировать OpenReplay в собственный стек.

## Сигнатура

```js
getSessionURL({ withCurrentTime?: boolean }): string | undefined
```

## Параметры

- `withCurrentTime: boolean`: Необязательный параметр, по умолчанию `false`. Если задано `true`, URL будет содержать метку времени того момента воспроизведения, в который был вызван метод.

## Возвращаемое значение

Этот метод возвращает URL активной сессии для воспроизведения на платформе OpenReplay. При необходимости URL может содержать метку времени момента, когда вызывается метод, чтобы сразу перейти к нужной точке в воспроизведении.
