**Summary AI** — это функция OpenReplay на основе искусственного интеллекта, которая создаёт краткие сводки записей сессий. Она помогает пользователям быстро понять ключевые события сессии, не просматривая всю запись целиком.


<Aside type="ai" title="Доступность" >
  - На данный момент доступно только пользователям Cloud (Serverless).
</Aside>


## Как использовать Summary AI
1. Перейдите к списку **Session Replays**.
2. Нажмите на сессию, чтобы открыть её запись.
3. Нажмите кнопку **Summary AI** рядом с разделом DevTools.
4. Просмотрите сгенерированную ИИ сводку сессии, чтобы быстро получить обзор поведения пользователя, а также проблем и ошибок.
