Начиная с версии v1.10.0 мы выпустили нашу утилиту командной строки (CLI). С её помощью наиболее распространённые задачи, которые вам приходилось выполнять для управления своей самостоятельно размещённой версией OpenReplay, теперь можно выполнить с помощью простой команды.


## Установка CLI

CLI поставляется с каждой самостоятельно размещённой версией OpenReplay.

## Использование CLI

CLI имеет следующий синтаксис:

```bash 
openreplay [command] <service>
```

Список доступных команд:

- `-h or --help`: Эта команда отобразит список доступных команд и сервисов.
- `-s or --status`: Если вы хотите узнать состояние развёрнутых сервисов, эта команда расскажет вам всё, что нужно знать.
- `-u or --upgrade`: Используется для обновления вашей установки до последней доступной версии.
- `-U or --deprecated-upgrade /path/to/old_vars.yaml`
- `-r or --restart`: Позволяет перезапустить все сервисы. Таким образом можно устранить неотвечающие сервисы.
- `-R or --Reload`: Перезагружает конфигурацию сервиса, поэтому используйте её только когда вы что-то изменили.
- `-p or --install-packages`: Обновляет все сторонние зависимости (такие как `kubectl`, `helm` и т.д.).
- `-i or --install domain.name.com`: Устанавливает OpenReplay на машину.
- `-e or --edit`: Редактирует установленную конфигурацию OpenReplay и перезагружает её.
- `-l or --logs SERVICE`: Показывает логи одного конкретного сервиса.

Список всех сервисов, доступных для CLI:

- alerts
- assets
- assist
- chalice
- db
- ender
- frontend
- heuristics
- http
- integrations
- ngix-controller
- peers
- sink
- sourcemapreader
- storage
- image-storage
- video-storage

## Пример

Чтобы получить состояние развёртывания, выполните приведённую ниже команду CLI:

```bash 
openreplay -s 
```
