С помощью этого метода вы можете задать значение одного из полей метаданных, настроенных на платформе OpenReplay.

<Aside type="note">
	Вы можете вызывать этот метод столько раз, сколько пожелаете, просто помните, что если вы используете один и тот же атрибут `key`, на платформе сохранится последнее переданное значение. Все предыдущие значения будут потеряны.
</Aside>

Подробнее см. в разделе [метаданные](/installation/metadata).

## Сигнатура
```js
import OR from '@openreplay/react-native'

// ...
OR.tracker.metadata('key', 'value')
```

## Параметры

- `key: string`: Ключ метаданных для сохранения.
- `value: string`: Значение для сохранения в качестве метаданных.

## Возвращаемое значение

Этот метод не возвращает никаких значений. Информация сохраняется непосредственно как часть записи и может быть просмотрена внутри платформы OpenReplay.
