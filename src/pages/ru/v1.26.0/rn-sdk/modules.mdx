React Native SDK экспортирует несколько модулей, позволяющих настроить вашу запись:

## Модули

- [ORTouchTrackingView](/rn-sdk/touch): Представление, используемое для отслеживания событий касания.
- [ORSanitizedView](/rn-sdk/sanitization): Представление, позволяющее очищать (или размывать) части вашей записи
- [ORTrackedInput](/rn-sdk/inputs): TextInput, который записывает события ввода
- [ORAnalyticsView](/rn-sdk/analytics): Помеченное представление, которое отправляет событие, как только оно появляется на экране, и когда оно его покидает
