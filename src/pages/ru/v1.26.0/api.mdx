API OpenReplay позволяет управлять данными пользователей и экспортировать захваченные события для внешнего анализа.

## Базовый URL (OpenReplay)

- Для плана Cloud Serverless: `https://api.openreplay.com/api/v1`
- Для Cloud Dedicated или самостоятельно размещаемых экземпляров с открытым исходным кодом: `https://YOUR_INSTANCE_DOMAIN/api/`

## HTTP API
- [Проекты](/api/projects): Для создания новых проектов.
- [Сессии](/api/sessions): Для получения всех сессий указанного пользователя.
- [События](/api/events): Для извлечения всех событий конкретной записи.
- [Пользователи](/api/users): Для управления и удаления данных пользователей.
- [Задачи](/api/jobs): Для управления задачами удаления данных пользователей.
- [Assist](/api/assist): Для получения живых сессий.
