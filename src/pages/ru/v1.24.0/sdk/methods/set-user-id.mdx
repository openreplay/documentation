Этот метод позволяет вам задать идентификатор пользователя для текущей активной сессии. Это позволяет связать пользователя с воспроизведением, что даёт возможность фильтровать и искать сессии конкретного пользователя.

## Сигнатура
```js
setUserID(id: string): void
```

## Параметры

- `id: string`: Идентификатор пользователя. Это строка, и она может быть любой, лишь бы это была строка, и она позволяет вам однозначно идентифицировать пользователя в вашей системе (то есть email, идентификатор в базе данных и т. д.).

## Возвращаемое значение

Этот метод не возвращает никаких значений, результаты будут видны на платформе OpenReplay.
