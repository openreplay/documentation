Трекер будет отслеживать события появления/исчезновения представлений на экране телефона.

## Добавление отслеживаемых представлений

SDK экспортирует `ORAnalyticsView`, который вы можете использовать для пометки важных представлений/компонентов.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORAnalyticsView
    screenName="Pay confirmation page"
    viewName="New fancy button"
  >
    <ImportantRedButton />
  </OR.ORAnalyticsView>

// ...
```
