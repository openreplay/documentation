## Компоненты UI

Очищенные представления будут размыты в итоговом воспроизведении. Вы можете добавить любой UI-компонент, и он будет автоматически удалён (для экономии памяти), как только перестанет отображаться на экране.

### ORSanitizedView

SDK экспортирует `ORSanitizedView`, который можно использовать, чтобы оборачивать ваши конфиденциальные компоненты и размывать их.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORSanitizedView>
    <SensitiveComponent />
  </OR.ORSanitizedView>

// ...
```
