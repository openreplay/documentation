## Методы

- [event()](/android-sdk/methods/event): Отправляйте пользовательские события, чтобы они записывались как часть ваших повторов, с помощью этого метода.
- [setUserID()](/android-sdk/methods/set-user-id): Идентифицируйте своих пользователей с помощью этого метода.
- [setMetadata()](/android-sdk/methods/set-metadata): Задайте поле метаданных и сохраните его как часть записи.
- [start()](/android-sdk/methods/start): Начните запись сессии.
