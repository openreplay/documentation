Трекер будет прослушивать события появления/исчезновения представлений.

## Добавление отслеживаемых представлений

### 

используйте `.observeView(title:viewName:)` для отслеживания событий ввода,

- `title: String` - заголовок экрана или функции
- `viewName: String` - имя представления трекера

```kotlin

TextField("Input", text: $text)
 .observeView(title: "Screen title", viewName: "test input name")
```
