OpenReplay позволяет управлять участниками команды, приглашая или удаляя их по мере необходимости. В этой статье описаны шаги по управлению участниками вашей команды.

## Приглашение участников команды

Чтобы пригласить участников команды:

1. Перейдите в **Preferences > Team** в панели управления OpenReplay.
2. Нажмите кнопку **Add Team Member**.
3. Заполните данные пользователя:
   - **Full Name**
   - **Email Address**
4. (Необязательно) Отметьте Admin Privileges, если пользователю нужен доступ к управлению командой и проектами.
5. (Только в Enterprise Edition) Назначьте роль, чтобы определить его доступ к проектам и функциям.
6. Нажмите **Invite**.

<div class="or-aside note">

1. **Для пользователей Self-Hosted**: первоначальная регистрация ограничена одним пользователем во время настройки. Дополнительных пользователей необходимо приглашать. Приглашения по электронной почте требуют настройки SMTP — без неё делитесь ссылками-приглашениями вручную. См. наше [руководство по SMTP](/configuration/configure-smtp).

2. **Для пользователей Cloud (Serverless или Dedicated)**: создатель аккаунта имеет роль **Owner** с правами администратора. Новые участники присоединяются как обычные Members, но им могут быть предоставлены **Admin Privileges**. Все приглашённые участники команды автоматически получают приглашения по электронной почте при добавлении

## Удаление участников команды

Чтобы удалить участника команды:

1. Перейдите в **Preferences > Team** в панели управления OpenReplay.
2. Найдите пользователя в списке команды.
3. Нажмите значок **Edit** рядом с его именем.
4. Нажмите значок **Delete**.
