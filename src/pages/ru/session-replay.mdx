Воспроизведение сессий позволяет понять поведение пользователей, воссоздавая их путь по вашему приложению. С OpenReplay вы можете легко устранять неполадки, анализировать взаимодействия и улучшать свой продукт.

## Руководства
- [Omnisearch](/session-replay/omnisearch): Быстрый поиск по всем сессиям с помощью нашей мощной функции поиска.
- [Сегменты](/session-replay/segments): Сохраняйте комбинации событий и фильтров в виде многократно используемых сегментов для поиска сессий.
- [Избранные фрагменты](/session-replay/highlights): Выделяйте и сохраняйте значимые фрагменты воспроизведения сессии.
- [От событий сессии к E2E-тестам](/session-replay/generate-e2e-test): Превращайте пути пользователей в e2e-тесты в Cypress, Playwright или Puppeteer.

## Расширенная настройка

- [Идентификация пользователя](/session-replay/identify-user): Отслеживайте отдельных пользователей в разных сессиях с помощью идентификации пользователей.
- [Отправка метаданных](/session-replay/metadata): Прикрепляйте пользовательские метаданные для обогащения аналитики по сессиям.
- [Canvas и WebGL](/session-replay/canvas): Воспроизводите сложный визуальный контент, включая взаимодействия на основе Canvas и WebGL.
- [Кросс-доменные iFrame](/session-replay/crossdomain-iframe): Включите отслеживание кросс-доменных iFrame.
- [WebSockets](/session-replay/websockets):  Записывайте каналы WebSocket и отображайте их в воспроизведении сессии.
- [Отслеживание ошибок](/session-replay/error-reporting): Выявляйте и устраняйте ошибки непосредственно из воспроизведений сессий.
