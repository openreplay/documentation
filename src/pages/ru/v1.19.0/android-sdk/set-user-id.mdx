Этот метод позволяет задать идентификатор пользователя для текущей активной сессии. Это позволяет связать пользователя с записью, что даёт возможность фильтровать и искать сессии конкретного пользователя.

## Сигнатура
```swift
OpenReplay.setUserID(userID: String)
```

## Параметры

- `userID: String`: Идентификатор пользователя. Это строка, и она может быть любой, пока она является строкой, и позволяет однозначно идентифицировать пользователя в вашей системе (например, email, идентификатор в базе данных и т. д.).

## Возвращаемое значение

Этот метод не возвращает никаких значений, результаты будут видны на платформе OpenReplay.
