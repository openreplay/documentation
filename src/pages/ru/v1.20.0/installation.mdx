В этом разделе собрано всё необходимое для начала работы с OpenReplay — от базовой до продвинутой настройки.

## Основные инструкции
- [Как идентифицировать пользователя](/installation/identify-user): Свяжите ваш внутренний идентификатор пользователя с записываемой сессией.
- [Как добавить пользователей в вашу команду](/tutorials/invite-team-members): Узнайте, как добавлять пользователей в вашу команду.
- [Как отправлять метаданные](/installation/metadata): Также называемые характеристиками или пользовательскими переменными, метаданные предоставляют дополнительную информацию о пользователях во время записи сессий.
- [Как отправлять пользовательские события](/installation/custom-events): Добавьте больше аналитики, записывая специфичные для предметной области события вместе с записями сессий.
- [Как настроить отчёты об ошибках](/installation/error-reporting): Сообщайте об ошибках и исключениях и просматривайте их на вкладке «Errors».
- [Как загрузить Source-Maps](/installation/upload-sourcemaps): Упростите отладку, просматривая контекст исходного кода, полученный из трассировок стека, в его исходном виде.
- [Как очищать данные](/installation/sanitize-data): Как очищать данные как на глобальном, так и на детальном уровне.
- [Как создать пользовательскую эвристику](/tutorials/custom-heuristics): Узнайте, как добавить пользовательскую логику в бэкенд для проверки специфического поведения пользователей.
- [Как использовать строку OmniSearch](/tutorials/omnisearch): Узнайте, как получить максимум от нашей мощной строки поиска.
- [Как создать собственные настраиваемые дашборды](/tutorials/custom-dashboard): Создавайте собственные метрики и дашборды с помощью этого руководства.
- [Как использовать OpenReplay Spot](/tutorials/spot): Записывайте баги прямо из браузера со всем контекстом, необходимым разработчикам для их исправления.
