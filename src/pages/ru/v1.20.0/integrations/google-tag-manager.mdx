Скопируйте и вставьте код трекера из панели управления OpenReplay в разделе 'Preferences > Projects'. Код уникален для каждого проекта.

![Tracking Code](/static/GTM-5.JPG#center)

Затем перейдите в свою учётную запись Google Tag Manager и выполните следующие шаги:

1. Нажмите *Add a new tag*

![Add New Tag](/static/GTM-3.png#center)

2. *Choose a tag type*, чтобы начать настройку
3. Выберите *Custom HTML* из списка, затем вставьте скопированный фрагмент в текстовое поле

![Custom HTML](/static/GTM-4.JPG#center)

4. Нажмите *Choose a trigger*, чтобы задать действия, и выберите **All Pages**

![All Pages](/static/GTM-7.JPG#center)

5. Назовите тег, а затем нажмите *Save*
6. *Submit* свои изменения

![Submit](/static/submit.JPG#center)

7. Убедитесь, что выбрано ваше рабочее окружение, затем нажмите *Publish*

![Publish](/static/GTM-10.png#center)
