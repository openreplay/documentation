## Методы и модули

- [Analytics](/android-sdk/analytics): Узнайте всё об отслеживании конкретных представлений.
- [Network](/android-sdk/network): Узнайте, как использовать NetworkListener.
- [Sanitization](/android-sdk/sanitization): Узнайте, как скрывать представления на записываемом экране.
- [Inputs](/android-sdk/inputs): Узнайте, как записывать события ввода.
- [event()](/android-sdk/event): Отправляйте пользовательские события, чтобы они записывались в составе ваших воспроизведений, с помощью этого метода.
- [setUserID()](/android-sdk/set-user-id): Идентифицируйте своих пользователей с помощью этого метода.
- [setMetadata()](/android-sdk/set-metadata): Задайте поле метаданных и сохраните его в составе записи.
- [start()](/android-sdk/start): Начните запись сессии.
- [stop()](/android-sdk/stop): Остановите запись сессии с помощью этого метода.
