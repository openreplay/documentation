С помощью этого метода вы можете установить значение одного из полей метаданных, настроенных на платформе OpenReplay.

<Aside type="note">
	Вы можете вызывать этот метод сколько угодно раз, просто помните, что если вы используете один и тот же атрибут `key`, в платформе будет сохранено последнее переданное значение. Все предыдущие значения будут потеряны.
</Aside>

Подробнее см. в разделе [метаданные](/installation/metadata). 

## Сигнатура
```kotlin
OpenReplay.setMetadata(key: String, value: String)
```

## Параметры

- `key: string`: Ключ метаданных для сохранения. 
- `value: string`: Значение, которое будет сохранено как метаданные.

## Возвращаемое значение

Этот метод не возвращает никаких значений. Информация сохраняется непосредственно как часть записи и может быть просмотрена внутри платформы OpenReplay.
