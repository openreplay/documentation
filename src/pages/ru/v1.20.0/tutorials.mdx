Следуйте приведённым ниже руководствам, чтобы получить максимум от OpenReplay.

## Инструкции

Учитесь на примерах из реальной практики с помощью приведённых ниже руководств.

### Плагины
- [Создайте собственные плагины](/tutorials/build-plugins): Узнайте, как создавать собственные пользовательские плагины в вашем самостоятельно размещённом экземпляре OpenReplay.
- [Assist](/tutorials/assist): Обеспечьте своим конечным пользователям наилучший опыт поддержки и адаптации с помощью воспроизведения сессий в реальном времени.
- [Плагин Fetch](/tutorials/fetch): Узнайте, как настроить и очистить информацию о запросах с помощью плагина Fetch.
- [Плагин Axios](/tutorials/axios): Узнайте, как настроить и очистить информацию о запросах с помощью плагина Axios.
- [Плагин Redux](/tutorials/redux): Узнайте, как настроить плагин Redux для захвата изменений состояния в записях ваших сессий.
- [Плагин VueX](/tutorials/vuex): Узнайте, как настроить плагин VueX для захвата изменений состояния в записях ваших сессий.
- [Плагин GraphQL](/tutorials/graphql): Узнайте, как настроить плагин GraphQL для захвата данных операций в записях ваших сессий.
- [Плагин Zustand](/tutorials/zustand): Отслеживание данных состояния из Zustand с помощью плагина Zustand Tracker.
- [Плагин Pinia](/tutorials/pinia): Отслеживание данных состояния из Pinia с помощью плагина Pinia Tracker.

### Возможности

- [OmniSearch](/tutorials/omnisearch): Воспользуйтесь нашей мощной функцией поиска, чтобы найти нужный тип сессии для вашего проекта.
- [Добавление пользователей в вашу команду](/tutorials/invite-team-members): Узнайте, как добавлять пользователей в ваш самостоятельно размещённый экземпляр OpenReplay.
- [Типы проблем](/tutorials/issues): Узнайте, что означают все автоматически обнаруживаемые проблемы и как получить от них максимум пользы.
- [Расширьте возможности отслеживания с помощью пользовательских событий](/tutorials/custom-events): Узнайте о пользовательских событиях и о том, как усилить свои возможности отслеживания.
- Дашборды:
   - [Создание пользовательских дашбордов](/tutorials/custom-dashboard): Узнайте, как использовать пользовательские дашборды для визуализации текущего состояния вашей системы.
   - [Создание пользовательской воронки](/tutorials/funnels): Узнайте, как создавать пользовательские воронки, чтобы понять, что вызывает падение конверсии.
   - [Отображение ошибок JavaScript на пользовательском дашборде](/tutorials/js-errors-dashboard): Добавьте пользовательскую метрику (ошибки JS) на ваши дашборды с помощью этого руководства.
- [Spot](/tutorials/spot): Записывайте баги прямо из вашего браузера со всем контекстом, который нужен разработчикам для их исправления.
