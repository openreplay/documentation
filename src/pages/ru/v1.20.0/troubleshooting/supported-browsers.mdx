OpenReplay поддерживается в следующих браузерах:
- Google Chrome 41+
- Firefox 17+
- Microsoft Edge 12+
- Internet Explorer 11+
- Safari 9+
- Safari на iOS 9+
- Opera 28+
- Opera Mobile 62+

Если вы записываете сессию в неподдерживаемом браузере, SDK выбросит исключение.
