**Умный поиск** — это функция поиска на основе ИИ в OpenReplay, которая позволяет пользователям формулировать запросы на естественном языке и мгновенно получать релевантные фильтры сессий. Это помогает находить конкретные сессии без необходимости вручную просматривать большие массивы данных.


<Aside type="caution" title="Доступность" >
  - На данный момент доступно только пользователям **Cloud (Serverless)**.
</Aside>

## Как открыть умный поиск
1. Перейдите к представлению списка **Session Replays**.
2. Нажмите на **строку умного поиска** в верхней части.
3. Введите запрос на естественном языке (например, *«Найти сессии с ошибками консоли»*).
4. ИИ автоматически создаст релевантные фильтры, чтобы сузить результаты сессий.
