Интегрируйте GitHub с OpenReplay и создавайте задачи прямо со страницы записи.

## Использование OpenReplay с открытым исходным кодом

#### 1. Создайте персональный токен доступа

1. Войдите в Github, затем перейдите в [Settings > Developers Settings > Personal access tokens](https://github.com/settings/tokens)
2. Нажмите 'Generate a new token'
3. Выберите области `repo:status`, `repo_deployment`,`public_repo`, `read:user` и `user:email`
4. Нажмите 'Generate token', затем скопируйте токен

#### 2. Включите GitHub в OpenReplay

1. Перейдите в панель управления OpenReplay, затем в 'Preferences > Integration'
2. Выберите GitHub. Если вы его не видите, значит у вас, должно быть, включена Jira (вы не можете использовать обе одновременно, поэтому сначала обязательно отключите интеграцию с Jira).
3. Вставьте ранее сгенерированный `token`

После этого во всех записях появится кнопка 'Create Issue', чтобы вы могли быстро создавать и назначать задачи в GitHub.

## Использование OpenReplay Cloud

Перейдите в 'Preferences > Integration' и нажмите на GitHub. Вы будете перенаправлены на GitHub для аутентификации. Это всё, что вам нужно сделать. После этого во всех записях появится кнопка 'Create Issue', чтобы вы могли быстро создавать и назначать задачи в GitHub.
