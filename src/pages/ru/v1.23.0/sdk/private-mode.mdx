## Приватный режим

**Приватный режим** — это новая настройка, которая блокирует запись всего текста и конфиденциальных данных. Когда он включён, трекер OpenReplay не захватывает никакой видимый текст, сетевые данные или логи консоли. Всё скрывается на стороне источника, поэтому никакие конфиденциальные данные никогда не покидают браузер.

Это полезно, если вы хотите иметь полную видимость поведения интерфейса, сохраняя при этом контент и данные пользователя полностью приватными.

## Как включить Приватный режим

Вы можете включить Приватный режим на уровне трекера при инициализации OpenReplay.

#### С помощью NPM

```js
import { tracker } from '@openreplay/tracker'

tracker.configure({
    projectKey: PROJECT_KEY,
    privateMode: true
})
tracker.start();
```

Когда для `privateMode` установлено значение `true`, скрывается следующее:

- Весь видимый текст на странице  
- Атрибуты `alt` у изображений  
- Значения `placeholder` в полях ввода  
- Тексты меток

- Сетевые запросы (URL, полезные данные, заголовки)  
- Логи консоли

Дополнительная настройка не требуется. После включения сессия будет записывать макет и взаимодействия, но ничего из контента.


## Когда его использовать

Используйте `privateMode`, если:

- Вам нужно обеспечить полную конфиденциальность данных пользователя
- Вы тестируете OpenReplay в **облачной среде**, но планируете **развернуть его на собственном сервере позже** и **не можете отправлять никакие реальные данные наружу**
- Вы работаете в регулируемых отраслях, где **приватность превыше всего**
