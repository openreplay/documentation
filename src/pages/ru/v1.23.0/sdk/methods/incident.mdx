Этот метод позволяет отметить значимое событие или инцидент во время сессии пользователя. Инциденты обычно используются для выделения важных событий, таких как ошибки, сбои или другие примечательные события, требующие внимания. Отмечая эти инциденты, вы можете легко находить и анализировать их в повторе сессии.

## Сигнатура

```js
// call incident function and pass label, startTime, and endTime (label and endTime are optional)
incident({ label: 'incident', startTime: Date.now() - 1000, endTime: Date.now() })
```

## Параметры

- `label: string`: Имя отмеченного события.
- `startTime: number`: Временная метка, обозначающая начало события.
- `endTime: number` : Временная метка окончания события.
