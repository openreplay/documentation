## Методы

- [event()](/ios-sdk/methods/event): Отправляйте пользовательские события для записи в составе ваших повторов с помощью этого метода.
- [setUserID()](/ios-sdk/methods/set-user-id): Идентифицируйте своих пользователей с помощью этого метода.
- [userAnonymousID()](/ios-sdk/methods/user-anonymous-id): Свяжите анонимный userID с записываемой сессией.
- [setMetadata()](/ios-sdk/methods/set-metadata): Задайте поле метаданных и сохраните его в составе записи.
- [start()](/ios-sdk/methods/start): Начните запись сессии.
- [stop()](/ios-sdk/methods/stop): Остановите запись сессии с помощью этого метода.
