Этот метод останавливает трекер и активную сессию. Если вы не используете этот метод, сессия будет остановлена автоматически через несколько минут после того, как вы закроете вкладку или покинете свой сайт.

## Сигнатура

```js
stop(): string | undefined
```

## Возвращаемое значение

Возвращает `sessionHash` остановленной сессии.
