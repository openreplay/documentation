Этот метод используется для запуска трекера. При запуске трекера начинается запись сессии.

## Сигнатура

```kotlin
OpenReplay.start(applicationContext: Context, projectKey: String, options: OROptions)
	```

### Параметры инициализации

Доступные параметры:

- `crashes: Bool` Включает crashlytics.
- `analytics: Bool` Включает аналитику для представлений.
- `performances: Bool` Включает слушатель производительности
- `logs: Bool` Включает слушатель логов
- `screen: Bool` Включает запись экрана
- `wifiOnly: Bool` Заставляет трекер запускаться только при наличии у пользователя wifi-соединения

По умолчанию: все `true`
