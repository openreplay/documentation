С помощью этого метода вы можете задать значение одного из полей метаданных, настроенных на платформе OpenReplay.

<Aside type="note">
	Вы можете вызывать этот метод сколько угодно раз, но имейте в виду, что если вы используете тот же атрибут `key`, на платформе будет сохранено последнее переданное вами значение. Все предыдущие значения будут потеряны.
</Aside>

Подробнее см. [metadata](/installation/metadata). 

## Сигнатура
```swift
ORTracker.shared.setMetadata(key: String, value: String)
```

## Параметры

- `key: string`: Ключ метаданных для сохранения. 
- `value: string`: Значение, которое нужно сохранить в качестве метаданных.

## Возвращаемое значение

Этот метод не возвращает никаких значений. Информация сохраняется непосредственно как часть записи и может быть просмотрена внутри платформы OpenReplay.
