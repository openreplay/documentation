Как интегрировать Jira Cloud с OpenReplay.

## 1. Создание нового токена API

1. Войдите в свой [токен JIRA](https://id.atlassian.com/manage/api-tokens).
2. Нажмите *Create API token*.
3. В поле метки введите «openreplay» и нажмите *Create*.
4. Нажмите *Copy to clipboard*.

## 2. Включение JIRA Cloud в OpenReplay

1. Перейдите в панель управления OpenReplay в раздел 'Preferences > Integration'
2. Выберите Jira. Если вы его не видите, значит, у вас должен быть включён GitHub (вы не можете использовать оба одновременно, поэтому сначала обязательно отключите интеграцию с GitHub).
3. Укажите свой `Username` и `API token`, затем введите URL вашего экземпляра Jira Cloud

После этого во всех записях появится кнопка 'Create Issue', чтобы вы могли быстро создавать и назначать задачи в Jira.
