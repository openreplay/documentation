Этот метод перезагружает все флаги (это может быть полезно, если в сессии пользователя произошли какие-либо изменения метаданных, userId или любой другой информации, которая может потребоваться для определённых условий флагов).

## Сигнатура

```js
reloadFlags(): Promise<void>
```

## Возвращаемое значение

Возвращает пустой промис, при успехе выполняет onFlagsCb
