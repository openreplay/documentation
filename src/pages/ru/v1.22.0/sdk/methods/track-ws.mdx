Этот метод позволяет отслеживать отдельные каналы websocket и направление сообщения (входящее/исходящее).

## Сигнатура

```ts 
trackWs: (channelName: string) => (msgType: string, data: string, dir: 'up' | 'down') => void
```

## Параметры

- `channelName: string`: Имя WS-канала.

## Возвращаемое значение

Возвращает хук для применения к ws-сообщениям, который принимает в качестве параметров тип сообщения (или имя), данные и направление сообщения (up/down === отправлено/получено).
