Заставляет трекер сбросить текущий пакет сообщений, не дожидаясь завершения. Полезно, когда необходимо закрыть вкладку или окно после сессии. Этот метод также отправляет сообщение в webworker, который запустит сетевой запрос с текущим пакетом сообщений.

## Сигнатура

```ts
forceFlushBatch(): void
```
