Приложение **OpenReplay MCP** подключает вашу инстанцию OpenReplay к хосту чата с поддержкой MCP (Claude Desktop, Claude Code, ChatGPT, Codex), чтобы вы могли исследовать продуктовую аналитику и воспроизведения сессий на естественном языке. Запросите график, углубитесь в воронку, найдите сессию, а затем откройте её воспроизведение — всё это прямо из чата.

Оно работает как с **OpenReplay Cloud**, так и с **самостоятельно размещёнными** инстанциями.

<Aside type="tip" title="Что вы получаете" >
  - Ищите сессии с теми же фильтрами, что и в интерфейсе OpenReplay.
  - Отображайте временные ряды, воронки, пользовательские пути, web vitals и разбивки top-of-X прямо в чате.
  - Смотрите воспроизведения сессий непосредственно в чате.
  - Объедините всё в цепочку: поиск → выбор сессии → открытие воспроизведения → попросите Claude составить сводку.
</Aside>

## Совместимость

Приложение использует стандартный MCP поверх stdio, поэтому оно работает с:

- **Claude Desktop** (macOS, Windows)
- **Claude Code**
- **ChatGPT** (через коннектор MCP)
- **Codex**
- Любым другим хостом, поддерживающим MCP
