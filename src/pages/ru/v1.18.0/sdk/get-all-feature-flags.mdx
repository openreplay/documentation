Этот метод возвращает все флаги (`IFeatureFlag[]`), которые активны в данный момент.

## Сигнатура

```js
getAllFeatureFlags(): IFeatureFlag[]
```

## Возвращаемое значение

Массив всех активных feature-флагов (IFeatureFlag)
