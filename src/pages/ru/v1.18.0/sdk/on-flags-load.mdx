Этот метод задаёт функцию обратного вызова, которая будет использоваться при загрузке Feature Flags.

## Сигнатура

```js
onFlagsLoad(callback: (flags: IFeatureFlag[]) => void): void
```

## Аргументы

Функция обратного вызова, которая будет использоваться после перезагрузки feature flags. Выполняется при каждом событии перезагрузки флагов.
