С помощью этого метода вы можете задать значение одного из полей метаданных, настроенных на платформе OpenReplay.

<Aside type="note">
Вы можете вызывать этот метод сколько угодно раз, только имейте в виду, что если вы используете один и тот же атрибут `key`, на платформе будет сохранено последнее переданное вами значение. Все предыдущие значения будут утеряны.
</Aside>

Подробнее см. в разделе [metadata](/installation/metadata). 

## Сигнатура
```js
setMetadata(key: string, value: string): void
```

## Параметры

- `key: string`: Ключ метаданных для сохранения. 
- `value: string`: Значение для сохранения в качестве метаданных.

## Возвращаемое значение

Этот метод не возвращает никаких значений. Информация сохраняется непосредственно как часть записи и может быть просмотрена на платформе OpenReplay.
