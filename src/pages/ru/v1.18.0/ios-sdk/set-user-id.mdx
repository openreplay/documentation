Этот метод позволяет задать идентификатор пользователя для активной в данный момент сессии. Это даёт возможность связать пользователя с записью, что позволит фильтровать и искать сессии конкретного пользователя.

## Сигнатура
```swift
ORTracker.shared.setUserID(_ userID: String)
```

## Параметры

- `userID: String`: Идентификатор пользователя. Это строка, и она может быть любой, лишь бы оставалась строкой и позволяла однозначно идентифицировать пользователя в вашей системе (например, email, идентификатор в базе данных и т. д.).

## Возвращаемое значение

Этот метод не возвращает никаких значений; результаты будут видны на платформе OpenReplay.
