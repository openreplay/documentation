Ваш экземпляр OpenReplay можно легко настроить. Здесь представлены пошаговые руководства по включению почтовых служб, защите вашей установки и удалению записей из базы данных.

## Пошаговые руководства
- [Настройка SMTP](/configuration/configure-smtp): Настройте SMTP для получения оповещений, еженедельных отчётов, приглашения новых пользователей в OpenReplay и сброса паролей.
- [Защита OpenReplay](/configuration/secure-or): Защитите OpenReplay, настроив SSL, reCaptcha и задав CSP.
- [Настройки прокси](/configuration/proxy-settings): Запускайте OpenReplay за прокси-сервером, таким как NGINX или Apache2.
- [Очистка хранилища](/configuration/cleanup-storage): Как хранятся записи и как массово удалять их из базы данных и очищать ваше хранилище.
- [Внешнее хранилище записей](/configuration/external-storage): Измените место хранения записей по умолчанию.
- [Внешняя база данных](/configuration/external-db): Используйте внешнюю базу данных Postgres для OpenReplay.
- [Единый вход (SAML)](/configuration/sso): Настройте единый вход (SSO) через SAML.
