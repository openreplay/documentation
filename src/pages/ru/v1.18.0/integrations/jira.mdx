Как интегрировать Jira Cloud с OpenReplay.

### 1. Создайте новый API-токен

1. Войдите в [токен JIRA](https://id.atlassian.com/manage/api-tokens).
2. Нажмите *Create API token*.
3. В поле метки введите «openreplay» и нажмите *Create*.
4. Нажмите *Copy to clipboard*.

### 2. Включите JIRA Cloud в OpenReplay

1. Перейдите в свою панель OpenReplay в раздел 'Preferences > Integration'
2. Выберите Jira. Если вы её не видите, значит у вас включён GitHub (вы не можете использовать оба одновременно, поэтому сначала обязательно отключите интеграцию с GitHub).
3. Укажите свой `Username` и `API token`, затем введите URL вашего экземпляра Jira Cloud

После этого во всех записях появится кнопка 'Create Issue', чтобы вы могли быстро создавать и назначать задачи в Jira.
