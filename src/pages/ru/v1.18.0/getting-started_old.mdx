:::caution[Использование с pnpm]
Из-за проблем в восходящих зависимостях Prettier плагин не будет обнаружен автоматически при использовании [pnpm](https://pnpm.io/). Чтобы он смог его найти, при запуске Prettier необходимо добавить следующий параметр:

```shell
prettier --write --plugin-search-dir=. .
```

Дополнительные настройки также требуются при использовании Prettier внутри VS Code. Подробнее см. в README плагина.
:::
