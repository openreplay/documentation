После того как вы [настроили](/mcp/setup) приложение MCP и вошли в систему, вы можете попросить Claude (или другой MCP-хост) запрашивать данные OpenReplay простым языком. На этой странице перечислено то, что MCP предоставляет в настоящее время, с примерами запросов для каждого случая.

<Aside type="tip" title="Совет" >
  Все запросы работают независимо от того, ссылаетесь ли вы на свой проект по **имени** (например, *"MyApp"*) или по **ID сайта** (например, *"site 1"*). Если вы не укажете его, MCP использует последний упомянутый вами проект.
</Aside>

## Поиск и фильтрация сессий

MCP поддерживает те же фильтры, что и интерфейс OpenReplay: браузер, ОС, устройство, страна, пути URL, клики, ввод, сетевые запросы, ошибки, метаданные, ID пользователей и длительность.

```
Show me 20 recent sessions in MyApp
```

```
Sessions from Chrome users in France or Tunisia
```

```
Sessions with failed requests to /api/users in MyApp
```

```
Sessions where the user visited the signup page then clicked "Subscribe"
```

```
Sessions longer than 20 minutes for user tahay@asayer.io
```

Результат отображается в виде интерактивного списка прямо в чате, поэтому вы можете перейти к любой сессии.

## Воронки

Воронки измеряют конверсию по последовательности URL, кликов, вводов или пользовательских событий. Отток показан пошагово.

```
Show me the checkout funnel last week: /cart → /checkout → /confirm
```

```
Conversion from /pricing to /signup this month
```

```
Visited /pricing then clicked Subscribe
```

```
Page view /checkout followed by a JS exception
```

## Просмотр повтора сессии

Повторы воспроизводятся во встроенном плеере прямо в чате. Вы можете перематывать, ставить на паузу и проверять детали так же, как в интерфейсе OpenReplay.

```
Show me the replay of session 7891234567890
```

```
Replay the last session in the list above
```

## Сочетайте и комбинируйте

Настоящая ценность раскрывается при объединении инструментов в цепочки. Несколько примеров:

- *"Find sessions where users hit a JS exception on /checkout this week, then show me the replay of the most recent one and tell me what happened."*
- *"Pull the funnel for /pricing → /signup → /dashboard, then for the step with the biggest drop-off show me 5 sessions that dropped off there."*
- *"Plot the timeseries of sessions from France this month. If there's a spike, find a few sessions from that day and summarize what they did."*

Поскольку Claude может рассуждать о результатах между вызовами инструментов, вы можете оставаться на уровне разговора вместо того, чтобы переключаться между панелями.
