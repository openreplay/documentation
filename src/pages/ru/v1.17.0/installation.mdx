Этот раздел содержит всё необходимое для начала работы с OpenReplay, от базовой до продвинутой настройки.

## Основные руководства
- [Как идентифицировать пользователя](/installation/identify-user): Свяжите ваш внутренний идентификатор пользователя с записываемой сессией.
- [Как добавить пользователей в вашу команду](/tutorials/invite-team-members): Узнайте, как добавлять пользователей в вашу команду.
- [Как отправлять метаданные](/installation/metadata): Также называемые traits или пользовательскими переменными, метаданные предоставляют дополнительную информацию о пользователях во время записи сессий.
- [Как отправлять пользовательские события](/installation/custom-events): Добавьте больше полезной информации, записывая события, специфичные для вашей предметной области, вместе с записями сессий.
- [Как настроить отчёты об ошибках](/installation/error-reporting): Сообщайте об ошибках и исключениях и просматривайте их на вкладке 'Errors'.
- [Как загружать Source-Maps](/installation/upload-sourcemaps): Упростите отладку, видя контекст исходного кода, полученный из трассировок стека, в его исходном виде.
- [Как очищать данные](/installation/sanitize-data): Как очищать данные как на глобальном, так и на детальном уровне.
- [Как создать пользовательскую эвристику](/tutorials/custom-heuristics): Узнайте, как добавить пользовательскую логику в бэкенд для проверки специфического поведения использования.
- [Как использовать панель OmniSearch](/tutorials/omnisearch): Узнайте, как извлечь максимум из нашей мощной строки поиска.
- [Как создавать собственные настраиваемые дашборды](/tutorials/custom-dashboard): Создайте собственные метрики и дашборды с помощью этого руководства.
