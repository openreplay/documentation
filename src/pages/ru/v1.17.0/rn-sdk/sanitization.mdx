## Компоненты UI

Санитизированные представления будут размыты в итоговом воспроизведении. Вы можете добавить любой компонент UI, и он будет автоматически удалён (для экономии памяти), как только перестанет отображаться на экране.

### ORSanitizedView

SDK экспортирует `ORSanitizedView`, который вы можете использовать, чтобы оборачивать чувствительные компоненты и размывать их.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORSanitizedView>
    <SensitiveComponent />
  </OR.ORSanitizedView>

// ...
```
