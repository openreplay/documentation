Этот метод используется для запуска трекера. После запуска трекера начинается запись сессии.

## Сигнатура

```swift
import OR from '@openreplay/react-native'

OR.tracker.startSession(options)
```

## Параметры инициализации

По умолчанию все параметры установлены в `true`:
- `crashes: Bool` Включает crashlytics.
- `analytics: Bool` Включает аналитическое отслеживание помеченных представлений.
- `performances: Bool` Включает слушатель производительности
- `logs: Bool` Включает слушатель логов
- `screen: Bool` Включает запись экрана

Параметры можно использовать для отключения определённых функций; в противном случае вы можете передать пустой объект (`{}`)
