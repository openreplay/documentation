Этот метод позволяет получить URL активной сессии. URL ведёт на панель управления OpenReplay, и этот метод полезен, если вы хотите интегрировать OpenReplay в собственный стек.

## Сигнатура

```js
getSessionURL(withCurrentTime?: boolean): string | undefined
```

## Параметры

- `withCurrentTime: boolean`: Необязательный параметр, по умолчанию равен `false`. Если `true`, URL будет содержать временную метку того момента в записи, когда был вызван метод.

## Возвращаемое значение

Этот метод возвращает URL активной сессии для воспроизведения на платформе OpenReplay. При желании URL может содержать временную метку момента вызова метода, чтобы сразу перейти к нужной точке в записи.
