OpenReplay Enterprise Edition поставляется с различными коннекторами для экспорта собранных данных (поведение пользователей и технические проблемы) в ваше хранилище данных. Мы поддерживаем:
- Amazon Redshift
- Google BigQuery
- PostgreSQL
- Snowflake
- ClickHouse
