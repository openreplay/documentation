Связывает анонимный userID с записываемой сессией.

## Сигнатура
```swift
ORTracker.shared.userAnonymousID(_ userID: String)
```

## Параметры

- `userID: String`: Анонимный идентификатор пользователя. Это строка, и она может быть любой, пока остаётся строкой, и позволяет однозначно идентифицировать пользователя в вашей системе (например, email, id в базе данных и т. д.).

## Возвращаемое значение

Этот метод не возвращает никаких значений, результаты будут видны на платформе OpenReplay.
