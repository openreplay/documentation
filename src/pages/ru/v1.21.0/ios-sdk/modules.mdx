## Методы и модули

- [Analytics](/ios-sdk/analytics): Узнайте всё об отслеживании конкретных представлений.
- [Network](/ios-sdk/network): Узнайте, как использовать NetworkListener.
- [Sanitization](/ios-sdk/sanitization): Узнайте, как скрывать представления на записанном экране.
- [Inputs](/ios-sdk/inputs): Узнайте, как записывать события ввода.
- [event()](/ios-sdk/event): Отправляйте пользовательские события, чтобы они записывались как часть ваших воспроизведений, с помощью этого метода.
- [setUserID()](/ios-sdk/set-user-id): Идентифицируйте своих пользователей с помощью этого метода.
- [setMetadata()](/ios-sdk/set-metadata): Задайте поле метаданных и сохраните его как часть записи.
- [start()](/ios-sdk/start): Начните запись сессии.
- [stop()](/ios-sdk/stop): Остановите запись сессии с помощью этого метода.
