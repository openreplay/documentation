Трекер будет прослушивать событие изменения значения (с debounce) и отправлять полученное значение на бэкенд OpenReplay.

## Добавление отслеживаемых полей ввода

SDK экспортирует `ORTrackedInput`, который вы можете использовать как обычные компоненты TextInput.

```js
import OR from '@openreplay/react-native'

// ... your app UI

  <OR.ORTrackedInput
    style={styles.input}
    onChangeText={onChangeNumber}
    value={number}
    placeholder="important placeholder"
  />

// ...
```
