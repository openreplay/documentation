Этот метод позволяет вам задать идентификатор пользователя для текущей активной сессии. Это позволяет связать пользователя с записью воспроизведения, что даёт возможность фильтровать и искать сессии конкретного пользователя.

## Сигнатура

```js
import OR from '@openreplay/react-native'

// ...
OR.tracker.setUserID('id')
```


## Параметры

- `userID: String`: Идентификатор пользователя. Это строка, и она может быть любой, пока остаётся строкой, и позволяет вам однозначно идентифицировать пользователя в вашей системе (например, email, идентификатор в базе данных и т. д.).

## Возвращаемое значение

Этот метод не возвращает никаких значений, результаты будут видны на платформе OpenReplay.
