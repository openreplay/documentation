В этом разделе собраны тематические руководства, которые помогут вам установить и настроить OpenReplay — от базовой установки до продвинутых конфигураций.

## Настройка пользователей

- [Как идентифицировать пользователя](/installation/identify-user): Свяжите ваш внутренний идентификатор пользователя с записываемой сессией.
- [Как отправлять метаданные](/installation/metadata): Метаданные, также называемые характеристиками или пользовательскими переменными, предоставляют дополнительную информацию о пользователях во время записи сессий.

## Настройка воспроизведения сессий

- [Как отправлять пользовательские события](/installation/custom-events): Добавьте больше аналитики, записывая события, специфичные для вашей предметной области, вместе с воспроизведением сессий.
- [Как настроить Canvas и WebGL](/installation/canvas): Узнайте, как интегрировать и настроить Canvas и WebGL для вашего проекта.
- [Как настроить междоменные iFrame](/installation/crossdomain-iframe): Настройте междоменные iFrame для бесшовной интеграции.
- [Как настроить WebSockets](/installation/websockets): Настройте WebSockets для обмена данными в реальном времени в вашем приложении.

## Сеть и конфиденциальность данных

- [Как настроить параметры сети (Веб)](/installation/network-options): Настройте сетевые конфигурации для вашей веб-среды.
- [Как настроить параметры сети (Мобильные устройства)](/installation/network-options-mobile): Настройте сетевые конфигурации для ваших сред iOS и Android.
- [Как очищать данные (Веб)](/installation/sanitize-data): Как очищать данные на глобальном и/или детальном уровне для ваших веб-приложений.
- [Как очищать данные (Мобильные устройства)](/installation/sanitize-data-mobile): Как очищать данные для приложений iOS и Android.

## Отладка и отслеживание ошибок

- [Как отслеживать ошибки](/installation/error-reporting): Отслеживайте ошибки и исключения и устраняйте неполадки с полным контекстом.
- [Как загружать Source-Maps](/installation/upload-sourcemaps): Упростите отладку, просматривая контекст исходного кода, полученный из трассировок стека, в его первоначальном виде.
