На этой странице описано, как очищать компоненты пользовательского интерфейса в ваших мобильных приложениях с помощью SDK OpenReplay для iOS и Android. Подробнее об очистке сетевых запросов см. в разделе [Сетевые опции (мобильные приложения)](/network-options-mobile).

## iOS - Очистка компонентов пользовательского интерфейса

Очищенные представления будут размыты в итоговой записи. Компоненты автоматически удаляются (для экономии памяти), как только перестают быть видимыми.

### SwiftUI

Добавьте модификатор `.sensitive()` к компоненту, который вы хотите очистить.

```swift
import ORTracker

Text("Sensitive Information")
    .sensitive()
```

### UIKit

Используйте метод `addIgnoredView`, чтобы пометить представление как конфиденциальное.

```swift
import ORTracker

ORTracker.shared.addIgnoredView(view)
```

### React Native

Оберните конфиденциальные компоненты компонентом `ORSanitizedView`, чтобы размыть их в записи.

```js
import OR from '@openreplay/react-native';

<OR.ORSanitizedView>
  <SensitiveComponent />
</OR.ORSanitizedView>
```

## Android - Очистка компонентов пользовательского интерфейса

Очищенные представления размываются в итоговой записи и автоматически удаляются из памяти, когда они находятся за пределами экрана.

### View

Пометьте представление как конфиденциальное с помощью метода `addIgnoredView`.

```kotlin
import com.openreplay.tracker.OpenReplay

OpenReplay.addIgnoredView(view)
```
