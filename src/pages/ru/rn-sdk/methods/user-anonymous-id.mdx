Свяжите анонимный userID с записываемой сессией.

## Сигнатура

```swift
ORTracker.userAnonymousID(userID: String)
```

## Параметры

- `userID: String`: Анонимный идентификатор пользователя. Это строка, и она может быть любой, если это строка, и она позволяет вам однозначно идентифицировать пользователя в вашей системе (например, электронная почта, id в базе данных и т. д.).

## Возвращаемое значение

Этот метод не возвращает никаких значений, результаты будут видны на платформе OpenReplay.
